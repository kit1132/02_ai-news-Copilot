## Microsoft 365 Copilot / Agent 365

### M365 三層リリースモデル開始（5月下旬〜）
Copilotの機能展開を **Frontier / Standard / Deferred** の3段階で管理する新モデルが開始。Deferred は主要変更を最大30日遅延。管理者は M365 admin center > Copilot > Settings で設定可能。最大100名の個別例外設定にも対応。今後M365全体に拡大予定だが時期は未定。
- https://blog-en.topedia.com/2026/04/new-three-tier-release-model-for-microsoft-365-starting-with-copilot/

### Copilot Chat ライセンス変更実施（5/16）
2,000シート超の組織で、未ライセンスユーザーのWord/Excel/PPT/OneNote内Copilotが利用不可に。Copilot Chat (Basic) とOutlook内Copilotは継続利用可。新ラベル体系「Copilot Chat (Basic)」「M365 Copilot (Premium)」を導入。
- https://chrismenardtraining.com/post/microsoft-365-copilot-chat-changes-on-april-15-2026-what-unlicensed-users-need-to-know/

### GPT-5.3-Codex がGitHub Copilot Business/Enterpriseのベースモデルに（5/17）
GPT-4.1を置き換え。GitHub初のLTSモデル（12ヶ月保証、〜2027/2/4）。アジェンティックコーディングタスクでGPT-5.2-Codex比25%高速。1x premium request unit。GPT-4.1は6/1の従量課金開始時に廃止予定。
- https://github.blog/changelog/2026-05-17-gpt-5-3-codex-is-now-the-base-model-for-copilot-business-and-enterprise/

### Copilot on Web モデル整理（5/20）
Geminiモデル全廃、GPT-5.2-Codex・GPT-5.4 nano削除。OpenAI + Claudeモデルに集約。
- https://github.blog/changelog/2026-05-20-updates-to-available-models-in-copilot-on-web/

### Copilot Critique（マルチモデルディープリサーチ）
Anthropic/OpenAIモデルを組み合わせ、モデル1が調査・起草→モデル2が検証・強化するデュアルモデルリサーチ機能。
- https://x.com/itsPaulAi/status/2038615654614348263

### Federated Copilot Connectors
Moody's、HubSpot、LSEG、Notionなどパートナー経由のリアルタイムデータをCopilot内で利用可能に。
- https://techcommunity.microsoft.com/blog/microsoft365copilotblog/what%E2%80%99s-new-in-microsoft-365-copilot--april-2026/4510935

### その他のM365 Copilot更新
- **PDFをCopilot Chatで直接オープン**: 別タブ不要でChat内閲覧
- **Outlook email grounding**: メールスレッドをプロンプトコンテキストに追加可能
- **AI Charts web part（SharePoint）**: 自然言語でインタラクティブチャート生成
- **File Actions with AI**: 自然言語でファイルのリネーム・移動・整理
- **Planner統合**: Copilot Chat内でタスク作成・期日確認・チーム割当

## Copilot Studio

### リアルタイム音声エージェント ガバナンスガイド公開（5/19）
大規模運用のための5段階フレームワーク: ビルダー管理→ビルド管理→リリース管理→ランタイム監視→ライフサイクル管理。テキストエージェントとの違い（レイテンシ要件・エスカレーション複雑性）を明示。アンチパターン集も掲載。
- https://www.microsoft.com/en-us/microsoft-copilot/blog/copilot-studio/the-in-depth-guide-to-managing-real-time-voice-agents-at-scale/

### Computer Use GA（5月）
エージェントがWebブラウザやデスクトップアプリを直接操作。APIのないレガシーシステム連携やWebフォーム自動入力に活用。

### 新機能まとめ
- **A2A（Agent-to-Agent）プロトコル**: エージェント間接続がGA
- **カスタムメトリクス分析**: 独自分析カテゴリを定義し組込み分析と並列表示
- **REST API経由エージェント評価**: CI/CDパイプライン統合可能
- **マルチターン会話テスト**: 対話フロー全体の評価（Preview）
- **Human-in-the-Loop改善**: Outlook経由で情報収集→動的パラメータとして再開
- **CLI GA: 5/31予定**: Power Platform CLI拡張でエージェント定義のデプロイ

## Claude Code

### v2.1.148（5/22）
Bashツールが全コマンドでexit code 127を返すリグレッション（v2.1.147由来）を修正。

### v2.1.145〜147の主な変更
- **ピン留めバックグラウンドセッション**: Ctrl+T でピン留め。アイドル時も生存、アップデート時にインプレース再起動
- **/resume でバックグラウンドセッション対応**: `claude --bg` やagent viewで開始したセッションも一覧表示（`bg` マーク付き）
- **/simplify → /code-review に改名**: 正確性バグをeffort level指定で検出。`--comment` でGitHub PR にインラインコメント投稿
- **フルスクリーンモード改善**: プロンプト入力時にスクロール位置が飛ばなくなった
- **ダイアログスクロール対応**: 矢印キー/PgUp/PgDn/マウスホイール
- **自動アップデーター改善**: ネットワーク障害リトライ、エラー詳細報告
- https://github.com/anthropics/claude-code/releases

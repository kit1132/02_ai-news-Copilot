## 最重要: Microsoft Scout / Work IQ APIs / GPT-5.5 GA

### Microsoft Scout発表 — 初の"Autopilot"パーソナルエージェント
- **概要**: 常時バックグラウンドで稼働し、ユーザーに代わって自律的に行動する新カテゴリ「Autopilot」エージェント。OpenClawフレームワーク基盤
- **主な機能**: 会議の自動スケジュール・準備資料生成、カレンダーの自動ブロック、ワークフローリスク検知（滞留した意思決定の検出等）
- **セキュリティ**: エージェントごとに個別のEntra ID。既存のデータ保護ポリシー（秘密度ラベル、DLP）を尊重
- **提供状況**: Private Preview（Frontier Program経由）。GitHub Copilotライセンスが必要
- **URL**: https://www.microsoft.com/en-us/microsoft-365/blog/2026/06/02/introducing-microsoft-scout-your-always-on-personal-agent/

### Work IQ APIs — エージェント向けM365インテリジェンスレイヤー
- **概要**: メール・カレンダー・会議・チャット・ファイル・組織情報を意味的にインデックス化し、エージェントが消費可能な形式で提供するAPI群
- **技術的特徴**: 従来の数百APIを10個の汎用ツール（MCP経由）に集約。トークン消費80%削減、処理速度2倍
- **4ドメイン構成**: Chat（Copilot応答+引用）/ Context（合成なしのデータ集約）/ Tools（エンティティ・アクション）/ Workspaces（テナント内セキュアストレージ）
- **提供状況**: Public Preview中、**6/16 GA予定**。消費ベース課金（Copilot Credits）
- **URL**: https://www.microsoft.com/en-us/microsoft-365/blog/2026/06/02/announcing-the-new-work-iq-apis/

### GPT-5.5 Microsoft FoundryでGA (6/3)
- **概要**: GPT-5.5がMicrosoft Foundryで一般提供開始。GPT-5.5 Proはプレミアムバリアント
- **特徴**: 長文コンテキスト推論強化、エージェント実行の安定性向上、Computer Use精度改善、トークン効率向上
- **価格**: GPT-5.5: $5/M入力・$0.50/Mキャッシュ・$30/M出力 / GPT-5.5 Pro: $30/M入力・$3/Mキャッシュ・$180/M出力
- **Foundryカタログ**: 11,000+モデル（OpenAI GPT-5.5、Anthropic Claude Opus 4.8/Sonnet 4.5/Haiku 4.5、Fireworks AIオープンソース、MAIファミリー）
- **URL**: https://azure.microsoft.com/en-us/blog/openais-gpt-5-5-in-microsoft-foundry-frontier-intelligence-on-an-enterprise-ready-platform/

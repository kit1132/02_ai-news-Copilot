- **Pro プランから Opus モデルを削除**（Opus 4.7 は Pro+ のみ）
- VS Code / Copilot CLI に残り使用量表示を追加
- 4/20-5/20 に想定外の制限を受けたユーザーは返金対応
- https://github.blog/changelog/2026-04-20-changes-to-github-copilot-plans-for-individuals/

- https://github.blog/news-insights/company-news/changes-to-github-copilot-individual-plans/

### Claude Code デスクトップ再設計 + Routines（4/14、補足）
- デスクトップアプリを全面再設計: マルチセッションサイドバー、ドラッグ&ドロップペイン、統合ターミナル、アプリ内ファイルエディタ、HTML/PDFプレビュー
- **Routines**（Research Preview）: プロンプト + リポジトリ + コネクタを1つの設定にパッケージ化し、セッション不要で自動実行
  - Scheduled Routines: cron 的な定期実行（ドキュメント整合性チェック等）
  - API Routines: HTTP エンドポイント経由でトリガー（Datadog / CI/CD 連携）
  - Webhook Routines: GitHub イベント（PR コメント、CI 失敗）に自動対応
- https://claude.com/blog/claude-code-desktop-redesign

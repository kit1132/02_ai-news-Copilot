# AI News Digest - 2026-04-07

## Microsoft 365 Copilot

### Copilot Chat 4/15 仕様変更の詳細が明確化（あと8日）
- 4/15 より Word / Excel / PowerPoint / OneNote での Copilot Chat アクセスが制限される変更の具体的影響が判明
- **2,000人以上の組織**: Copilot Chat (Basic) のボタン・サイドパネルが完全削除。アプリ内アクセス不可に
- **2,000人未満の組織**: 完全削除ではなく「標準アクセス」に格下げ。ピーク時に品質・パフォーマンスが制限される
- Outlook での Copilot Chat アクセスは全組織で維持。M365 Copilot App 内のエージェントも引き続き利用可能
- 新ラベリング: 無料ユーザーは「Copilot Chat (Basic)」、有料ライセンスは「M365 Copilot (Premium)」に区分
- https://office-watch.com/2026/microsoft-removes-copilot-chat-word-excel-powerpoint-april-2026/
- https://samexpert.com/copilot-chat-enterprise-restriction-april-2026/

### Word for iPhone で Copilot 共同作成機能を追加
- Word 文書内で Copilot を共同作成者（co-creator）として直接利用可能に
- iPhone 向け Word アプリが対象。文書の編集・構造調整・内容の洗練をリアルタイムで Copilot と共同作業
- https://techcommunity.microsoft.com/blog/microsoft365insiderblog/work-with-copilot-as-a-co%E2%80%91creator-directly-in-word-for-iphone/4507902

### Project Manager Agent がワールドワイド展開中（4月〜）
- AI によるプロジェクト計画・進行管理を支援するエージェント。3月にパブリックプレビュー開始、4月からワールドワイド展開
- https://techcommunity.microsoft.com/blog/microsoft365copilotblog/what%E2%80%99s-new-in-microsoft-365-copilot--march-2026/4506322

## Copilot Studio / Power Platform

### 2026 Release Wave 1 新機能の詳細
- **OpenAPI v3 コネクタ構築**: OpenAPI v3 仕様を直接インポートしてコネクタ作成可能に。v2 へのダウングレード不要
- **エージェントワークフローに人間入力**: 「request for information」アクションで、エージェントフローを一時停止し Outlook 経由でレビュアーから情報を収集（プレビュー）
- **M365 Copilot → Copilot Studio コピー**: M365 Copilot で作成したエージェントを Copilot Studio にコピーして高度なカスタマイズが可能に
- **GCCH 環境対応**: Agent Builder が Government Community Cloud High 環境で利用可能に
- https://learn.microsoft.com/en-us/power-platform/release-plan/2026wave1/microsoft-copilot-studio/planned-features

## AI ツール・開発環境

### Claude.ai 大規模障害（4/6）
- 4/6 にClaude.ai で約2時間の障害が発生。ログイン、チャット、音声モード、Claude Code に影響
- Downdetector で 2,500 人以上が報告。10:30 AM ET 頃から障害発生、12:44 PM ET に復旧
- https://www.techradar.com/news/live/claude-anthropic-down-outage-april-6-2026

### Claude API: max_tokens 上限を 300K に引き上げ
- Message Batches API で Claude Opus 4.6 / Sonnet 4.6 の max_tokens 上限が 300K に拡大
- `output-300k-2026-03-24` ベータヘッダーで長文生成・大規模コード生成に対応
- 1M コンテキストベータ（Sonnet 4.5/4 向け）は 4/30 終了。Sonnet 4.6 / Opus 4.6 では標準対応のため移行推奨
- https://releasebot.io/updates/anthropic

### Claude Code: /powerup レッスン・パフォーマンス改善
- インタラクティブな `/powerup` レッスン機能を追加（学習コンテンツ）
- セッション再開（resume）の高速化・安定性向上
- ダイアログ、フック、編集、スクロール、PowerShell 権限周りの修正多数
- プライバシー強化とセッションハンドリングの高速化
- https://releasebot.io/updates/anthropic/claude-code

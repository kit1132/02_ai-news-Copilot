### GitHub Copilot データ学習ポリシー変更 — 明日（4/24）発効
- Free / Pro / Pro+ ユーザーのインタラクションデータ（入出力・コードスニペット・コンテキスト）をモデル学習に使用。**デフォルト ON**
- **プライベートリポジトリのコードも対象**（Copilot 使用中に限る）
- GitHub Community Discussion #188488 で **232 ダウンvote**。開発者コミュニティの強い反発
- opt-out: `/settings/copilot/features` → Privacy で無効化。過去に opt-out 済みの場合は設定維持
- Business / Enterprise / Student / Teacher は対象外
- https://github.blog/news-insights/company-news/updates-to-github-copilot-interaction-data-usage-policy/

### Anthropic Claude Code — Pro プラン除外テスト＆即日撤回（4/21-22）
- 4/21に $20 Pro プランから Claude Code を除外（最低 Max 5x $100/月 が必要に）。価格ページの Pro 列で Claude Code に ✕ を表示
- Anthropic 成長責任者 Amol Avasare：「**新規 prosumer サインアップの約2%を対象にしたテスト**」と説明。既存 Pro / Max 加入者は影響なし
- **数時間後に撤回**。ランディングページとドキュメントの更新は「テスト対象外の98%にとって混乱を招くミスだった」と認め、変更を revert
- 背景: エージェント型利用の急増で「現行プランはこの使い方を想定していない」（Avasare）。今後の料金改定の布石とみられる
- https://www.theregister.com/2026/04/22/anthropic_removes_claude_code_pro/

## AIツール・開発エージェント

### Claude Code 新機能（4月中旬〜下旬）
- **Ultraplan（Early Preview）**: CLI からクラウド上でプラン作成 → Web エディタでレビュー・コメント → リモート実行またはローカルに pull back
- **Monitor ツール**: バックグラウンドイベントを会話にストリーミング。ログの tail やリアルタイム反応が可能に
- `/loop` のインターバル省略時自動ペーシング、`/team-onboarding`（セットアップのリプレイ可能ガイド化）、`/autofix-pr`（ターミナルから PR 自動修正を有効化）
- `/resume` が大規模セッション（40MB+）で最大67%高速化。MCP 起動の高速化
- https://code.claude.com/docs/en/changelog

### Power Pages — エージェントコードプラグイン拡張
- Power Pages の GitHub Copilot / Claude Code CLI 向けエージェントコードプラグインに **3つの新スキル**を追加
- サーバーサイドロジック（これまで AI アシストビルディングで欠けていた機能）に対応
- https://www.microsoft.com/en-us/power-platform/blog/2026/04/09/whats-new-in-power-platform-april-2026-feature-update/

### Cursor — Bugbot Learned Rules
- PR フィードバック（開発者のリアクション・リプライ・人間レビューアーのコメント）から学習するルールを Bugbot に追加
- バグ検出精度の向上と false positive の低減を狙う
- https://cursor.com/changelog

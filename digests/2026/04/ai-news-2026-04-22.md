### M365 Community Conference 2026 開幕（4/21-23, Orlando）
- Day 1（4/21）: Jeff Teper キーノート「Building for the Future: Microsoft 365, Agents and AI」。Copilot・Teams・SharePoint のAI統合ロードマップを発表。マルチエージェント連携システムの実演
- Day 1（4/21）: Ryan Cunningham キーノート「Business Apps and Agents」。M365 Copilot + Copilot Studio + Power Apps + **Agent 365** の統合デモ。エージェント構築からIT管理者向けガバナンスまで一気通貫
- Day 2（4/22、本日）: Vasu Jakkal & Rohan Kumar キーノート「Securing AI: Building Trust in the Era of AI」。Purview・Defender・Entra・Security Copilot 統合によるプロンプトインジェクション・モデル改ざん・Shadow AI 対策
- M365 E7 Frontier Suite（E5 + Entra Suite + Copilot + Agent 365）がトランザクション可能に
- 200+ セッション、21 ワークショップ、200+ Microsoft エンジニア参加
- https://techcommunity.microsoft.com/blog/microsoft_365blog/your-guide-to-the-microsoft-365-community-conference/4505384

### Security Copilot × M365 E5 バンドル展開開始（4/20）
- MC1261596 に基づき、M365 E5 ライセンスに Security Copilot を追加費用なしで包含する段階展開が 4/20 開始（〜6/30）
- **400 Security Compute Units（SCU）/ 1,000 ユーザー**（上限 10,000 SCU/月/テナント）
- Defender・Entra・Intune・Purview 横断のAI脅威分析・インシデント要約・エージェント防御
- テナントにはアクティベーション7日前と当日に通知。データ共有がデフォルトONのため管理者確認推奨
- https://mc.merill.net/message/MC1261596

### GitHub Copilot 個人プラン大幅変更（4/20）
- **新規サインアップ一時停止**: Pro / Pro+ / Student プランの新規登録を一時停止
- **使用量制限の厳格化**: Pro+ は Pro の 5 倍以上の制限枠。VS Code / CLI に残量表示を追加
- **モデル制限**: **Opus モデルが Pro プランから除外**。Opus 4.7 は Pro+ のみ。Opus 4.5/4.6 は Pro+ からも削除
- 背景: エージェント型ワークフローの長時間・並列セッションが従来プラン設計の想定を大幅超過
- 4/20〜5/20 に GitHub サポートへ連絡で残期間の返金対応
- https://github.blog/changelog/2026-04-20-changes-to-github-copilot-plans-for-individuals/

### M365 Copilot 4月のロールアウト
- **会議ビデオハイライトリール**: Copilot Chat で会議要約時に、文字要約に加えてナレーション付きハイライト動画を生成
- **AI in SharePoint（Public Preview）**: メタデータ自動付与・ライブラリ自動適応・情報構造化支援。5月に全世界展開
- **Excel × Work IQ**: Copilot 編集時に、メール・会議・チャット・ファイルから関連コンテキストを自動取得してマルチステップ編集精度を向上
- **Copilot Search 管理者制御**: SharePoint Online サイトを権威ソースとして指定、ドメイン除外設定が4月展開

## AIツール・開発エージェント

### Claude Haiku 3 正式廃止（4/19 実施済み）
- `claude-3-haiku-20240307` への API リクエストがエラーを返すように。**猶予期間なしのハード廃止**

- Vertex AI (Google Cloud) でも同日廃止

### GitHub Copilot CLI v1.0.33（4/20）
- `--resume` / `--continue` で `--remote` フラグ自動継承
- `auto` モデル選択: Copilot が各セッションに最適なモデルを自動選択（4/17 発表の機能）
- 週次使用量の 75% / 90% 到達時に警告表示
- ドキュメントファイルをプロンプトに添付してエージェントに読ませる機能
- `--connect` フラグでリモートセッションに直接接続
- https://github.blog/changelog/2026-04-20-changes-to-github-copilot-plans-for-individuals/

### GitHub Copilot データ学習ポリシー変更（4/24 発効 — 2日後）
- Free / Pro / Pro+ ユーザーのインタラクションデータをモデル学習に使用（デフォルトON）
- Business / Enterprise は除外
- opt-out 設定は Settings > Copilot から。X上で駆け込み opt-out 情報が拡散中
- https://github.blog/changelog/month/04-2026/

# AI News Digest - 2026-04-01

## Microsoft Copilot / Power Platform

### 2026 Release Wave 1 本日プロダクション展開開始
本日4/1より、2026 Release Wave 1のプロダクション展開が開始。今回はEarly Accessなし、GA直接展開。Power Platform・Copilot Studio・Dynamics 365全体で数百の新機能が対象。「エージェンティック・エコシステム」への移行を強調。
- https://learn.microsoft.com/en-us/power-platform/release-plan/2026wave1/

### Copilot Tuning in Agent Builder — 4月Frontierロールアウト
Agent Builder内にCopilot Tuningテンプレートが導入。複雑な文書作成・ガイドライン検証・組織固有のスタイルへの編集をテンプレート化し、コンテキスト・ツール・モデルを組織のデータでカスタマイズ可能。4月にFrontier、6月にワールドワイド展開予定。M365 Copilotライセンス5,000席以上が対象。
- https://learn.microsoft.com/en-us/copilot/release-plan/2026wave1/

### Project Manager Agent — 4月GA展開
AIアシストによるプロジェクト計画・管理エージェント。3月にPublic Preview開始済み、4月にワールドワイドGA。タスク管理を起点に、プロジェクト調整を簡素化。
- https://learn.microsoft.com/en-us/copilot/release-plan/2026wave1/

### サードパーティモデルプロバイダー制御 — 4月下旬ロールアウト
管理者がMicrosoft Admin CenterからAnthropicやxAI等のサードパーティAIモデルを特定のユーザー/グループに割り当て可能に。Entra IDグループベースのアクセス管理に対応。4月下旬から展開開始。
- https://m365admin.handsontek.net/microsoft-365-copilot-admins-will-able-enable-third%E2%80%91party-model-providers-specific-users-groups/

### Copilotモバイルアプリ チャットファーストUI — 4月下旬
M365 Copilotモバイルアプリが新しいチャットファーストデザインに刷新。ナビゲーション・チャット入力・音声操作を改善。iOS先行、Android続行。管理者設定不要で全ユーザー自動適用。
- https://m365admin.handsontek.net/microsoft-365-copilot-new-chat-first-design-copilot-mobile-app/

### Copilot Analytics包括メトリクス — 4月下旬〜5月
Edge・OneNote・Outlook・Word・Excel・PowerPointでのCopilot利用状況メトリクスが拡充。Copilot DashboardとAnalyticsに反映。4月下旬から5月下旬にかけてワールドワイド展開。
- https://m365admin.handsontek.net/microsoft-365-copilot-comprehensive-copilot-metrics-copilot-analytics/

### 管理者向けガバナンス機能 — 4月展開
- **Authoritative Sources**: SharePoint Onlineサイトを権威あるコンテンツとして指定可能に（Copilot Search）
- **Domain Exclusion**: Web Grounding対象から除外するドメインを指定可能に（コンプライアンス・コンテンツガバナンス）
- **Purview DLP**: Copilot/Copilot Chatの外部Web検索での機密データブロック機能が3月下旬〜4月下旬で展開完了
- https://learn.microsoft.com/en-us/copilot/microsoft-365/release-notes

### Copilot Chat仕様変更リマインダー — 4/15適用
2,000シート以上の組織で、M365 Copilotアドオン未割当ユーザーはWord/Excel/PowerPoint/OneNoteでCopilot利用不可に。ライセンスなし=「Copilot Chat (Basic)」、あり=「M365 Copilot (Premium)」の二層構造へ。同日にPower Platform & Copilot Studioアップデートイベント開催。
- https://blogs.imperial.ac.uk/office365/2026/03/20/copilot-chat-changes-april-2026/

### GitHub Copilot: GPT-5.1系モデル本日非推奨化
GPT-5.1 / GPT-5.1-Codex / GPT-5.1-Codex-Mini / GPT-5.1-Codex-Max が本日4/1で非推奨に。後継はGPT-5.3-Codex。
- https://www.ai-souken.com/article/github-copilot-updates-2026

## 開発ツール

### Cursor: セルフホステッドクラウドエージェント（3/25）
コードとツール実行を完全に自社ネットワーク内に保持するセルフホステッドクラウドエージェントに対応。コードベース・ビルド出力・シークレットすべてが内部マシンで実行。Cursor-hostedと同等機能（隔離VM、フル開発環境、マルチモデルハーネス、プラグイン）。
- https://cursor.com/changelog

### Cursor: Composer 2（3/19）
フロンティアレベルのコーディング性能。Standard $0.50/M入力・$2.50/M出力、Fast（デフォルト）$1.50/M入力・$7.50/M出力。
- https://cursor.com/changelog

### Cursor: JetBrains IDE対応（3/4）
Agent Client Protocol（ACP）経由でIntelliJ IDEA・PyCharm・WebStorm等のJetBrains IDEに対応。OpenAI・Anthropic・Google・Cursorのフロンティアモデルが使用可能。
- https://cursor.com/changelog

### Devin: プレビューエージェントトグル（3/27）
「Preview upcoming features」トグルでストリーミング思考と高速実行が利用可能に。インラインファイルプレビュー（HTML/PDF/SVG）、Focus Mode（Cmd+Shift+F）も追加。
- https://docs.devin.ai/release-notes/overview

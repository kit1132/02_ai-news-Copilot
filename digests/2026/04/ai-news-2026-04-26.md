## AI ツール・業界動向

### SpaceX が Cursor を $60B で買収するオプション契約を締結（4/21）
SpaceX（xAI 統合後）が AI コーディングスタートアップ Cursor の買収権を $600億で取得。$100億のコラボレーション料を前払いするか、今夏の SpaceX IPO 後に $600億で完全買収するかの二択構造。Cursor は $500億超評価額での $20億 Series E（a16z 主導、Nvidia・Thrive Capital 参加予定）を中断して本ディールを選択。25歳の CEO Michael Truell が率いる Cursor は 2025年1月に $25億評価額 → 5月 $90億 → 11月 $293億（Series D $23億）と急成長。Microsoft も SpaceX 前に買収を検討していたことが CNBC 報道で判明。AIコーディングツール市場の争奪戦が本格化。
- https://www.cnbc.com/2026/04/21/spacex-says-it-can-buy-cursor-later-this-year-for-60-billion-or-pay-10-billion-for-our-work-together.html
- https://www.cnbc.com/2026/04/22/microsoft-looked-at-buying-cursor-before-spacex-deal-sources-say.html

## Microsoft 365 Copilot / Copilot Studio

### Windows 11 エンタープライズ向け Copilot アンインストールポリシー提供開始
4月 Patch Tuesday 以降、RemoveMicrosoftCopilotApp ポリシーが一般提供。Group Policy / Intune / SCCM 経由で管理デバイスから Copilot を削除可能に。適用条件: Windows 11 25H2、M365 Copilot と Microsoft Copilot が両方インストール済、ユーザーが手動インストールしていない、過去28日間に未起動。企業からの長期的な要望に応えた対応。
- https://www.bleepingcomputer.com/news/microsoft/microsoft-now-lets-admins-uninstall-copilot-on-enterprise-devices/

### Agent 365 / M365 E7 GA まで5日（5/1）— 価格構成の詳細
Agent 365: $15/user/月（スタンドアロン）。M365 E7（Frontier Suite）: $99/user/月で M365 E5（$60、7月から）+ M365 Copilot（$30）+ Entra Suite（$12）+ Agent 365（$15）をバンドル。単品合計 $117 に対し約15%割引。エージェントの可視化・ガバナンス・セキュリティの統合管理基盤として位置づけ。
- https://samexpert.com/microsoft-365-e7-licensing-guide/
- https://blogs.microsoft.com/blog/2026/03/09/introducing-the-first-frontier-suite-built-on-intelligence-trust/

### MCP Apps が M365 Copilot Chat で利用可能に（Developer Blog, 4/7）
宣言型エージェントが Copilot Chat 内でインタラクティブ UI（経費承認フォーム、プロジェクトダッシュボード、ドキュメント比較画面等）をレンダリング可能に。サンドボックス化された iFrame 内で安全に実行。Inline モード（軽量ウィジェット）と Side-by-side モード（複雑な編集用ワークスペース）の2形態。M365 Agents Toolkit + GitHub Copilot CLI で MCP サーバー付きエージェントを会話だけでゼロからデプロイ可能。
- https://devblogs.microsoft.com/microsoft365dev/mcp-apps-now-available-in-copilot-chat/

## Power Platform

### Test Engine 非推奨化（4月発効）
Power Apps Test Engine が4月付で deprecated。ドキュメントと GitHub リポジトリの Microsoft によるメンテナンスは終了。テスト自動化は Power Automate のテスト機能やサードパーティツールへの移行が必要。
- https://learn.microsoft.com/en-us/power-platform/important-changes-coming

### モデル駆動型アプリのクラシック表示が完全廃止（4月）
メーカーがクラシック表示に切り替えるオプションが削除され、全アプリがモダン（Fluent 2）表示に統一。カスタムテーミング GA と合わせて、ブランドカラー・フォント・ヘッダースタイリングの統一管理が可能に。
- https://www.microsoft.com/en-us/power-platform/blog/2026/04/09/whats-new-in-power-platform-april-2026-feature-update/

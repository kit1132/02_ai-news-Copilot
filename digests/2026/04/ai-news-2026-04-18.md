## Copilot Studio / M365 Copilot

### MC1282306: M365 変更管理モダナイズ（4月中旬〜下旬展開）
- リリースオーディエンス制を導入: **Frontier**（早期実験）/ **Standard**（GA即時、デフォルト）/ **Deferred**（30日遅延、セキュリティ・コンプライアンス審査用）
- Message Center の構造刷新: コンプライアンスファースト、法務・セキュリティ・ヘルプデスク準備向けのアクション可能な通知
- **MCP Server によるAI変更管理**: Microsoft Release Communications MCP Server と Microsoft MCP for Enterprise を提供。リアクティブな火消しからプロアクティブなシグナル駆動計画へ
- 初期スコープは M365 Copilot 機能が対象
- https://mc.merill.net/message/MC1282306

### MC1263276: サードパーティモデルプロバイダーのユーザー割当
- 管理者が M365 Copilot 内でサードパーティ AI モデルプロバイダーを特定ユーザー・グループに割当可能に
- 4月下旬から展開開始
- https://mc.merill.net/message/MC1263276

### MC1281426: Copilot Hub モデル駆動アプリ設定変更
- Power Apps の AI フォーム入力支援・自然言語グリッド検索・データ可視化の管理者制御が Copilot Hub > Settings に集約
- 環境グループ・Entra グループ対応の細粒度制御を追加
- https://mwpro.co.uk/blog/2026/04/15/power-platform-copilot-hub-new-model-driven-app-settings-for-data-entry-data-exploration-and-summary-mc1281426/

# AI News Digest - 2026-06-05

## GitHub Copilot — 従量課金の「請求ショック」が深刻化

6/1の従量課金（AI Credits）移行後、開発者からの反発が拡大。6/4付でVisual Studio Magazineが「Copilot Billing Shock Hits Developers」と題した続報を掲載。

- **具体的コスト事例**: Pro+（$39/月）ユーザーが2時間で月間クレジットの8%を消費。Redditでは月額$29→$750に跳ね上がると試算する報告も
- **エージェント利用が特に高額**: 単一の変更リクエストで$6超を消費するケースあり。パワーユーザーは10x〜50xのコスト増を予測
- **代替ツールへの流出**: RooCode、LM Studio、OpenRouter等への移行検討が拡大。Anthropic直接利用やCursorへの乗り換えを表明するユーザーも増加
- **GitHub/Microsoftの対応**: 直接的な公式回答はまだない状態

https://visualstudiomagazine.com/articles/2026/06/04/copilot-billing-shock-hits-developers.aspx
https://www.theregister.com/ai-and-ml/2026/06/02/github-copilot-users-threaten-exit-as-metered-billing-kicks-in/5249826

## Claude Code v2.1.162（6/3リリース）

v2.1.161（6/2）に続く安定性・UX改善リリース。

- `claude agents --json` に `waitingFor` フィールド追加（パーミッションプロンプト待ち等の状態を可視化）
- Remote Controlが起動メッセージではなく永続フッターピルとして表示
- WebFetch権限ルールが組み込みプリアプルーブドドメインに適用されない問題を修正
- Windows権限ルールのバックスラッシュ・大小文字パス不一致を修正
- MCP per-serverタイムアウト値1000ms未満の処理を修正
- スラッシュコマンドのオートコンプリート: クリックで即実行ではなくプロンプトに入力される動作に変更
- Windsurf → Devin Desktopへのメニュー名変更対応

https://code.claude.com/docs/en/changelog

## Cursor — Composer 2.5詳細とBugbot課金変更

### Composer 2.5（5/18リリース）ベンチマーク詳細

Moonshot Kimi K2.5ベースで構築。25倍の合成コーディングタスクとターゲットRLで訓練。

- **SWE-Bench Multilingual**: 79.8%（Opus 4.7/GPT-5.5と同等水準）
- **CursorBench v3.1**: 63.2%
- **コスト**: $0.50/M入力トークン（競合の約1/10）。高速版は$3.00/M入力
- 長時間タスクの安定性・複雑な指示への追従性が大幅改善

https://cursor.com/blog/composer-2-5

### その他の変更

- **Bugbot**: UBB（使用量ベース課金）に移行。シート料金廃止、PR1件あたり$1.00〜$1.50。カスタムレビューロジック・エフォートレベル設定可能
- **Enterprise組織管理**: 複数チームの一元管理がGA。セキュリティ・ガバナンス・予算・機能を個別制御可能
- **Teams**: 使用量制限増加、Premium席新設。90%のチームでコスト削減見込み（7/1〜）

https://cursor.com/changelog

## 高優先

### GitHub Copilot: 個人プラン大幅変更 + 6/1 従量課金開始まで1週間
- **新規サインアップ一時停止**: Pro/Pro+/Student プランの新規登録を停止中
- **モデル制限**: Pro プランから Opus モデル削除。Pro+ では Opus 4.7 のみ残存（4.5/4.6 は削除）
- **利用上限引き下げ**: エージェント型ワークフローの計算コスト増大が理由。VS Code / CLI で上限表示開始
- **6/1 従量課金移行**: 月額プランは自動移行。AI Credits ベース（1 Credit = $0.01）。コード補完・Next Edit は含まれず消費なし
- https://github.blog/news-insights/company-news/changes-to-github-copilot-individual-plans/
- https://github.blog/news-insights/company-news/github-copilot-is-moving-to-usage-based-billing/

### Anthropic Mythos 1: Claude Code / Claude Security 向け次世代モデルの存在が判明
- `claude-mythos-1-preview` モデル ID が Claude Code 内で一時的に確認された後、非表示に
- **Project Glasswing**: Mythos Preview を用いて10,000件超のクリティカル脆弱性を自律発見（FreeBSD NFS 17年もの RCE 含む）
- Claude Security（Enterprise向け脆弱性トリアージダッシュボード）への統合を準備中
- 正式リリースは6月下旬〜7月上旬と観測筋が予測
- https://www.testingcatalog.com/anthropic-prepares-mythos-1-for-claude-code-and-claude-security/

### Cursor 3.5（5/20）: Automations が Agents Window に統合
- **Automations in Agents Window**: cursor.com/automations に加え、エディタ内の Agents Window からもオートメーション作成・管理が可能に
- **マルチリポ対応**: 1つのオートメーションに複数リポジトリをアタッチ可能。リポジトリ横断でエージェントが推論
- **リポなしオートメーション**: コード以外のツール監視・アクション実行用。リポジトリ不要のオートメーションが作成可能に
- https://cursor.com/changelog/05-20-26

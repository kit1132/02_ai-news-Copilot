## AI ツール・業界動向

### AIコーディングツール「定額制の終焉」が業界トレンドに
4月中に GitHub Copilot（個人プラン制限強化・新規サインアップ停止）、Anthropic Claude Code（プロンプトキャッシュ1時間→5分短縮）、Cursor（フロンティアモデルを Max Mode 限定化）の3社が一斉にコスト構造を変更。背景はエージェント型ワークフローによる計算コストの急増。GitHub は「少数のリクエストがプラン価格を超えるコストを発生させることが常態化した」と説明。業界全体が定額制から従量課金モデルへ移行する転換点として注目されている。
- https://medium.com/activated-thinker/the-flat-rate-ai-coding-subscription-era-is-ending-what-github-copilot-claude-code-and-cursor-9763e043a63a

### Claude Code 最新リリース: /tui コマンド・プッシュ通知ツール追加
v2.1.113〜v2.1.117 の5リリースが4/17-22に集中投下。主な新機能: `/tui fullscreen`（フリッカーフリーレンダリング）、モバイルプッシュ通知ツール（Remote Control 連携）、Vim visual mode、カスタムテーマ、hooks からの MCP tools 直接呼び出し。4/20 の品質問題修正（推論努力度・キャッシュバグ・verbosity の3件）も完了。
- https://github.com/anthropics/claude-code/releases
- https://code.claude.com/docs/en/changelog

## Microsoft 365 Copilot / Agent 365

### Agent 365 GA まで残り4日（5/1）— 日本語解説記事が充実
M365 E7（$99/user/月）と Agent 365（$15/user/月）の5/1 GA に向け、日本語コミュニティで詳細解説記事が相次いで公開。エージェントガバナンス方針の事前策定、承認フロー・監査ログの運用体制整備が推奨されている。7/1 の M365 価格改定も控えており、5/1 は導入判断の重要なタイミング。
- https://licensecounter.jp/microsoft365/blog/2026/04/microsoft365-e7.html
- https://www.ebisuda.net/tech/2026/04/11/microsoft-agent-36515-microsoft-agent-365-a-practical-guide-to-the-new-standalon/
- https://www.btncon.com/blog/m365-e7-frontier-suite

### AI in SharePoint（旧 Knowledge Agent）5月世界展開
2025年9月に Knowledge Agent としてローンチした機能がリブランド。自然言語でサイト・ライブラリ・ページ・リストを計画・構築可能。技術知識不要で SharePoint サイトの AI 活用が可能になる。5月から全世界展開開始。
- https://supersimple365.com/whats-new-in-microsoft-365-and-copilot-april-2026/

### Copilot in Word: 変更履歴追跡が Frontier/Beta Channel で利用可能
法務・財務・コンプライアンス向けに、Copilot による編集時の変更履歴追跡機能が追加。ドキュメントの完全性が求められる業務での AI 活用を促進。現在 Windows デスクトップ版（Frontier/Beta Channel）で利用可能、Web・Mac 版は近日対応。
- https://techcommunity.microsoft.com/blog/microsoft365copilotblog/copilot-in-word-new-capabilities-for-document-workflows/4508974

## Power Platform

### Power Pages: GitHub Copilot / Claude Code CLI 向け agentic code plugin に新スキル3種
Power Pages 向けの agentic code plugin に、サーバーサイドロジック対応の新スキル3種が追加。AI アシスタントによるサイト構築で欠けていたサーバーサイド機能を補完。GitHub Copilot と Claude Code CLI の両方で利用可能。
- https://www.microsoft.com/en-us/power-platform/blog/2026/04/09/whats-new-in-power-platform-april-2026-feature-update/

### Power BI April 2026 Feature Summary
モバイル In-report Copilot が強化され、オープンエンド質問・フォローアップ会話に対応。レポートのコンテキストに基づいた AI 生成ビジュアルと引用元表示が可能に。iPhone/iPad では音声入力にも対応。Direct Lake 計算列（Preview）、ベーステーマスイッチャー、旧ファイルピッカー廃止（SU04）も含む。
- https://powerbi.microsoft.com/en-us/blog/power-bi-april-2026-feature-summary/

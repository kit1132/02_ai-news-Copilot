### Office 365 コネクタ Teams 無効化 — 5/18-22 開始

## Microsoft 365 / Copilot

### Teams Efficiency Mode 展開完了（mid-May）
- ハードウェア制約デバイス向けにパフォーマンス最適化。送信カメラ解像度の動的調整、起動時リソース消費削減
- デフォルト有効（Settings > General > "Never use efficiency mode" でオプトアウト可）
- MC1287373: https://mc.merill.net/message/MC1287373

### Outlook Lite 完全廃止（5/25）
- Android 向け Outlook Lite アプリが 5/25 に完全リタイア。メールボックスアクセス不可になる
- MC1276508: https://mc.merill.net/message/MC1276508

### 共有カレンダー自動アップグレード（5月〜7月）
- Outlook/Exchange Online の共有カレンダーがレガシー MAPI モデルから REST モデルへ自動移行
- MC1287370: https://mc.merill.net/message/MC1287370

### Copilot Chat ライセンス変更 — 実施済み（5/16）
- 大規模組織で Word/Excel/PPT/OneNote から Copilot Chat 削除が完了。影響確認フェーズ

## 開発ツール

### Claude Code v2.1.143（5/15）
- **プラグイン依存関係強制**: `claude plugin disable` が依存プラグインの存在時に拒否（disable-chain ヒント付き）
- **コンテキストコスト予測**: `/plugin` marketplace にターンごと・呼び出しごとのトークン推定表示
- **Worktree 設定**: `worktree.bgIsolation: "none"` でバックグラウンドセッションが直接作業コピーを編集可能に
- **PowerShell**: `-ExecutionPolicy Bypass` がデフォルトに
- **バグ修正**: stop hooks の無限ループ、`/loop` キャンセル、バックグラウンドセッションのモデル・effort level 保持
- https://github.com/anthropics/claude-code/releases

### GitHub Copilot CLI v1.0.48（5/14）
- モデルピッカーに実トークン単価表示（AI Credits 課金対応）
- `/fork` にオプション名パラメータ追加、セッション元表示
- CJK 文字・絵文字の描画修正、`/context` のトークンリミット表示修正
- Azure DevOps 環境で GitHub MCP Server の自動無効化

### GitHub Copilot Individual プラン変更
- 新規サインアップ一時停止、使用量制限強化
- **Opus モデルが Pro プランから除外**（Pro+ のみ Opus 4.7 利用可）
- 6/1 AI Credits 課金開始に向けた段階的制限
- https://github.blog/news-insights/company-news/changes-to-github-copilot-individual-plans/

### Anthropic: Agent SDK Credits 導入
- 全有料サブスクリプションに「Agent SDK」クレジット枠を新設
- OpenClaw 等サードパーティエージェントからのプログラマティック利用が再開
- https://venturebeat.com/technology/anthropic-reinstates-openclaw-and-third-party-agent-usage-on-claude-subscriptions-with-a-catch/

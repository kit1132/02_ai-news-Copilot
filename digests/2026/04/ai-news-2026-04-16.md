# AI News Digest - 2026-04-16

## 本日の注目トピック

### Power Apps × M365 Copilot統合の新段階 — App Skills GA・MCP Server公開プレビュー
- 4/15ブログ「Making business apps smarter with AI, Copilot, and agents in Power Apps」公開
- **M365 Copilot in model-driven apps が GA**、canvas appsは Public Preview
- **App Skills**（データ入力・探索・可視化・要約）が GA。Power Apps MCP Server経由で外部エージェントにアプリ機能を公開可能

- 5/1以降、Agent FeedはPower Apps MCP Server経由のタスクのみサポート。移行期限に注意
- https://www.microsoft.com/en-us/power-platform/blog/power-apps/public-preview-power-apps-mcp-and-enhanced-agent-feed-for-your-business-applications/

## Microsoft 365 Copilot / Copilot Studio

### M365 Copilot Agent Store 大幅拡充（4/13）
- ビジネスアプリをCopilot Chat内に直接埋め込み、コンテキストスイッチなしで操作可能に
- 新規連携エージェント: **Adobe Express**（デザイン）、**Figma**（AIダイアグラム生成）、**Miro**（ボード・フローチャート）、**monday.com**（プロジェクト可視化）、**Optimizely**（キャンペーン管理）、**Box**（ファイルプレビュー）、**Coursera**（学習リソース）、**Wix/Base44**（Webサイト構築）
- https://www.microsoft.com/en-us/microsoft-365/blog/2026/04/13/bring-your-everyday-business-apps-into-the-flow-of-work-with-agents-in-microsoft-365-copilot/

### Copilot in Word — ドキュメントワークフロー新機能（4/13）
- **変更履歴の記録に対応**: 法務・財務・コンプライアンス向けに、Copilotの編集を変更履歴として追跡可能に
- Frontier プログラム（Office Insiders Beta Channel）で先行利用可能
- https://techcommunity.microsoft.com/blog/microsoft365copilotblog/copilot-in-word-new-capabilities-for-document-workflows/4508974

### M365 Roadmap更新（4/13週）
- **エージェントのTeams共有**: 作成したエージェントをTeamsチームに共有、メインチャネルに通知
- **Agent Builder → Agent Store投稿**: カスタムエージェントを「Built by your org」セクションに公開（管理者承認制）
- **PowerPoint Copilot**: スライドの作成・編集・レイアウト改善を自然言語で指示可能に。スライドコンテキストに基づくテキスト生成
- **SharePoint Knowledge Agent**: メタデータ自動化、陳腐化コンテンツ修正、Copilot対応コンテンツ品質向上
- **Teams 話者言語の自動検出**: リアルタイムで各話者の言語を検出・更新
- https://sharepointstuff.com/2026/04/14/microsoft-roadmap-roundup-13-april-2026/

## AIツール・開発環境

### Claude.ai 大規模障害（4/15）
- Claude.ai、API、Claude Codeで障害発生。**15,000人以上**に影響
- 10:53 AM ET に確認 → 11:40 AM ETに全面ダウン → **1:42 PM ETに復旧**（約3時間）
- Pro ユーザーのログイン障害、Free ユーザーの使用量トラッキングエラーが報告
- https://status.claude.com/history

### Claude Code v2.1.109（4/15）
- Extended-thinking indicator の改善（回転プログレスヒント表示）
- v2.1.108（4/14）のPrompt Caching 1h / /recap / Skill toolに続くパッチリリース
- https://code.claude.com/docs/en/changelog

### GitHub Copilot Pro トライアル一時停止（4/10〜）
- 無料トライアルシステムの不正利用急増に対応し、**新規・既存トライアルともに一時停止**
- Copilot Free および有料 Pro/Pro+ サブスクリプションは引き続き利用可能
- 改善されたセーフガード実装後に再開予定
- https://github.blog/changelog/2026-04-10-pausing-new-github-copilot-pro-trials/

### GitHub Copilot データ学習ポリシー変更 — 4/24オプトアウト期限
- **4/24以降**: Free/Pro/Pro+ユーザーのインタラクションデータ（入出力、コードスニペット、コンテキスト）をAIモデル学習に使用
- **Business/Enterpriseは対象外**。有料組織リポジトリでの作業も除外
- Settings > Privacy でオプトアウト可能。以前オプトアウト済みの場合は設定が維持される
- https://github.blog/news-insights/company-news/updates-to-github-copilot-interaction-data-usage-policy/

### GitHub Copilot — リモートCLIセッション・データレジデンシー
- **リモートCLIセッション**: Web/GitHub Mobileから実行中のCLIセッションを監視・操作可能。リアルタイム同期、フォローアップコマンド対応
- **データレジデンシー**: US/EUリージョン対応。推論処理と関連データが指定地域内に保持。GPT-5.4/Claude Sonnet 4.6/Opus 4.6等を利用可能
- https://github.com/features/copilot/whats-new

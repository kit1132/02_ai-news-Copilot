## Copilot Studio / Power Platform

### 「新しい Copilot Studio」発表 — UI刷新・AI コア再構築（6/9）
Copilot Studio が大規模リニューアル。構成タブが9→4に簡素化され、オーサリング体験が大幅に効率化。**Skills**（マークダウン形式の再利用可能な指示、GitHub Copilot / Claude Code のスキルもインポート可能）、新ワークフローデザイナー（ノード単位テスト＋バージョニング）、**Agent ノード**（ワークフロー内からエージェント呼び出し）を搭載。世界全域GA。
- https://techcommunity.microsoft.com/blog/copilot-studio-blog/meet-the-new-copilot-studio-rebuilt-for-more-complex-multi-step-work/4526488

### Copilot Studio Platform Build 2026.5.4 リリース
- 多言語音声エージェント: 非リアルタイムTTSで言語ごとに音声・速度・ピッチ設定可能
- マルチエージェント分析: インベントリエンドポイント追加（エージェント別KPI・時系列トレンド）
- エンティティ複数値抽出がデフォルト有効化
- **破壊的変更**: GPT-5 Autoがenhanced task completionエージェントで利用不可に
- Safe sharing governance: 共有・公開・実行時にコネクション共有ポリシーを強制
- https://learn.microsoft.com/en-us/power-platform/released-versions/copilotstudio/2026.5.4

### Copilot Agent Kit May 2026 リリース（旧 Copilot Studio Kit → リブランド）（6/8公開）
Power CAT チームのテスト自動化・ガバナンス支援キットが **Copilot Agent Kit** に改名。Agent Metrics Dashboard / Conversation Analyzer Dashboard を新設。Wave 2 Agent Library に5テンプレート追加（AI Learning Advisor, Status Update Agent, SME Finder, Project Delta Digest, Personal News Digest）。Agent Review Tool が CI/CD パイプライン統合に対応。

## M365 Message Center / ライセンス

### Agent 365 ライセンス前提条件変更（6/1〜）
Agent 365 の新規購入に **M365 E5** が必須前提条件に。FLW は Defender+Purview（F5レベル）、SMB は M365 Business Premium が必要。M365 E7 は既に E5 を含むため影響なし。
- https://learn.microsoft.com/en-us/partner-center/announcements/2026-june

### M365 Business with Copilot — 7/1から恒久SKU化
M365 Business Standard with Copilot（$23.50/ユーザー/月）と Business Premium with Copilot（$32/ユーザー/月）がプロモーションから恒久SKUに移行。Copilot Business 単体 $18（15%オフ、12月末まで延長）。
- https://learn.microsoft.com/en-us/partner-center/announcements/2026-june

### Work IQ API — 6/16 GA・Copilot Credits 課金
消費ベース課金（Copilot Credits）。管理者は GA 前に課金設定・アクセスポリシー・支出制限の構成が必要。Chat / Context / Tools の3ドメイン。
- https://devblogs.microsoft.com/microsoft365dev/work-iq-production-ready-intelligence-for-every-agent/

### MC1304290: M365パッケージング更新（6月中旬〜8/1）
Defender for O365 P1、Intune 改善等がスイートに追加。7/1価格改定（E3 $36→$39、Business Basic $6→$7 等）。

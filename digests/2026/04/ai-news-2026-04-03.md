# AI News Digest - 2026-04-03

## Microsoft 365 Copilot

### Researcher に Critique / Council モードが追加（Frontier プログラム）
- **Critique モード**: レポート生成モデルとレビューモデルの2つを使い、事実精度・分析深度・表現品質を改善するフィードバックループを構築。DRACO ベンチマークで Perplexity Deep Research（Claude Opus 4.6）比 +7.0pt（13.88%改善）
- **Council モード**: Anthropic モデルと OpenAI モデルを並行実行し、ジャッジモデルが両者の合意点・相違点を要約。マルチモデル合議による精度向上
- 3/30 より Frontier プログラムユーザーに提供開始。テナント管理者がモデル・機能アクセスを制御可能
- https://dataconomy.com/2026/04/02/microsoft-upgrades-365-copilot-researcher-with-new-critique-mode/

### Copilot Chat 仕様変更（4/15）に向けた動き
- 4/15 より Word/Excel/PowerPoint/OneNote での Copilot Chat アクセスが終了。有料 M365 Copilot ライセンス必須に
- Tech Community で継続的な議論。管理者向け準備期間中
- https://techcommunity.microsoft.com/discussions/microsoft365copilot/removal-of-copilot-chat-availability-in-m365-apps/4502990

### M365 Copilot 米政府クラウド拡大

- https://www.nextgov.com/artificial-intelligence/2026/04/microsoft-expands-copilot-agentic-tools-government-clouds/412588/

## AIツール・開発環境

### Cursor 3 リリース（4/2）
- **並列エージェント実行**: ローカル・worktree・クラウド・リモートSSH 環境で複数エージェントを同時実行。Agent Tabs でサイドバイサイド表示
- **Design Mode**: ブラウザ上の UI 要素を直接アノテーションしてエージェントにフィードバック
- **/best-of-n コマンド**: 同一タスクを複数モデルで並列実行し、隔離 worktree で結果を比較
- セルフホステッドクラウドエージェント対応（3/25〜）と合わせ、エンタープライズ向け機能が充実
- https://cursor.com/blog/cursor-3

### Claude Code 最新アップデート（4/1〜）
- `/powerup`: アニメーションデモ付きの対話型レッスン機能
- `CLAUDE_CODE_PLUGIN_KEEP_MARKETPLACE_ON_FAILURE` 環境変数追加（オフライン環境対応）
- `.husky` を保護ディレクトリに追加
- レートリミットダイアログの無限ループ修正、`--resume` でのプロンプトキャッシュミス修正
- Message Batches API の max_tokens 上限を Opus 4.6 / Sonnet 4.6 で 300k に引き上げ
- https://github.com/anthropics/claude-code/releases

### Devin アップデート（4/1）
- **PWA 対応**: デスクトップ・モバイルにインストール可能に（Chrome/Edge/iOS Safari）
- **セッションステータスインジケーター**: ブラウザタブの favicon にステータスドット表示（緑: 作業中、オレンジ: 待機中）
- **AskDevin 強化**: Ask / Plan モード対応、高度なコード検索で詳細・正確な回答
- https://docs.devin.ai/release-notes/overview

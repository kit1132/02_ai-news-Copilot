## 本日のトップニュース

### Copilot Cowork、iOS/Android 対応 + Skills・プラグイン拡張（5/5発表）
- Copilot Cowork がモバイル（iOS/Android）に拡張。移動中にタスクを委任し、デスクトップで結果を確認可能に。**Frontier参加者向けベータ、GA は7月予定**
- **Cowork Skills**: 繰り返しワークフローを再利用可能な指示セットとして定義。構造・トーン・プロセスを一度設定すれば繰り返し利用可能
- **プラグイン拡張**: Power BI、Viva Engage、Loop、Fabric、Dynamics 365 との統合。パートナー連携も拡大（LSEG、Miro、monday.com、S&P Global Energy が近日対応）
- https://www.microsoft.com/en-us/microsoft-365/blog/2026/05/05/copilot-cowork-from-conversation-to-action-across-skills-integrations-and-devices/

### GPT-5.5 Thinking + ChatGPT Images 2.0、M365 Copilot・Copilot Studio で利用開始
- GPT-5.5 Thinking が M365 Copilot 全体（Copilot Chat、Word、Excel、PowerPoint）にロールアウト中。深い分析・多段階推論・複雑なタスク処理を強化
- Copilot Studio では「GPT-5.5 Reasoning」として早期リリースサイクル環境で利用可能。Microsoft Foundry でも提供開始
- ChatGPT Images 2.0 も同時展開。ビジュアルクリエイティビティが Copilot 内で直接利用可能に
- https://techcommunity.microsoft.com/blog/microsoft365copilotblog/available-today-gpt-5-5-thinking-and-chatgpt-images-2-0-in-microsoft-365-copilot/4514243

## Microsoft 365 / Copilot

### Federated Copilot Connectors GA（5/5発表）
- MCP ベースのフェデレーテッドコネクタが GA。データをMicrosoft Graphにインデックスせず、ソースシステムからリアルタイム取得
- 初期パートナー: **HubSpot、LSEG、Moody's、Notion**。M365 Chat、Researcher エージェント、Excel Agent Mode で利用可能
- 管理者は M365 Admin Center から有効化・管理。ソース側の権限制御がそのまま適用
- https://learn.microsoft.com/en-us/microsoft-365/copilot/connectors/federated-connectors-overview

### Agent 365: Shadow AI 検出・Registry Sync・Windows 365 for Agents の詳細判明
- **Shadow AI 検出**: Defender / Intune 経由で未管理のローカル・クラウドエージェントを検出。初期対応は OpenClaw、GitHub Copilot CLI・Claude Code への拡張予定
- **Registry Sync（Public Preview）**: AWS Bedrock 上のエージェントを自動検出・インベントリ化。Google Cloud 接続も対応。ライフサイクル管理（開始・停止・削除）
- **Windows 365 for Agents（Public Preview、US のみ）**: エージェント専用のセキュアな Windows 環境
- **GA ローンチパートナー**: Genspark、Zensai、Egnyte、Zendesk、Kasisto、Kore、n8n
- https://www.microsoft.com/en-us/security/blog/2026/05/01/microsoft-agent-365-now-generally-available-expands-capabilities-and-integrations/

## Power Platform

### MC1298714: バックアップ保持期間 28日→7日に短縮（5/11発効）
- Production 環境のデフォルトバックアップ保持期間が **28日から7日に変更**
- **Managed Environments を有効化すれば28日まで延長可能**。管理者がコントロール
- ポイントインタイム復元ウィンドウに影響。削除済み環境には影響なし
- https://m365admin.handsontek.net/power-platform-backup-retention-production-environments-changing-28-7-days/

## AIツール（Anthropic / Claude）

### Claude Managed Agents Memory: Public Beta 開始
- Managed Agents にセッション横断の永続メモリ機能が追加。ファイルシステムベースでマウントし、既存の bash/code ツールで読み書き
- **組織・ユーザースコープ**: 全社 read-only ストア + ユーザー個別 read/write ストアの運用が可能
- 全書き込みが Claude Console のセッションイベントとして記録。バージョンロールバック・特定コンテンツの編集も対応
- https://claude.com/blog/claude-managed-agents-memory

### Claude Code v2.1.128（5/4リリース）
- `/mcp` がサーバーのツール数を表示、0ツール接続を警告。`--plugin-dir` が .zip アーカイブに対応
- `/resume` が大規模セッションで**最大67%高速化**。古い大規模セッションは再読込前に要約を提案
- 10MB超の stdin パイプ入力でのクラッシュループ修正、フルスクリーンモードでの長URL折り返しクリック修正
- https://github.com/anthropics/claude-code/releases

## 開発ツール

### Cursor: Enterprise Admin 管理機能強化 + CVE-2026-26268 修正済み（5/4）
- **モデル制御**: プロバイダー・モデル単位での allow/blocklist 設定。速度・コンテキストウィンドウサイズ別のモデル構成ブロックも可能
- **ソフトリミット**: ハードリミットに加え、50%/80%/100% で自動アラートを送信するソフトリミット設定
- **CVE-2026-26268（CVSS 8.1）**: 悪意あるリポジトリの Git hook 経由で任意コード実行が可能な脆弱性。v2.5 で修正済み（4/28 公開）
- https://cursor.com/changelog

### Devin: Enterprise 向けレビュー権限・Focus Mode 等（5/1）
- **Stacked Review Permissions**: Enterprise 管理者が PR レビューのアクセスレベルを階層化（手動のみ / PR作成時自動 / プッシュ毎自動）
- **Preview Agent Toggle**: ストリーミング思考・高速実行を有効化。安定性は限定的
- **Focus Mode**: サイドバー・ヘッダー・右パネルを非表示にした集中モード（Cmd+Shift+F）
- **Inline File Previews**: HTML/PDF/SVG をセッションサイドバー内でセキュアにレンダリング
- https://docs.devin.ai/release-notes/2026

### GitHub Copilot: プレビュー課金体験開始 + Pro から Opus 除外
- 6/1 の利用量ベース課金切替に先立ち、**5月初旬にプレビュー請求書体験を提供開始**。管理者・ユーザーが予想コストを事前確認可能
- **Pro tier から Opus モデルを除外**。Opus 4.7 は Pro+ のみ。レート制限が VS Code / CLI で直接表示されるように
- **Cloud Agent の起動が20%以上高速化**（GitHub Actions カスタムイメージで最適化）
- https://github.blog/news-insights/company-news/github-copilot-is-moving-to-usage-based-billing/

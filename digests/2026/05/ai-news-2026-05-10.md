# AI News Digest - 2026-05-10 (土)

## 開発ツール・AIエージェント

### Cursor 3.3 — PR Review・Build in Parallel・Split PRs（5/7）
Cursor 3.3にPRレビュー機能が内蔵された。レビュースレッド・コミット履歴・変更差分をIDE内で一元管理でき、PRの作成からマージまでCursor内で完結する。「Build in Parallel」機能により、プランの独立したタスクを非同期サブエージェントで並列実行可能。また「Split PRs」クイックアクションで、マルチタスク時の変更を論理単位で自動分割しPR化できる。
- https://cursor.com/changelog/05-07-26

### GitHub Copilot CLI v1.0.44（5/8）
パスコンプリーションのちらつき修正、スラッシュコマンドが入力途中でも表示可能に（複数スキルを1メッセージで呼び出し可）。`userPromptSubmitted` hookがLLMをバイパスして直接応答を返す機能を追加。`/user list`・`/user switch`の高速化、プレリリースビルド取得オプション、シェルエイリアス対応改善なども含む。
- https://github.com/github/copilot-cli/releases

### Claude Code v2.1.137〜v2.1.138（5/9）

- v2.1.138: 内部修正

# AI News Digest - 2026-03-24

## Windows Copilot 統合縮小

- **MicrosoftがWindows 11のCopilot統合を縮小**。Photos、Widgets、Notepad、Snipping ToolからCopilot機能を削除し、Settings・File ExplorerへのCopilotブランディング拡張計画も撤回。ユーザーからの「AI過剰」批判（"Microslop"）やプライバシー懸念への対応。Pavan Davuluri EVPは「本当に有用でよく作られた体験に集中する」と発言。Insiderプレビューで3-4月に順次反映予定
- https://techcrunch.com/2026/03/20/microsoft-rolls-back-some-of-its-copilot-ai-bloat-on-windows/
- https://www.theregister.com/2026/03/23/windows_quality_commitment/

## Power Platform 3月フィーチャーアップデート（3/19公開）

- **Power Platform Inventory GA**: テナント管理者向けに、クラウドフロー・Copilot Studioエージェントフロー・Workflowsを横断した統合ビューが一般提供開始。コネクタ・アクション・利用データの追加拡張も予定
- **新しいUsageページ（パブリックプレビュー）**: Power Apps・Power Automate・Copilot Studioの採用トレンドとリソースレベル分析のモダンダッシュボード
- **Agent API for Power Pages**: サイト作成者がカスタムチャットUIを構築し、Copilot Studioエージェントとシームレスに統合可能
- **Power Pagesプラグイン for GitHub Copilot CLI & Claude Code（パブリックプレビュー）**: 自然言語でサイトを記述するとスキャフォールディング・Web API統合・権限設定・デプロイまで自動処理
- https://www.microsoft.com/en-us/power-platform/blog/power-apps/whats-new-in-power-platform-march-2026-feature-update/

## Copilot Studio 新機能

- **Copy agents from M365 Copilot to Copilot Studio**: M365 Copilotで作成したエージェントをCopilot Studioに移行し、マルチステップワークフロー・カスタム統合・広範なデプロイが可能に
- **(Preview) Request for information action**: エージェントフローを一時停止し、Outlook経由でレビュアーから情報収集後、動的パラメータとして再開
- **(Preview) Work IQ tools**: M365 CopilotとエージェントをWork IQサービスに接続し、ファイル・メール・会議・チャットからリアルタイム業務インサイトにアクセス
- **Client SDK（Android/iOS/Windows）**: ネイティブアプリにCopilotエージェントを埋め込み、マルチモーダル対話を実現
- **Code Interpreter GA**: 自然言語からPythonコードベースのアクションを生成する機能が一般提供
- https://learn.microsoft.com/en-us/microsoft-copilot-studio/whats-new

## ガバナンス・セキュリティ

- **DLPポリシーがCopilot × 感度ラベル付きファイルに適用**: Word/Excel/PowerPointの感度ラベル付きファイルへのCopilot処理をDLPポリシーでブロック可能に。ローカルデバイス含む全保存先対象。3月末ロールアウト開始

- **エージェント所有権の再割り当て**: 管理者が共有エージェントの所有権を移管可能に
- https://learn.microsoft.com/en-us/microsoft-copilot-studio/whats-new

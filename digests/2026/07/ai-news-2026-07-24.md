# AI News Digest — 2026-07-24（金）

Microsoft 一次は全据え置き（CS What's New は June、Release Notes は 7/15 バッチ、Released Versions は 2026.6.3）。本日の主役は**昨日予告した OpenAI サブプロセッサー（GPT-5.6）の既定有効化が本日発効**したこと。公式 Learn ページで一次確認でき、適用範囲・除外事項が判明した。AI 開発ツールは **Claude Code v2.1.218**、Anthropic は **Claude Opus 4.7 の Fast モードを本日提供終了**。

## テナント管理・ガバナンス（本日発効・要対応）

- **OpenAI 運用モデル（GPT-5.6 含む）のサブプロセッサー既定有効化が本日 2026-07-24 に発効**（MC1422074・opt-out 制）。公式 Learn ページで一次確認。判明した要点:
  - **適用範囲は M365 Copilot・M365 アプリ内 Copilot・Copilot Studio の 3 レイヤー**（Copilot Studio ではエージェント作成時にモデルを選択。GPT-5.6 は Word/Excel/PowerPoint/Copilot Chat/Cowork の優先エンジンとして展開）。**Azure OpenAI（Microsoft 運用の OpenAI モデル）には非該当**
  - opt-out は M365 admin center →「AI providers operating as Microsoft subprocessors」→ OpenAI →「No users」を選択（AI 管理者/グローバル管理者ロールが必要）。**Copilot Studio/Power Platform では M365 側で有効化後、Power Platform admin center 側の追加制御（外部 LLM 許可設定）も必要**
  - **制約**: 政府/ソブリンクラウド（GCC/GCC High/DoD）は対象外、in-country processing コミットメントの対象外、ただし EU Data Boundary には含まれる。**除外事項**: FedRAMP High 未認証・PCI DSS AOC 未提供・HITRUST CSF 未提供・SOC 1 Type 2 未提供 → 規制業種は要確認
  - https://learn.microsoft.com/en-us/microsoft-365/copilot/openai-subprocessor
  - https://mc.merill.net/message/MC1422074

## AI開発ツール

- **Claude Code v2.1.218**（7/22 の 2.1.217 から前進）: ① **`/code-review` をバックグラウンドのサブエージェントとして実行**（レビュー出力で会話が埋まるのを防止）、② `--ax-screen-reader` モードでキーボードショートカットの削除操作を読み上げ、③ **Windows でパスに `\u` プレフィックス区間があるとファイルアクセス不能になる不具合を修正**、④ MCP 接続診断に HTTP ステータスコード/エラー詳細を表示、⑤ 改行を Ctrl+J でエンコードする端末での複数行ペースト対応、⑥ compaction 後のトークン使用量表示が古くなる不具合を修正、⑦ Bedrock の application-inference-profile ARN のスペンド計測を改善
  - https://raw.githubusercontent.com/anthropics/claude-code/main/CHANGELOG.md
- **Anthropic — Claude Opus 4.7 の Fast モードを本日 7/24 提供終了**: 提供終了後は `claude-opus-4-7` へ `speed: "fast"` 指定するとエラーを返す。**Claude Opus 4.8 の Fast モードへ移行が必要**。7/23 掲載の Claude for Teachers 提供開始・Developer Platform の Sonnet 5（1M コンテキスト）は継続
  - https://platform.claude.com/docs/en/release-notes/overview

## Copilot Studio / Power Platform

### 1. Copilot Studio CLI GA: 5/31に迫る
Power Platform CLI拡張としてCopilot Studio CLIが5/31にGA予定。エージェント定義のデプロイ・更新をコマンドラインから実行でき、CI/CDパイプラインへの組み込みが可能になる。
- https://learn.microsoft.com/en-us/microsoft-copilot-studio/whats-new

### 2. Question Themes Analysis GA: 5/29予定
エージェントへのユーザー質問パターンを自動分類・可視化するQuestion Themes Analysisが5/29にGA予定。カスタムメトリクスと組み合わせて運用分析を強化。
- https://learn.microsoft.com/en-us/power-platform/release-plan/2026wave1/microsoft-copilot-studio/planned-features

## Microsoft 365 Copilot

### 3. Copilot Calendar Agent ロールアウト
自然言語でカレンダー管理ルールを設定し、Copilotがスケジュールを自動管理するCalendar Agent機能が展開中。Classic OutlookへのSmart Calendar統合と連動。
- https://supersimple365.com/whats-new-in-microsoft-365-and-copilot-may-2026/

### 4. Copilot Capacity Packs 導入
月25,000メッセージのプリペイドCapacity Packsが利用可能に。PPACでの使用量モニタリングと組み合わせ、従量課金前のコスト予算管理を強化。
- https://changepilot.cloud/blog/may-2026-top-10-microsoft-365-message-center-roadmap-updates

### 5. Anthropic Claudeモデル M365アプリでデフォルト有効化
Word/Excel/PowerPointでのAnthropic Claudeモデルが、対象テナントでopt-inからデフォルト有効に変更。従来のオプトイン方式からの転換。
- https://changepilot.cloud/blog/may-2026-top-10-microsoft-365-message-center-roadmap-updates

### 6. 共有エージェント所有権移管
管理者が共有エージェントの所有権を別ユーザーに移管可能に。新オーナーに編集・削除権限を付与し、旧オーナーのアクセスを無効化する。
- https://changepilot.cloud/blog/may-2026-top-10-microsoft-365-message-center-roadmap-updates

## 開発ツール

### 7. GitHub Copilot Individual plan 変更加速
Pro/Pro+/Studentプランの新規サインアップを一時停止。ProプランからOpusモデルを削除（Pro+では引き続きOpus 4.7利用可）。6/1の従量課金（AI Credits）移行に向けた準備段階。
- https://github.blog/news-insights/company-news/changes-to-github-copilot-individual-plans/

### 8. GitHub Copilot CLI v1.0.52-53（5/23-24）
v1.0.52: コンテキストウィンドウティア選択（~200K vs 1M）をend-to-endで適用。セッション再開時に保存ディレクトリを復元。垂直スクロールバー追加。v1.0.53: マルチラインプロンプト表示修正、`/skills`ピッカーの`--config-dir`対応。
- https://github.com/github/copilot-cli/releases/tag/v1.0.52

### 9. Devin 2.2: Computer Useで自己検証 + Auto-Triage
Devin 2.2でLinuxデスクトップ上のアプリ実行・テストが可能に（スクリーンレコーディング付き）。コード→テスト→修正のクローズドループを実現。Auto-TriageはDatadog/Sentry/PagerDutyからバグ・インシデントを自動検知・調査し、PRまたはコンテキストレポートを生成。
- https://cognition.ai/blog/introducing-devin-2-2
- https://cognition.ai/blog/auto-triage

### 10. Claude Code v2.1.149-150（5/22-23）
v2.1.149: `/usage`コマンドがカテゴリ別利用状況（skills/subagents/plugins/MCP）を表示。`/diff`でキーボードスクロール対応（j/k/PgUp/PgDn）。GFMタスクリストレンダリング。PowerShell `cd`バリアント経由のサンドボックス回避を修正。v2.1.150: 内部インフラ改善。
- https://github.com/anthropics/claude-code/releases

### 11. Cursor: Composer 2.5 + Bugbot従量課金移行
Composer 2.5が長時間タスクでの指示追従を改善（$3.00/M input、$15.00/M output）。Bugbotが$40/席/月のサブスクから使用量課金に移行し、エフォートレベル（Default/High/Custom）をチーム管理者が設定可能に。Automationsにマルチリポ環境対応追加。
- https://cursor.com/changelog

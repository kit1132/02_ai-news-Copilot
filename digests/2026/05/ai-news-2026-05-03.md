# AI News Digest: 2026-05-03 (土)

## 明日GA予定のリリース

### Power Apps Agent Feed + MCP Server GA（5/4）
- Agent Feedがモデル駆動型アプリ内でGA。エージェントのアクティビティをアプリ内で直接監視・承認するUIを提供
- **5/1以降、Agent FeedはPower Apps MCP Server経由のエージェントのみサポート**。移行未完了の場合は即日対応が必要
- 低リスク操作はバックグラウンド実行、高リスク操作（メール送信等）は明示的な承認フローとして表示
- https://learn.microsoft.com/en-us/power-apps/maker/model-driven-apps/power-apps-mcp-server

### Flex Routing Power Platform環境拡張（5/4）
- Power Platform環境のFlex Routing拡張が5/4展開予定
- https://learn.microsoft.com/en-us/power-platform/release-plan/2026wave1/

## Microsoft 365 / Copilot

### M365 Message Center 新規通知
- **MC1282308**: Message Center投稿の見出し構造が5/16から標準化（「What and Why」「Rollout Schedule」等）。スクリプト・自動化の修正が必要な場合あり
- **MC1189663**: アクション可能メッセージの外部アクセストークンが5/15廃止。Microsoft Entra認証への移行必須
- **MC1188599**: OneDrive/SharePointのアクセス要求・拒否ページがモダンUIにリニューアル（5月展開）
- **MC1266027**: Teams会議ウィンドウ最小化時に挙手・リアクション可能な新体験（5月中旬）
- https://mc.merill.net/

### Dataverse Purview監査イベント変更
- 5月以降、DataverseからMicrosoft Purviewに送信される監査イベントに前後のフィールド変更値が含まれなくなる
- Purview連携で監査証跡を活用している組織は影響確認が必要
- https://learn.microsoft.com/en-us/power-platform/important-changes-coming

## AIツール・開発

### Devin UI改善（5月初旬）
- **Preview upcoming features トグル**: エージェントセレクターからストリーミングthoughts・高速実行を先行体験可能に
- **インラインファイル表示**: HTML/PDF/SVGをセッションサイドバーで直接レンダリング（コード/レンダー切替・ダウンロード対応）
- **Focus mode**（Cmd+Shift+F）: サイドバー・ヘッダー・右パネルを非表示にした集中モード
- **Agentsタブ**: 子セッション作成時に自動表示、各子セッションのステータス・TODO・PRを一元管理
- https://docs.devin.ai/release-notes/2026

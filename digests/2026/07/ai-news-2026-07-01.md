# AI News Digest — 2026-07-01

## 開発ツール

### Claude Code 2.1.197（6/30）— Sonnet 5 を既定モデル化、1Mコンテキスト
- **Claude Sonnet 5 が Claude Code の既定モデルに**。ネイティブ1Mトークンコンテキスト、プロモ料金 $2/$10 per Mtok（〜8/31）。利用には更新が必要
- Fable 5/Mythos 5 が輸出規制で停止中の文脈で、可用な Sonnet 5 へ既定を引き上げた形。長文リポジトリ全体を1コンテキストに載せる用途に直結
- ソース: https://github.com/anthropics/claude-code/blob/main/CHANGELOG.md / https://www.anthropic.com/news/claude-sonnet-5

### Claude Code 2.1.196（6/29）— 組織既定モデル・MCPセキュリティ強化
- 組織管理者が既定モデルを設定可能（「Org default」「Role default」表示）。可読なセッション名、チャット内ファイル添付のクリック展開
- セキュリティ: `claude mcp list`/`get` が自己承認リポジトリの `.mcp.json` サーバーを起動しないよう修正。ストリーミングのアイドル監視（5分無応答で中断・再試行）が全プロバイダーで既定ON
- ソース: https://code.claude.com/docs/en/changelog

### GitHub Desktop 3.6（6/26）— Copilotがコミット作成・コンフリクト解消を担当
- Copilotがコミットメッセージ生成（`.github/copilot-instructions.md`/`AGENTS.md` 反映）とマージコンフリクトのAI解消を担当。Git worktree対応、モデルピッカー＋BYOK
- GitHub Copilot for Jira が GA（モデル選択、Confluenceコンテキスト=MCP、カスタムエージェント）
- ソース: https://github.blog/changelog/2026-06-26-github-desktop-3-6-worktrees-and-deeper-copilot-integration/

## Copilot Studio / エージェント

### Computer Use が全商用リージョンでGA
Copilot Studioの Computer Use 機能が一般提供開始。エージェントがブラウザやデスクトップアプリのUIをビジョンベースで直接操作可能に。セレクタベースの脆弱な自動化ではなく、レイアウト変更にも適応する。Power Platform admin center経由でガバナンス・監視を一元管理。
- https://techcommunity.microsoft.com/blog/copilot-studio-blog/computer-using-agents-in-microsoft-copilot-studio-are-now-generally-available/4519427

### リアルタイム音声エージェントの運用ガイド公開（5/19）
Copilot Studio公式ブログに「The in-depth guide to managing real-time voice agents at scale」掲載。5段階ガバナンスフレームワーク（ビルダー→ビルド→リリース→ランタイム→ライフサイクル）を提示。エスカレーション設計をファーストクラス要件として位置付け。
- https://www.microsoft.com/en-us/microsoft-copilot/blog/copilot-studio/the-in-depth-guide-to-managing-real-time-voice-agents-at-scale/

### Agent 365 "What's New" May 2026
Agent 365の概要ダッシュボード（登録エージェント数・アクティブユーザー・リスクシグナル等のリアルタイム指標）が利用可能に。AWS Bedrock/Google Cloud との Registry Sync で、マルチクラウドのエージェントを自動検出・インベントリ管理。Microsoft Defender によるエージェント脅威のランタイム検出・ブロック。6月にContext mapping、ポリシーベース制御、Intune/Defender Public Previewが予定。
- https://techcommunity.microsoft.com/blog/agent-365-blog/what%E2%80%99s-new-in-agent-365-may-2026/4516340

## Microsoft 365 Copilot

### Copilot Chat ライセンス変更 5/16 実施
M365 Copilotライセンスなしユーザーは「Copilot Chat (Basic)」に移行。Word/Excel/PowerPoint/OneNote内のフルCopilot体験（高度な推論・モデル選択）はライセンス保有者限定に。ライセンスなしでもアプリ内Copilotは残るが「standard access」扱い（品質はキャパシティ依存）。
- https://learn.microsoft.com/en-us/microsoft-365/copilot/release-notes

### Researcher に Critique & Council 機能
Researcher がマルチモデル対応。**Critique**: GPTが生成→Claudeがレビュー（またはその逆）で品質向上。**Council**: GPTとClaudeの回答を横並び比較、合意点・相違点・各モデル固有の洞察を表示。Frontierプログラムで利用可能。
- https://techcommunity.microsoft.com/blog/microsoft365copilotblog/introducing-multi-model-intelligence-in-researcher/4506011

### Workday Sana Self-Service Agent がM365 Copilotに統合
Copilot Chat内からWorkdayのHR・財務タスクを直接実行。休暇残日数の確認→申請、経費ステータス確認、人事評価の準備タスク表示などが自然言語で完結。Microsoft Marketplace からアプリを有効化するだけで導入可能（追加ライセンス不要）。
- https://newsroom.workday.com/2026-05-13-Workday-Brings-Sana-Self-Service-Agent-for-HR-and-Finance-Into-Microsoft-365-Copilot

## Power Platform

### Power Platform May 2026 Feature Update（5/14公開）
主要な変更点:
- **Power Fx User Defined Types (UDTs)**: GA。v3.26044以降、新規アプリでデフォルト有効。複合データ構造の関数間受け渡しが可能に
- **MCP対応カスタムツール & リッチUI**: Public Preview。model-driven appでMCP駆動のFluent UIウィジェット
- **InfoPath→Canvas App移行**: Canvas Authoring MCP ServerとPowerCAT Skillで、GitHub Copilot/Claude Code等からAI支援マイグレーション
- **Data Grid Control**: Public Preview。検索バー、ソート、複数行選択、カラム型設定
- **Generative Pages入力サポート**: GA。コマンドバーやグリッドアクションからコンテキスト付きで起動可能
- **Self-Healing デスクトップフロー**: Preview。手動介入なしで失敗を自動修正
- **Record with Copilot**: Preview。自然言語+自動記録でフロー作成
- **Copilot-readyテンプレート**: 新規環境作成時にAI Builder credits・コネクタ・サンプルデータを自動プロビジョニング
- https://www.microsoft.com/en-us/power-platform/blog/2026/05/14/whats-new-in-power-platform-may-2026-feature-update/

## AIコーディングツール

### Claude Code v2.1.144（5/19）・v2.1.143（5/16）
**v2.1.144**: `/resume`でバックグラウンドセッション（`--bg`/agent view起動）を一覧表示。`/model`がセッション限定に変更（`d`キーでデフォルト設定）。「extra usage」→「usage credits」にリネーム。API到達不能時のスタートアップハング修正（75s→15sタイムアウト）。

**v2.1.143**: プラグイン依存関係の強制（`disable`時に依存プラグインがあれば拒否）。`/plugin`マーケットプレイスにターンあたり/呼び出しあたりのトークンコスト予測表示。`worktree.bgIsolation: "none"`オプション追加。

**週間リミット50%増量（〜7/13）**: Pro/Max/Team/Enterprise対象。5/6の時間リミット倍増に続く3回目のキャパシティ介入。
- https://github.com/anthropics/claude-code/releases

### GitHub Copilot: Pro/Pro+サインアップ一時停止 & Opus削除
Pro/Pro+/Studentの新規サインアップ停止中。Pro tierからOpusモデル削除（Pro+ではOpus 4.7継続）。5/20までに解約すれば全額返金。Copilot CLI v1.0.48（5/14）でトークン価格表示対応。6/1の使用量課金移行が迫る。
- https://github.blog/news-insights/company-news/changes-to-github-copilot-individual-plans/

### Cursor 3.4（5/13）& Composer 2.5
**Cursor 3.4**: クラウドエージェント開発環境。マルチリポ環境、Dockerfileベースセットアップ（ビルドシークレット対応）、ガバナンス・監査ログ・環境レベルセキュリティ制御。

**Composer 2.5**: 長時間タスクと複雑な指示への対応強化。合成タスクでのトレーニング強化。初週は利用量2倍。

**Bugbot**: 席料廃止→使用量課金に移行。エフォートレベルをカスタマイズ可能に。
- https://cursor.com/changelog

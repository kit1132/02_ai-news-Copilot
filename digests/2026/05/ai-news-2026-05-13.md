# AI News Digest - 2026-05-13

## Anthropic / Claude

### Claude Platform on AWS — 一般提供開始（5/11）
AnthropicのClaude PlatformがAWS上でGA。AWSアカウント経由でClaude APIのフル機能セットを利用可能に。AWSが初のクラウドプロバイダー対応。
- **認証・課金**: AWS IAM認証、CloudTrail監査ログ、AWS Marketplace請求（既存コミットメント充当可）
- **対応機能**: Claude Managed Agents（beta）、Advisor Strategy（beta）、Web Search/Fetch、コード実行、Files API（beta）、Skills（beta）、MCP Connector（beta）、プロンプトキャッシング、Citations、バッチ処理
- **リージョン**: 東京含む世界17リージョンで提供。データ処理はAnthropicが実施（AWSセキュリティ境界外）
- Amazon Bedrock上のClaudeは引き続き利用可能（棲み分け）
- https://aws.amazon.com/about-aws/whats-new/2026/05/claude-platform-aws/

### Claude Code v2.1.139 — Agent View リリース（5/12）
複数のClaude Codeセッションを1つのCLIダッシュボードから管理する新機能「Agent View」がリサーチプレビューとして提供開始。
- `claude agents` コマンドで全セッション一覧を表示。各行にセッションID・入力待ち状態・最終応答・タイムスタンプを表示
- エージェントをバックグラウンドに送り、必要な時だけ復帰する非同期ワークフローが可能に
- Pro / Max / Team / Enterprise / API プランで利用可能
- https://code.claude.com/docs/en/agent-view

## Power Platform

### Power Apps Blog: MCP Serverにクローズドループ学習を導入（5/12）
「Power Apps MCP server introduces closed-loop learning for enterprise agents」が公開。エージェントが組織の業務パターンを学習する仕組みを導入。
- Agent Canvas上でユーザーが修正を加えると、データ入力ツールが修正内容に基づき改善（フィードバックループ）
- 早期導入・積極的修正を行う組織ほどエージェント精度が向上する先行者優位の設計
- Agent Feed（5/4 GA済み）との連携で、低リスク操作は自動実行・高リスク操作は人間承認のフローを制御可能
- https://www.microsoft.com/en-us/power-platform/blog/power-apps/

## Microsoft 365 Copilot

### Planner Agent — Copilot Chat内でタスク管理が可能に
Copilot Chat内にPlanner統合が展開中。チャットから直接タスク作成・期日確認・チーム割り当て確認が可能。
- 自然言語でタスク作成（単一タスク・複数タスク一括・メール/チャット/ファイルからのグラウンディング作成）
- 既存タスクの編集（タスク名・ステータス・日付・優先度・プラン変更）もCopilot Chat内で完結
- https://techcommunity.microsoft.com/blog/plannerblog/planner-agent-brings-work-management-directly-into-microsoft-365-copilot/4511720

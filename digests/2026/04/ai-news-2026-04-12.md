# AI News Digest - 2026-04-12

## Claude for Word 公開ベータ GA — M365への本格侵攻（4/11）

Anthropicが **Claude for Word** をPublic Betaで公開。Microsoft Wordのネイティブサイドバー アドインとしてTeam / Enterpriseプラン向けに提供（Mac/Windows両対応、Microsoft Marketplace経由でインストール）。Researcher の Claude 二段階レビュー（Critique Mode）に続き、M365全体へのClaude展開が加速している。

- **機能**: 選択テキストの書き換え、インライン Wordコメントへの応答、セクション要約、文書の事実・文体整合性の監査、.docxファイルの直接編集
- **クロスアプリ統合**: Claude for Excel / Claude for PowerPoint と直接連携。**単一の会話スレッドで Word / Excel / PowerPoint の3文書を横断** できる。Wordレポートとその裏付けExcelモデル間のデータ整合チェック、Word原稿とPowerPointスライドの文言整合等が1つのセッションで実行可能
- **法務領域が主戦場**: Artificial LawyerがAnthropicの「ネイティブレッドライン」機能を法律事務所向けに強調。M365 Copilot Researcher の Critique Mode（GPT→Claude 二段階）と合わせ、Microsoftは「Word内に Claude（Anthropic）と Copilot（OpenAI/Microsoft）が共存する」マルチモデル時代に正式突入
- **制約**: ベータ段階。最終クライアント成果物・法的提出書類・機密情報では人間によるレビュー必須

## Claude Code v2.1.101 — /team-onboarding と企業TLS対応（4/10）

v2.1.100 に続いて v2.1.101 がリリース。エンタープライズ環境での導入を後押しする変更が中心。

- **`/team-onboarding`**: ローカルのClaude Code使用状況から **チームメイト向けramp-upガイドを自動生成** するコマンド。オンボーディング資料作成の工数削減
- **OS CA証明書ストアをデフォルトで信頼**: 企業TLSプロキシ配下で追加設定なしに動作。バンドルCAのみを使う場合は `CLAUDE_CODE_CERT_STORE=bundled`
- **`/ultraplan` / リモートセッション**: Web設定を先に済ませる必要がなく、**デフォルトのクラウド環境を自動作成**
- **Bug fix**: POSIX `which` フォールバックのコマンドインジェクション脆弱性修正、長時間セッションのメモリリーク修正、Bedrock SigV4認証失敗修正、サブエージェントのMCPツール継承問題修正、5分ハードコード タイムアウト修正
- **その他**: Brief モードのリトライ、tool-not-available エラーの理由表示、レート制限メッセージに該当リミットとリセット時刻表示、`OTEL_LOG_USER_PROMPTS` など OTEL環境変数対応

- https://github.com/anthropics/claude-code/releases/tag/v2.1.101

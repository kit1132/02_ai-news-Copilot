# AI News Digest: 2026-06-01 (日曜)

## GitHub Copilot: 従量課金（UBB）本日開始

GitHub Copilot全プランが本日6/1よりUsage-Based Billing（AI Credits制）に移行。月額Pro/$10で1,000クレジット、Pro+/$39、Business/$19/user、Enterprise/$39/user。コード補完・Next Edit Suggestionsは消費対象外。月額契約ユーザーは自動移行、年額契約は期間満了まで旧体系。開発者コミュニティでは900以上のDownvote・400件超のコメントと反発が大きい。エージェントセッション1回で$30-40消費するとの試算も。

- Code ReviewもGitHub Actions分数を消費する形に変更
- Copilot Max（高リミット）は既存ユーザーのアップグレードのみ提供開始

https://github.blog/news-insights/company-news/github-copilot-is-moving-to-usage-based-billing/

## Anthropic: 6/15課金分離 & モデル廃止予告

6月15日よりClaude Agent SDK・`claude -p`・Claude Code GitHub Actions・サードパーティエージェントの利用が、サブスクリプション枠から分離され独立クレジットプールへ移行。Pro=$20/月、Max 5x=$100/月、Max 20x=$200/月。API従量課金レート適用、繰越なし。

- 対話的利用（Claude.ai、Claude Codeターミナル操作、Claude Cowork）は従来通りサブスク枠内
- Claude Sonnet 4（claude-sonnet-4-20250514）とClaude Opus 4（claude-opus-4-20250514）が6/15にAPI廃止。Sonnet 4.6 / Opus 4.7への移行推奨

https://codersera.com/blog/anthropic-june-2026-billing-change-claude-code/

## M365 Copilot: 新デザイン6月ロールアウト開始

- **Purview DLP**: Copilot Chat Webサーチ内の機密データ保護が6月Worldwide GA

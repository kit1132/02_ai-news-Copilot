# AI News Digest — 2026-07-02

昨日（7/1）ダイジェストは Fable 5 を「一般向け停止継続」と報じたが、その後 **輸出規制が解除され Fable 5 が全世界復帰**。約3週間のサーガが決着した。Microsoft 側は Copilot Studio 基盤に新ビルド 2026.6.3 が着地。

## 🔴 最重要

### Anthropic 輸出規制サーガ終結 — Fable 5 / Mythos 5 が復活
米商務省が **6/30 に Fable 5 / Mythos 5 への輸出規制を解除**。Fable 5 は **7/1 に全世界で復帰**（Claude Platform / Claude.ai / Claude Code / Claude Cowork）。**7/7 まで**は Pro / Max / Team / Enterprise で週次利用上限の **最大50%**、以降は usage credits 経由。
- **安全対策**: 再訓練した分類器が Amazon 報告の技法を **99%超ブロック**。ブロック時は要求を自動で **Opus 4.8 にフォールバック**（トレードオフとしてコーディング/デバッグの benign リクエスト誤検知が増加）
- Mythos 5 は 6/26 承認済みの米国組織（Annex A）に加え今回さらに再展開
- 6/9 リリース → 6/12 停止（Amazon のジェイルブレイク報告が発端）から続いた騒動が収束。**昨日 7/1 時点の「停止継続」報告を更新するもの**

- https://www.coindesk.com/tech/2026/07/01/anthropic-restores-ai-models-fable-mythos-after-the-u-s-lifts-export-controls

## 🟠 Microsoft 公式

### Copilot Studio 基盤 Build 2026.6.3（6/30 初出）
昨日は「2026.6.2 が最新・新ビルドなし」と報じたが、Preview(US)/仏/瑞/韓/独/加/印/北欧/南米へ **2026.6.3** が展開開始。
- **新機能**: エージェントの作成/更新/一覧/取得/削除＋コンポーネント取得用の **Power Platform API 新エンドポイント**。エージェント運用分析に per-agent 経時メトリクス（健全性を「エラー率」で表示）。**GPT-5 Mini fine-tuned モデルをデフォルト有効化**。GCC / High でエージェントインベントリをデフォルト有効化
- **破壊的変更**: Skill エージェントが未サポートの参照ファイル形式をブロック（従来受理していた形式を拒否）
- **セキュリティ**: エージェントユーザー未設定のエージェントが Graph 等の制限リソースのトークンを取得できないよう防止。コンポーネント取得の認可エラーが HTTP 500 → **403 Forbidden** に変更。Agent365 監査ログが同意プロバイダーチェック不要に（テナント単位オプトアウト可）
- https://learn.microsoft.com/en-us/power-platform/released-versions/copilotstudio/2026.6.3

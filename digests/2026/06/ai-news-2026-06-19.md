### GitHub Copilot デスクトップアプリ GA（6/17）
GitHub Copilotアプリが **macOS / Windows / Linux で正式GA**。エージェント駆動開発のデスクトップ拠点という位置づけ。
- **Canvases**: 計画・PR・ターミナル・ブラウザセッションをエージェントと双方向で共有し、作業を可視化・操作可能にする面
- **並列セッション**: リポジトリ横断でブランチ/worktree単位の並行実行
- **Cloud Automations**: マシン起動に依存しないクラウド上の定期エージェント実行
- Issue/PR/プロンプトからセッション開始 → 差分レビュー → 既存チェック/マージ要件でPR作成、まで一気通貫
- 管理者向け: `disableBypassPermissionsMode` で権限スキップを無効化するエンタープライズ制御を追加
- https://github.blog/changelog/2026-06-17-github-copilot-app-generally-available/

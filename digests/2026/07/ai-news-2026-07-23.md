# AI News Digest — 2026-07-23（木）

Microsoft 一次の据え置きが続く中、本日は実装レイヤーで動きが 3 件。**M365 Copilot 拡張機能の What's New に「July 2026」節が追加**（宣言型エージェント manifest 1.8）、テナント管理者向けに**OpenAI モデル（GPT-5.6）のサブプロセッサー化が 7/24 に既定有効化**（MC1422074・opt-out 制）、AI 開発ツールは **Claude Code v2.1.217**。

## Microsoft 365 Copilot（拡張開発・新規）

- **宣言型エージェント manifest 1.8 が公開**（拡張機能 What's New に「July 2026」節が新規追加。前回 7/16 確認時は May が最新だった）: ① `EmailActions` 機能（メールのトリアージ・監督付き送信・削除・受信ルール・自動応答・フォルダー管理の書き込み操作）、② `MeetingActions` 機能（会議のスケジュール・時間調整投票・時間インサイト）、③ ワーカーエージェントを `id` の代わりにファイルパス（`file` プロパティ）で参照可能に、④ 埋め込みナレッジに `embedded_resource_snapshot_id` 追加。あわせて Copilot 利用状況レポート API 3 種に `version` パラメーターが追加
  - https://learn.microsoft.com/en-us/microsoft-365/copilot/extensibility/whats-new

## テナント管理・ガバナンス（要対応・期日 7/24）

- **MC1422074: OpenAI モデル（GPT-5.6 含む）が M365 Copilot のサブプロセッサーとして利用可能に**。設定は既定で無効だが、管理者が opt-out しなければ **2026-07-24 に自動的に有効化**される。M365 admin center でいつでも確認・変更可。データ処理のサブプロセッサー範囲に関わるため、ガバナンス方針を持つテナントは 7/24 までに設定確認を推奨
  - https://mc.merill.net/message/MC1422074
- **Power Platform ライセンス（7月版）補足**: Copilot Credits の購入方法の推奨順が Pay-As-You-Go 優先に整理。**Copilot Studio エージェントによる Work IQ API 利用は M365 Copilot ライセンスに含まれず Copilot Credits を消費する**点が明記（コスト設計時の注意）

# AI News Digest — 2026-05-01

## 最重要: Agent 365 / M365 E7 本日GA

### Microsoft Agent 365 — 本日一般提供開始
- **Agent 365**（$15/user/月）と **Microsoft 365 E7 Frontier Suite**（$99/user/月）が本日5/1にGA
- E7 は M365 E5 + M365 Copilot + Entra Suite + Agent 365 を統合した初の新エンタープライズ層（E5以来11年ぶり）
- Agent 365 はエージェントの **Observe / Govern / Secure** 三本柱のコントロールプレーン
  - Observe: 全エージェントの集中レジストリ、OpenTelemetry ベースのテレメトリ
  - Govern: IT承認済みエージェントブループリント（権限・DLPポリシー・監査要件を定義）
  - Secure: Purview 統合、エージェント活動の監査証跡
- **マルチベンダー対応**: OpenAI / Anthropic Claude / ServiceNow / Workday / LangChain 等のエージェントを一元管理
- GA時点では**ライセンスユーザー代理エージェントのみ**。自律エージェント（Agent Identity Authentication — 独自メールボックス・OneDrive を持つエージェント）は Frontier Preview 継続、トライアルライセンスは12月まで延長
- 販売チャネル: EA / EAS / CSP / MCA
- https://www.microsoft.com/en-us/microsoft-agent-365
- https://trustmarque.com/microsoft-365-e7-agent-365-whats-launching-1-may-and-what-it-means

## 最優先: M365 Copilot / Copilot Studio / Power Platform

### 1. Copilot Chat ライセンス変更 実施済み（5/16）
- **大規模組織（2,000+ユーザー）**: Word/Excel/PowerPoint/OneNoteで未ライセンスユーザーのCopilot Chat利用が不可に
- **小規模組織（2,000未満）**: standard accessに格下げ。品質・パフォーマンスはサービスキャパシティ依存、アップグレード促進UI表示
- **ラベリング変更**: Copilot Chat (Basic) / M365 Copilot (Premium) の二層構造
- Outlookのメール・カレンダーアシスタンスは引き続きライセンス不問で利用可能
- https://learn.microsoft.com/en-us/copilot/microsoft-365/microsoft-365-copilot-licensing

### 2. Office 365コネクタ Teams無効化 開始（5/18〜22）
- **本日5/18より段階的に無効化開始**。5/22までに全テナントで完了予定
- Workflows webhook への移行が必須。MessageCardペイロード・プライベートチャネル対応済み
- Secure Future Initiative（SFI）に基づくセキュリティ強化施策
- 対話型ボタンが必要な場合はAdaptive Cardsへの移行が必要
- https://mc.merill.net/message/MC1181996

### 3. Copilot Studio Computer Use GA
- **全商用リージョンで一般提供開始**
- ビジョン+推論ベースでブラウザ・デスクトップアプリのUIを直接操作。セレクタベースRPAと異なり、レイアウト変更に自動適応
- エージェントのモデル選択でOpenAI/Anthropicモデルが利用可能
- APIのないレガシーシステムへの自動化を大幅に簡素化
- https://techcommunity.microsoft.com/blog/copilot-studio-blog/computer-using-agents-in-microsoft-copilot-studio-are-now-generally-available/4519427

### 4. Power Platform May 2026 Feature Update（5/14公開）
大規模アップデート。主要項目:
- **Power Fx UDTs GA**: ユーザー定義型がv3.26044で一般提供。新規アプリではデフォルト有効。ParseJSONの型安全化
- **Agentic Apps Public Preview**: カスタムMCPツール+Fluent UIウィジェットでCopilot内のアプリ対話を実現
- **Grid Container Control GA**: CSS Grid風レイアウト。X/Y絶対配置を置換。ドラッグ＆ドロップ対応
- **Data Grid Modern Control Public Preview**: 高性能テーブルビュー。検索・ソート・複数選択対応
- **InfoPath移行支援 GA**: Canvas Authoring MCP Server + PowerCAT Skillで、GitHub Copilot/Claude Code等からInfoPathフォームをPower Apps Canvas Appsに変換
- **Power Automate Self-Healing Preview**: デスクトップフローの堅牢性問題を自動修復
- **Desktop Flow Testing GA**: テストケースの作成・管理・自動検証
- **Solution import性能40%向上**: 依存関係の並列処理。選択的コンポーネントデプロイ対応
- **Copilot-ready環境テンプレート**: 新規環境作成時にAI Builder credits・主要コネクタ・サンプルデータを自動プロビジョニング
- https://www.microsoft.com/en-us/power-platform/blog/2026/05/14/whats-new-in-power-platform-may-2026-feature-update/

- **Plugin依存管理**: `claude plugin disable`が依存プラグイン存在時に拒否（disable-chainヒント付き）。`enable`は推移的依存を強制有効化

- バックグラウンドセッションがアイドル復帰後もモデル・effortレベルを保持

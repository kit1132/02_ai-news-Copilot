## Microsoft 365 Copilot

### 三層リリースモデル（Frontier / Standard / Deferred）展開開始
M365 Copilotに新しいリリース管理モデルが5月下旬から適用。Frontierは早期アクセス（プレリリース）、Standardは通常GA、Deferredは最大30日遅延。管理センターの Copilot > Settings > Copilot Release preferences で設定可能。最大100ユーザーを例外指定可能。
- https://blog-en.topedia.com/2026/04/new-three-tier-release-model-for-microsoft-365-starting-with-copilot/

### Copilot Chatライセンス変更（5/16実施済み）
5/16にライセンス変更が実施完了。Copilot ChatはM365ライセンス保有者向けの無償AI機能として継続。

### Outlook メールグラウンディング・PDF直接表示
Copilot Chatでメールスレッドのテキストをプロンプトコンテキストに直接追加可能に。PDFもCopilot Chat内で直接表示。
- https://learn.microsoft.com/en-us/microsoft-365/copilot/release-notes

### SharePoint AI機能の世界展開
自然言語でチャートやビジュアルを記述→AIがコンポーネント生成。「File Actions with AI」でファイルのリネーム・移動・整理を自然言語指示。5月中に世界展開中。

### Claude models default-on
Anthropic Claudeモデル（Word/Excel/PowerPoint）が対象テナントでデフォルト有効に。従来のオプトインから変更。

## Copilot Studio

### Real-time Voice Agents ガバナンスガイド公開（5/19）
Copilot Studio Blogに「The in-depth guide to managing real-time voice agents at scale」公開。北米でGA済み。Dynamics 365 Contact Center経由で提供。5段階のガバナンスライフサイクル（Builder/Build/Release/Runtime/Lifecycle）を定義。
- https://www.microsoft.com/en-us/microsoft-copilot/blog/copilot-studio/the-in-depth-guide-to-managing-real-time-voice-agents-at-scale/

### Voice Agent Evaluation GA予定（5/31）
音声エージェントの品質評価機能が5/31にGA予定。テキスト変換ベースで会話品質を評価。

### Question Themes Analysis GA予定（5/29）
未分類ユーザー質問を自動テーマ分類。週次で分析が自動実行される。

### Hold & Resume for Voice Agents
音声通話の一時保留・再開機能が追加。通話途中で中断→再開が可能に。

## Anthropic / Claude

### Claude Managed Agents: Self-hosted Sandboxes + MCP Tunnels（5/19）
Code with Claude Londonイベントで2機能を発表。

**Self-hosted Sandboxes（Public Beta）**: エージェントのツール実行を自社インフラ（またはCloudflare/Daytona/Modal/Vercel等のプロバイダー）で実行。オーケストレーションはAnthropic側、ツール実行は顧客側。ファイル・リポジトリが外部に出ない。

**MCP Tunnels（Research Preview）**: プライベートネットワーク内のMCPサーバーにエージェントが到達。軽量ゲートウェイがアウトバウンド接続のみ使用。インバウンドファイアウォールルール不要、エンドツーエンド暗号化。
- https://claude.com/blog/claude-managed-agents-updates

### Claude Code v2.1.143〜v2.1.145
**v2.1.143（5/16）**: plugin dependency enforcement（依存プラグインの自動有効化/無効化防止）。`worktree.bgIsolation: "none"` 設定追加。プラグインマーケットにコンテキストコスト（トークン見積もり）表示。

**v2.1.144（5/19）**: `/resume` がバックグラウンドセッション対応。`/model` がセッション限定に変更（`d` キーでデフォルト設定可）。"extra usage"→"usage credits"にリネーム。API到達不能時のスタートアップ75秒ハング→15秒タイムアウトに修正。40以上のバグ修正。

**v2.1.145（5/19）**: 最新安定版。
- https://www.claudeupdates.dev/

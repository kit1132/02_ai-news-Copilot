# AI News Digest - 2026-04-29

## Microsoft 365 Copilot / Agent 365

### Agent 365 GA まであと2日（5/1）— ライセンス詳細が明確化
Agent 365（$15/user/月）と M365 E7 Frontier Suite（$99/user/月）が5/1にGA。GA時点ではライセンスユーザー代理エージェントのみ対象、自律エージェント（独自ID）は Frontier Preview 継続。販売チャネルは EA / EAS / CSP / MCA。組織はデプロイガイダンスの確認と管理者トレーニングの準備が推奨される。
- https://m365admin.handsontek.net/microsoft-agent-365-generally-available-may-1-2026/
- https://licensing.guide/may-1-ga-is-not-yet-the-final-frontier-for-agent-365/

### M365 Copilot: サードパーティモデルプロバイダーの管理者制御（MC1263276）
管理者が Microsoft Admin Center から Anthropic や xAI などのサードパーティモデルプロバイダーを特定のユーザー/グループに割り当て可能に。4月下旬〜5月にかけてワールドワイドでロールアウト中。
- https://mc.merill.net/message/MC1263276

### Researcher に「Critique」機能 — GPT起案 + Claude レビュー
M365 Copilot の Researcher で、GPT が初期回答を起案し Claude がレビューする「Critique in Researcher」が導入。回答の正確性・完全性・引用整合性を向上させる二重チェック構成。
- https://levelupm365.com/2026/03/31/microsoft-365-roadmap-updates-april-2026/

## Anthropic / Claude

### Claude for Creative Work — 9コネクタを一斉リリース（4/28）
Anthropic が Adobe、Blender、Autodesk Fusion、Ableton、Splice、Affinity（Canva）、SketchUp、Resolume Arena、Resolume Wire の9つのクリエイティブツール向けコネクタをリリース。Adobe コネクタは Photoshop / Premiere / Express 等 50以上のツールに対応。Blender コネクタは Python API 経由でシーン分析・バッチ処理が可能。Anthropic は Blender Development Fund のパトロンにも就任。RISD、Ringling、Goldsmiths の3大学との教育パートナーシップも発表。
- https://www.anthropic.com/news/claude-for-creative-work
- https://blog.adobe.com/en/publish/2026/04/28/adobe-for-creativity-connector

### Claude AI 大規模障害（4/28、約78分間）
4/28 17:34〜18:52 UTC に Claude.ai、Claude Code、Anthropic API が同時ダウン。Downdetector 報告は一時12,000件超。18:52 UTC までに復旧完了。4/20 の障害に続き、8日間で2回目の大規模障害。
- https://status.claude.com/
- https://gvwire.com/2026/04/28/claude-ai-goes-down-for-thousands-of-users-tuesday-downdetector-shows/

### Claude Code v2.1.121 リリース（4/28）
v2.1.119/120 のリグレッション問題を経て、v2.1.121 がリリース。主な変更点:
- MCP サーバー設定に `alwaysLoad` オプション追加（tool-search deferral をスキップ）
- `claude plugin prune` コマンド追加（孤立プラグイン依存の整理）
- `/skills` にタイプフィルタ検索ボックス追加
- PostToolUse フックが全ツールで `updatedToolOutput` に対応（従来は MCP のみ）
- iTerm2 + tmux 環境のターミナル表示崩れを修正
- https://github.com/anthropics/claude-code/releases

## 開発ツール

### GitHub Copilot CLI v1.0.36（4/24）
12項目の機能追加・改善。`/remote` コマンド（リモート制御のオン/オフ切替）、`/keep-alive`（スリープ防止）、サブコマンドピッカーに選択インジケータ（❯）追加、`changes` ステータスライン表示、`preToolUse.matcher` の正規表現マッチ修正。Claude Opus 4.6 のデフォルト推論エフォートが medium に変更。
- https://github.com/github/copilot-cli/releases

### Visual Studio 2026: IntelliSense が Copilot 補完に優先する設定がデフォルト化
VS 2026 v18.5 で IntelliSense と Copilot 補完の競合問題を解決。IntelliSense がアクティブな間は Copilot 補完が抑制され、IntelliSense を確定/解除後に Copilot が再開する動作がデフォルトに。設定で従来の同時表示にも変更可能。
- https://visualstudiomagazine.com/articles/2026/04/28/visual-studio-2026-gives-intellisense-priority-over-copilot.aspx

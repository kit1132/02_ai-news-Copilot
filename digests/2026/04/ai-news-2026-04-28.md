## Microsoft 365 Copilot / Copilot Studio

### Agent 365 / M365 E7 GA まで3日（5/1）
Agent 365（$15/user/月）と M365 E7 Frontier Suite（$99/user/月）の GA が5/1に迫る。GA 時点ではライセンスユーザー代理のエージェントのみ対象で、自律エージェント（独自IDを持つもの）は引き続き Frontier Preview。E7 は E5 + Copilot + Entra Suite + Agent 365 のバンドルで単品合計 $117 に対し約15%割引。
- https://licensing.guide/may-1-ga-is-not-yet-the-final-frontier-for-agent-365/
- https://samexpert.com/agent-365/

### Copilot アジェンティック機能 GA のエンゲージメント指標公開（4/22）
Word/Excel/PowerPoint のアジェンティック機能 GA 後の初期指標が公式ブログで公開。Word エンゲージメント 52%増・満足度 21%増、Excel エンゲージメント 67%増・満足度 65%増、PowerPoint 新規ユーザーリテンション 36%改善。Copilot がドラフト提案から複数ステップのアプリ内操作へシフトした効果が数値で裏付けられた。
- https://www.microsoft.com/en-us/microsoft-365/blog/2026/04/22/copilots-agentic-capabilities-in-word-excel-and-powerpoint-are-generally-available/

### Copilot Studio に実験モデル追加：Grok 4.1 Fast / GPT-5.3 Thinking / GPT-5.4 Instant
有償実験プレビューとして新モデル3種が追加。既存の Claude Opus 4.6 / Sonnet 4.5 / 4.6、ChatGPT-5 に加え、モデル選択肢がさらに拡大。マルチエージェントオーケストレーション（Fabric / M365 Agents SDK / A2A）GA と合わせ、エージェント基盤の成熟が進む。
- https://www.microsoft.com/en-us/microsoft-copilot/blog/copilot-studio/new-and-improved-multi-agent-orchestration-connected-experiences-and-faster-prompt-iteration/

### 「What's New in M365 Copilot | April 2026」月次記事は未公開
Tech Community の月次まとめ記事は4月28日時点で未公開。March 2026 版が最新。月末〜5月初旬の公開が見込まれる。

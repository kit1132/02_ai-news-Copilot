# AI News Digest - 2026-05-02

## M365 Copilot / Copilot Studio

### Copilot に GPT-5.4 Thinking / GPT-5.3 Instant モデル追加
M365 Copilot と Copilot Studio で GPT-5.4 Thinking（複雑なマルチステップタスク向け深い推論）と GPT-5.3 Instant（高速応答）が利用可能に。従来の GPT-5 に加え、タスク特性に応じたモデル選択が可能になった。
- https://learn.microsoft.com/en-us/microsoft-365/copilot/release-notes

### Copilot Cowork（Claude 統合）が M365 E7 で稼働開始
5/1 GA の M365 E7 Frontier Suite（$99/user/月）に Copilot Cowork が含まれ、Claude が Outlook・Teams・Excel・Word・PowerPoint 内でエージェントとして動作。OpenAI と Anthropic のマルチモデル構成で、タスクに応じて最適なモデルへルーティングされる。
- https://www.computerworld.com/article/4142551/m365-copilot-gets-its-own-version-of-claude-cowork.html

### Excel Copilot に GPT-5.5 / Claude Opus 4.7 対応、チャット/編集切替・Plan Mode
Excel Copilot が Windows・Mac・Web 全体で強化。チャット/編集スイッチャー、Plan Mode（変更計画の事前確認）、変更追跡の明確化、Python 分析対応を追加。モデルとして GPT-5.5 と Claude Opus 4.7 が選択可能に。
- https://www.microsoft.com/en-us/microsoft-365/roadmap

### Copilot Notebooks 機能拡張（5月ロールアウト）
Copilot Notebooks に外部 Web リンクを参照として追加可能に。ノートブック内容から PowerPoint プレゼンテーションや Word ドキュメント（レポート・提案書等）を直接生成する機能も追加。
- https://techcommunity.microsoft.com/blog/microsoft365copilotblog/what%E2%80%99s-new-in-microsoft-365-copilot--april-2026/4510935

### Work IQ API Public Preview（A2A 対応）
REST API および MCP リモートサーバー経由で Work IQ インテリジェンスにアクセス可能に。M365 のファイル・メール・会議・チャットからのリアルタイムワークインサイトを Copilot Studio エージェントで活用できる。
- https://techcommunity.microsoft.com/blog/copilot-studio-blog/work-iq-api-public-preview-build-copilot-powered-agents-with-a2a/4516286

## Power Platform

### Flex Routing が Power Platform 環境に拡張（5/4 有効化）
Flex Routing が Power Platform 環境・Copilot Studio・Dynamics 365 に拡張。5/4 からデフォルト ON に切替。Power Platform 管理センターの環境ごとの「Generative AI features」設定で制御可能。
- https://blog-en.topedia.com/2026/04/flex-routing-extends-to-power-platform-environments/

### Copilot Studio CLI サポート GA（5/31 予定）
Power Platform CLI と統合された Copilot Studio 用 CLI が 5/31 に GA 予定。エージェント駆動の開発ワークフロー（CI/CD パイプラインでのエージェント管理・デプロイ）を CLI からサポート。
- https://m365admin.handsontek.net/microsoft-copilot-studio-cli-support-microsoft-copilot-studio/

### Dragon Copilot 価格改定（5/1 発効）
Dragon Copilot のユーザーあたりライセンスが全提供地域で値下げ。アンビエントエンカウンターと生成 AI 機能の簡素化された従量課金モデルも導入。
- https://learn.microsoft.com/en-us/partner-center/announcements/2026-april

## AI 開発ツール

### Claude Code v2.1.126（5/1）
主な変更点:
- `ANTHROPIC_BASE_URL` でゲートウェイ指定時、`/model` ピッカーがゲートウェイの `/v1/models` エンドポイントからモデル一覧を取得
- `claude project purge [path]` コマンド追加（プロジェクトの全 Claude Code 状態を削除、`--dry-run`/`--all` 対応）
- **セキュリティ修正**: `allowManagedDomainsOnly` / `allowManagedReadPathsOnly` が上位の managed-settings ソースに sandbox ブロックがない場合に無視されるバグを修正
- `claude auth login` が WSL2/SSH/コンテナ環境でブラウザコールバック不可時にターミナルへの OAuth コード貼り付けに対応
- Windows: PowerShell 7 の自動検出改善、PowerShell ツール有効時に Bash ではなく PowerShell をプライマリシェルとして使用
- 画像 2000px 超貼り付けで自動ダウンスケール（セッション破損修正）
- API リトライカウントダウン "0s" 固定、スリープ復帰後の Stream idle timeout、CJK 文字化け等のバグ修正多数
- https://github.com/anthropics/claude-code/releases

### GitHub Copilot CLI v1.0.40（5/1）
主な変更点:
- MCP サーバー向け `client_credentials` OAuth グラントタイプ対応（ヘッドレス認証）
- `/research` がオーケストレーター/サブエージェントモデルに刷新、より精度の高いディープリサーチ
- Autopilot モード: デフォルト継続メッセージ上限を 5 に制限（`--max-autopilot-continues` で設定可能）
- `copilot completion` サブコマンドでシェル補完スクリプト生成
- PR ブランチ装飾表示修正、`/clear`・`/new` でカスタムエージェント選択リセット
- Prompt モード（`-p`）: repo hooks と workspace MCP がデフォルト無効に（セキュリティ強化）
- https://github.com/github/copilot-cli/releases

### Cursor Security Review ベータ（5/1）
Teams / Enterprise プラン向けに常時稼働セキュリティエージェントのベータを開始:
- **Security Reviewer**: 全 PR を自動チェック（脆弱性・認証リグレッション・プライバシーリスク・プロンプトインジェクション）。diff の該当箇所にインラインコメントで重大度と修正案を提示
- **Vulnerability Scanner**: スケジュールスキャンで既知脆弱性・依存関係の古さ・設定問題を検出。Slack 通知対応
- 既存の SAST/SCA/シークレットスキャナーの MCP サーバーをプラグイン可能
- https://cursor.com/changelog/04-30-26

### Devin: Fast Mode・Desktop Testing・Review 強化
- **Fast Mode**: 従来比約 2x 高速レスポンス（4x ACU 消費）
- **Desktop Testing**: Computer Use で Linux デスクトップアプリの E2E テストに対応。QA 承認後にアプリ操作の録画を送信
- **Devin Review**: チャットエージェントが会話内でコード編集を提案し、Devin Review を離れずにコミットとして適用可能
- **Secure Mode**: 非エンタープライズ組織でも利用可能に
- https://docs.devin.ai/release-notes/2026

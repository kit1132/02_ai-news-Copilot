# AI News Digest: 2026-05-05 (月)

## 本日のトップニュース

### Anthropic、Blackstone・Goldman Sachs等と$1.5Bのエンタープライズ AI 合弁会社を設立（5/4発表）
- Anthropic、Blackstone、Hellman & Friedman が各約$300M、Goldman Sachs が約$150M を出資。Apollo、General Atlantic、GIC、Sequoia Capital 等も参加
- Claudeを企業の基幹業務に直接デプロイする「AI ネイティブ・エンタープライズサービス企業」を設立。大手コンサルティングファームとの直接競合を狙う
- OpenAI も TPG・Bain Capital と同様の構造を追求中と報道
- https://fortune.com/2026/05/04/anthropic-claude-consulting-industry-joint-venture-blackstone-goldman-sachs/

## Microsoft 365 / Copilot

### MC1296874: Calendar Agent（M365 Copilot）展開開始
- M365 Copilot に Calendar Agent 機能が追加。自然言語でカレンダー管理を自動化（会議の承認・辞退・削除等）
- 4月下旬〜5月上旬に Frontier ライセンス向けにロールアウト中
- 既存のポリシー・ユーザー設定・コンプライアンスを尊重。新規管理者制御の追加なし
- https://mc.merill.net/message/MC1296874

### Copilot Notebooks: Mind Map GA + Word/PPT生成（5月展開中）
- **Mind Map が5月にGA到達**。ノートブック内コンテンツのトピック・関連性を可視化し、Copilotで深掘り可能
- **Word文書生成**: ノートブックのコンテンツ・参照から報告書・サマリー・提案書を直接生成
- **PowerPoint生成**: ノートのコンテンツからスライドを自動作成。フォーカス・詳細レベル・デザインテーマを選択可能
- **外部Webリンク参照**: ノートブックにWebリンクを追加し、チャット・出力の情報源として活用
- https://techcommunity.microsoft.com/blog/microsoft365copilotblog/copilot-notebooks-enhancements-to-support-creation-collaboration-and-learning/4505360

### M365 Admin Center: Anthropic モデル デフォルト有効化（5/4〜）
- 商用クラウド（EU/EFTA/UK除く）でM365 Copilot の Anthropic モデルがデフォルト有効に変更
- Excel/PowerPoint で Claude Opus 4.7 利用可能。Word は5月下旬GA予定
- EU/EFTA/UK は管理者オプトイン必須（4/3設定追加済み）
- https://learn.microsoft.com/en-us/microsoft-365/copilot/copilot-anthropic-apps

### MC1227085: Engage Teams Live Events 新規作成停止（5/4発効）
- Engage 経由の Teams Live Events の新規スケジュールが5/4に停止
- 既存のスケジュール済みイベントは2027年2月28日までサポート継続
- Teams Town Hall への移行を推奨。Teams Live Events 自体は6/30廃止予定
- https://mc.merill.net/message/MC1227085

## AIツール・開発

### Claude Code: /ultrareview 無料トライアル本日期限（5/5）
- Pro/Max サブスクライバーに付与された3回の無料 ultrareview ランが**本日5/5に期限切れ**
- 以降は Extra Usage として$5〜$20/回で課金。リモートサンドボックスで複数のレビューエージェントが並列実行し、再現検証済みのバグのみ報告
- v2.1.126（5/1）が引き続き最新バージョン
- https://code.claude.com/docs/en/ultrareview

### Cursor: Canvas + Bugbot 自己学習（5月初旬）
- **Canvas**: エージェントがデータ可視化・PR レビュー・評価分析をインタラクティブな React ベース UI として生成。Agents Window のサイドパネルに常駐
- **Bugbot Learned Rules**: PR フィードバック（ダウンボート・リプライ・人間レビュアーコメント）から自動でルールを生成。110,000+リポジトリが有効化、44,000+ルール生成済み。PR マージ時のバグ解決率が80%近くに上昇
- **Team Marketplace**: リポジトリ接続なしでチームマーケットプレイス作成可能に。プラグイン（MCP Server、スキル、サブエージェント等）の管理を強化
- https://cursor.com/changelog

### Devin: v3 API GA + セッション一覧リデザイン
- **v3 API が正式リリース**。ロールベースアクセス制御、セッション帰属、全レガシー API 機能を統合
- **セッション一覧ページ再設計**: インライン PR プレビュー、メッセージスニペット、ステータスインジケーター表示。作成日でソート可能
- **マージコンフリクト自動通知**: Devin が作成した PR にコンフリクトが発生すると自動通知
- **Enterprise**: ナレッジ管理（CRUD）、プレイブック管理、スケジュールセッション（定期/単発）機能追加
- https://docs.devin.ai/release-notes/2026

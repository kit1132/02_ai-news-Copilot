# AI News Digest: 2026-05-08 (木)

## 本日のトップニュース

### Claude Managed Agents: Dreaming・Outcomes・Multiagent Orchestration 発表
- Code with Claude カンファレンス（5/6 SF）で発表。5/7に詳細ブログ公開
- **Dreaming**（research preview）: スケジュール実行でエージェントの過去セッション・メモリストアをレビューし、パターン抽出・メモリ最適化。単一エージェントでは見えない反復ミスやチーム共通ワークフローを検出
- **Outcomes**（public beta）: 成功基準をルーブリックとして記述→別コンテキストのグレーダーが評価→不合格時は具体的改善点を指摘して再実行。エージェントの推論に影響されない独立評価
- **Multiagent Orchestration**（public beta）: リードエージェントがタスクを分割し、個別モデル・プロンプト・ツールを持つスペシャリストに委任。共有ファイルシステム上で並列実行、途中チェックイン可能
- https://claude.com/blog/new-in-claude-managed-agents

### Claude Code Desktop リデザイン + 使用量上限倍増
- **Claude Code Desktop** の全面リデザイン: 新セッションサイドバー、ドラッグ&ドロップワークスペース、統合ターミナル・ファイルエディタ、高速 diff、プレビュー拡張、Mac SSH対応
- **使用量上限 2倍**: Pro / Max / Team / Enterprise の全プランおよび Opus API のレート制限を倍増。SpaceX 計算基盤の追加容量による
- https://www.anthropic.com/news/claude-opus-4-7

### Copilot Studio: Agent Evaluations GA・Content Moderation GA
- **Agent Evaluations が GA**: カスタムテストセットでエージェント性能を検証。多様なシナリオでの強み・改善点を特定
- **Content Moderation Settings が GA**: サポート対象リージョンで、マネージドモデルの有害コンテンツ感度をメーカーが制御可能に
- **Prompt Tool に Claude Opus 4.6 / Sonnet 4.5 追加**（paid experimental preview、US）: プロンプトに最適なモデルを選択する幅が拡大
- https://learn.microsoft.com/en-us/microsoft-copilot-studio/whats-new

## Microsoft 365 / Copilot

### M365 Roadmap: Legal Agent in Word・Work IQ API・Registry Sync
- **Legal Agent in Word**: 契約書レビュー・レッドライン・交渉を構造化された法務ワークフローで実行する新エージェント
- **Work IQ API for Developers**: A2A（agent-to-agent）、MCP、REST の3インターフェースで M365 Copilot のインテリジェンスレイヤーにアクセス。エンタープライズセキュリティ・コンプライアンス・ガバナンスを継承
- **Agent 365 Registry Sync**: M365 サービス横断で登録されたエージェントを中央レジストリに同期。ガバナンス・可視性・ライフサイクル管理を一元化
- https://www.microsoft.com/en-us/microsoft-365/roadmap

### Copilot Notebooks: OneNote iOS マルチモーダルキャプチャ
- OneNote モバイルアプリ（iOS）で**音声文字起こし・画像・テキスト入力を1セッションで同時キャプチャ**し、構造化された Copilot Page に変換。5月ロールアウト中
- 外部 Web リンクの参照追加も5月展開（URL を貼ると Copilot がノートブックのチャットやアウトプットに活用）
- https://learn.microsoft.com/en-us/microsoft-365/copilot/release-notes

## Copilot Studio

### Request for Information アクション（Human-in-the-Loop）
- エージェントフロー中に**人間の入力を要求するアクション**を追加。指定レビュアーに Outlook 経由で詳細確認を送信し、応答を受け取るまでフローを一時停止。応答は動的パラメータとして後続ステップに使用
- https://learn.microsoft.com/en-us/power-platform/release-plan/2026wave1/microsoft-copilot-studio/planned-features

### テーマグルーピング分析・ROI 追跡
- 関連するユーザー質問を**テーマ別にグループ化**してパターンを分析。自律型・会話型エージェントの時間・コスト削減を追跡し ROI を可視化
- https://learn.microsoft.com/en-us/microsoft-copilot-studio/whats-new

## 開発ツール

### Claude Code v2.1.131-132（5/6 リリース）
- v2.1.132: `CLAUDE_CODE_SESSION_ID` 環境変数をBashサブプロセスに追加。`CLAUDE_CODE_DISABLE_ALTERNATE_SCREEN=1` で全画面レンダラーをオプトアウト可能に
- 外部 SIGINT（IDE停止ボタン等）でグレースフルシャットダウンが正常動作するよう修正
- **MCP サーバーの非プロトコルデータ stdout 書き込みによる無制限メモリ増加（10GB+ RSS）を修正**
- Ctrl+V 画像ペースト中に「Pasting…」ヒント表示
- JetBrains IDE 2025.2 / Cursor / VS Code 1.92-1.104 でのスクロール問題修正
- v2.1.131: VS Code 拡張の Windows アクティベーション問題修正
- https://code.claude.com/docs/en/changelog

### GitHub Copilot CLI v1.0.43 + VS Code April Releases
- **CLI v1.0.43**（5/6）: `/statusline` にユーザー名トグル追加。auto mode がサーバーサイドモデルルーティングに対応。悪意のあるベアリポジトリによる RCE を修正
- **VS Code**（5/6 公開）: ワークスペース・GitHub リポジトリ横断の**セマンティック検索**。`/remote` でデバイス間セッション継続（Experimental）。`/chronicle` でチャット履歴検索（Experimental）
- **GitHub Secret Scanning in MCP が GA**: MCP 対応 AI コーディングエージェントでコミット前にシークレット漏洩を検出
- https://github.blog/changelog/label/copilot/

### Cursor v3.3（5/6 リリース）
- エージェントの**コンテキスト使用量の内訳表示**: ルール・スキル・MCP・サブエージェント別に診断可能
- コンテキスト問題のデバッグとセットアップ改善に活用
- https://cursor.com/changelog

### Devin: v3 API 正式リリース・新機能
- **Devin v3 API**: 全 Devin 機能のプライマリ API として正式リリース。ロールベースアクセス制御・セッション帰属・新機能を追加
- **Preview Agent Toggle**: ストリーミング思考表示・高速実行
- **Inline File Previews**: HTML/PDF/SVG をセッションサイドバー内でセキュアにレンダリング（コード/レンダートグル・ダウンロード対応）
- **Devin Manages Devins**: Devin が他の Devin を委任・管理。各 Devin は独立 VM で動作し、メイン Devin がコーディネーター
- https://docs.devin.ai/release-notes/overview

# AI News Digest — 2026-06-13

前日（6/12）から新規の動きは3件。Power Platformの6月月次更新が公開され、Copilot Studio 2026.5.5 とClaude Codeの新ビルドが出た。

## 最優先: Power Platform / Copilot Studio

### Power Platform「June 2026 Feature Update」公開 (6/11)
未公開だった6月の月次What's New記事が公開。ガバナンスとエージェント運用系のGAが中心。

- **Power Apps MCP Server クローズドループ学習（GA）**: エージェントがユーザーの修正をagent feed経由で自動学習し、構造化メモリとして保持して以降の応答に反映
- **高度なコネクタポリシー（GA）**: AIツールが利用するコネクタの「アクション・内部のMCPサーバー」単位でガバナンス可能に。大規模統制のニーズに対応
- **Power Platform インベントリにコネクタ利用データ（Public Preview）**: 各リソースが使うコネクタ・操作をインベントリで把握。コネクタ廃止・変更の影響範囲を全リソースで特定可能
- **Power Automate デスクトップフロー バージョン比較（GA）**: サブフロー・アクション・変数・UI要素・画像を並べて差分比較
- **Power Automate「Run Power App」アクション（Preview）**: デスクトップフローからPower Appを直接開きネイティブ通信チャネルを確立（有人自動化向け）
- **Release Planner Code App（GA）**: Power Apps code apps + Reactで構築されたリリース計画探索アプリのサンプル（カード/カレンダー/タイムライン表示）
- https://www.microsoft.com/en-us/power-platform/blog/2026/06/11/whats-new-in-power-platform-june-2026-feature-update/

### Copilot Studio Build 2026.5.5 リリース (6/9)
2026.5.4に続く週次ビルド。セキュリティ・ガバナンス強化が目立つ。日本は引き続き2026.5.4（Preview US他10リージョンに2026.5.5展開中）。

- **A2Aプロトコルで `tasks/get`**: 外部エージェントが現在のタスク状態を取得可能に
- **GPT-5.5 Chatでインラインエージェント実行**: ツールベースのオーケストレーションがより高機能に
- 環境レベルの平均エージェント品質・健全性メトリクスをページング付きで表示
- 評価シナリオでプロンプトごとにクライアント側コンテンツモデレーションレベルを設定可能（従来は常にHigh固定）
- プエルトリコ・スペイン語（es-PR）GA
- **破壊的変更**: カスタムメトリクス分析サマリにlabel trend・over-timeフィールドが常時含まれる（空でも）。フィールド欠落/null前提の連携に影響
- **セキュリティ**: Anthropicレスポンスのエラー詳細をログから秘匿。情報バリア有効テナントではEmbedded Knowledge利用エージェントを「自分のみ」超で公開・共有不可。AI生成URLの抑制/無害化を既定で有効化。Agent StatusがDLP違反詳細（コネクタ・HTTPリクエスト等）を表示
- https://learn.microsoft.com/en-us/power-platform/released-versions/copilotstudio/2026.5.5

## 高優先: AIツール

### Claude Code v2.1.174 / v2.1.175 (6/12)
6/11のv2.1.173に続くリリース。モデルピッカー周りの修正が中心。

- **v2.1.175**: `enforceAvailableModels` 管理設定追加。有効時、`availableModels` 許可リストがDefaultモデルも制約（不許可に解決されるDefaultは許可リスト先頭にフォールバック）。ユーザー/プロジェクト設定で管理リストを拡大不可
- **v2.1.174**: `/model` ピッカーがDefaultの解決先モデルファミリを非表示にする問題を修正（Opus/Sonnetが正しく表示）。「Fable 5がクレジット消費中」バナーが従量課金エンタープライズで誤表示される問題を修正。Bedrock GovCloud（`us-gov-*`）の推論プロファイルプレフィックス誤り（400エラー）修正。バックグラウンドセッションが他セッションの `ANTHROPIC_*` 環境変数を継承する問題を修正。`wheelScrollAccelerationEnabled` 設定追加。[VSCode] `/usage` にキャッシュミス・ロングコンテキスト・サブエージェント・スキル/エージェント/プラグイン/MCP別の利用内訳を追加
- https://code.claude.com/docs/en/changelog

## 最重要: Microsoft 5/5 大型発表群

### 1. Copilot Cowork — iOS/Android展開 + Skills + プラグイン

5/5にCopilot Coworkがモバイル（iOS/Android）に拡張。Microsoft 365 Copilotアプリからタスクを委任し、完了結果を受け取る形で動作する。

- **Skills**: 再利用可能な指示セットを保存し、Coworkに特定タスクを自分の手順・トーンで実行させる機能
- **プラグイン/コネクタ**: HubSpot、LSEG、Moody's、Notionが利用可能。Miro、monday.com、S&P Global Energy等が近日追加予定
- **Fabric IQ + Power BI / Dynamics 365** との統合も発表
- Frontier プログラム限定（E7ライセンス）

https://www.microsoft.com/en-us/microsoft-365/blog/2026/05/05/copilot-cowork-from-conversation-to-action-across-skills-integrations-and-devices/

### 2. Microsoft Agent 365 GA + Shadow AI検出

Agent 365が正式GA。組織内のAIエージェントを一元管理するコントロールプレーン。

- エージェントインベントリの一元化、ロールベースアクセス、DLP境界設定、アクティビティ監視
- **Shadow AI検出**: OpenClaw、Claude Code等のローカルエージェントを含む非公認AIエージェントの検出・管理機能（Preview）
- M365 E5+ライセンスで利用可能

https://www.microsoft.com/en-us/microsoft-365/blog/2026/05/05/microsoft-365-copilot-human-agency-and-the-opportunity-for-every-organization/

### 3. Federated Copilot Connectors GA

HubSpot、LSEG、Moody's、Notionのコネクタが GA。Excel統合は夏予定。パートナーエコシステムの拡大が本格化。

### 4. 2026 Work Trend Index 公開

- ナレッジワーカーの78%が週次でAIエージェントを利用（2024年の12%から急増）
- M365上のアクティブエージェント数は前年比15倍（大企業では18倍）
- Copilot Chat利用の49%が分析・意思決定・問題解決等の高度な認知作業
- **課題**: AIユーザーの4人に1人しか「リーダーがAI戦略で足並みを揃えている」と回答せず（"Transformation Paradox"）

https://www.microsoft.com/en-us/worklab/work-trend-index/agents-human-agency-and-the-opportunity-for-every-organization

## Anthropic / Claude

### 9. Code with Claude 2026（5/6 サンフランシスコ）

Anthropic主催の開発者イベント。SF（5/6-7）→ London（5/19-20）→ Tokyo（6/10）の3都市開催。

主要発表:
- **Claude Managed Agents 新機能3つ**:
  - **Dreaming**: 過去セッションを振り返りパターンを発見し自己改善する研究プレビュー機能
  - **Outcomes**: 成功条件を定義し、Claudeが反復して目標達成を目指す機能
  - **Multi-agent orchestration**: リードエージェントがタスクを分割し、専門エージェントに委任。並列実行・共有ファイルシステム対応

https://9to5mac.com/2026/05/07/anthropic-updates-claude-managed-agents-with-three-new-features/

### 10. Claude Code Desktop リデザイン

並列セッション管理に特化した大幅リデザイン。

- サイドバーでの複数セッション管理、ドラッグ&ドロップレイアウト
- 統合ターミナル、アプリ内ファイルエディタ、高速diffビューア
- HTML/PDFプレビュー、ローカルサーバープレビュー
- CLIプラグインとの完全互換、SSH対応がMacにも拡張
- Pro/Max/Team/Enterpriseプランで利用可能

https://claude.com/blog/claude-code-desktop-redesign

### 11. Claude Code v2.1.128〜v2.1.136（5/4〜5/8）

4バージョンリリース。主な変更:
- `--plugin-url`フラグ、`--plugin-dir`でZIPアーカイブ対応
- `worktree.baseRef`設定、`sandbox.bwrapPath`/`sandbox.socatPath`管理設定
- `CLAUDE_CODE_SESSION_ID`環境変数、`CLAUDE_CODE_DISABLE_ALTERNATE_SCREEN`
- Hooksにeffort level（`effort.level` / `$CLAUDE_EFFORT`）を渡す機能
- `settings.autoMode.hard_deny`でautoモードの無条件ブロック
- OAuth並行セッション競合、MCP OAuth refresh token、WSL2画像貼り付け等50+の修正

https://code.claude.com/docs/en/changelog

## Copilot Studio / エージェント基盤

### Copilot Studio 月次ブログ公開（5/11）: エージェントガバナンス・ワークフロー・アプリ統合の強化
Nitasha Chopra VP & COO による「New and improved: Agent governance, intelligent workflows, and connected app experiences」が公開。April 2026の主要アップデートをまとめた月次記事。
- **Apps in Agents GA**: Copilot Chat内にリッチなインタラクティブアプリ体験を直接埋め込み可能に
- **Analytics Viewer Role GA**: エージェント分析への読み取り専用アクセスロール。運用可視性と構成権限の分離を実現
- **Workflows Agent一元管理**: 管理者制御の環境でDLPポリシー一貫性を確保。エージェントノードによるCopilot Studioエージェントのワークフロー内呼び出し
- **Custom Metrics**: ビジネス目標に基づくアウトカムベースの測定指標をカスタム定義可能に
- **Agent Usage Estimator拡張**: Dynamics 365エージェント（Sales Qualification Agent、Customer Service Agent）を追加。統合Copilotクレジット消費予測
- **Agent Store拡充**: Adobe Express、Box、Figma、Monday.com、Wix がエンタープライズグレードのセキュリティ付きで統合
- **GPT-5.5 Thinking/Reasoning**: Early Release環境で利用可能に
- https://www.microsoft.com/en-us/microsoft-copilot/blog/copilot-studio/

### Work IQ API Public Preview — A2A・MCP・REST の3形式で公開
Work IQ APIがパブリックプレビューとして利用開始。M365 Copilotの基盤インテリジェンスに外部からアクセスする3つの方法を提供。
- **Agent-to-Agent（A2A）**: カスタムエージェントがCopilotインテリジェンスとピアとして対話。コンテキスト維持・作業委任・グラウンデッド応答
- **MCP（ローカルサーバー）**: GitHub CopilotからWork IQ CLI経由でMCP準拠エージェントが組織コンテキストにアクセス
- **REST（5月提供開始）**: ストリーミング不要なアプリ・統合向けの標準API
- GA予定: 2026年夏。ライセンス不要ユーザーへの従量課金も夏に提供予定
- https://techcommunity.microsoft.com/blog/copilot-studio-blog/work-iq-api-public-preview-build-copilot-powered-agents-with-a2a/4516286

## Microsoft 365 Copilot

### Outlook Copilot 新機能発表（5/10）: Classic Outlook対応・キャンバスドラフト・スケジューリング強化
5月のOutlook Copilot更新が公開。Classic OutlookへのCopilot機能拡張が目玉。
- **Classic Outlook Copilot Insights**: メール内テキスト選択→Copilotに質問が可能に（New Outlookでは提供済み、Classic版に拡大）
- **キャンバス内直接ドラフト**: Copilotがメール本文に直接下書きを生成し、目的・対象者・トーンを確認しながら反復改善
- **プロアクティブスケジューリング**: 会議リキャップ・準備カードの自動表示
- **Find a Timeペイン**: 過去の出席傾向・Teamsステータス・ハイブリド出社日を考慮した最適時間帯提案
- 5月初旬にTargeted Releaseテナントで展開開始、月末までにGA予定
- https://www.windowslatest.com/2026/05/10/new-outlook-and-outlook-classic-features-in-may-2026-include-copilot-insights-teammates-calendars-automapped-calendars/

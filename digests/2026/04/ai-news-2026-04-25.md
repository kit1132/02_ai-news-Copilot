## Microsoft 365 Copilot / Copilot Studio

### M365 Community Conference 閉幕（4/21-23, Orlando）

- https://gigazine.net/gsc_news/en/20260424-microsoft-365-copilot-agent/

### Claude Sonnet が M365 Copilot Chat（Frontier）に追加
Claude Opus 4.7（4/17追加済）に加え、Claude Sonnet も Copilot Chat の Frontier プログラムで利用可能に。モデルセレクタから選択可能。Anthropic は Microsoft のサブプロセッサとして動作し、EU Data Boundary 対象外。EU/UKテナントはオプトイン必要。
- https://app.cloudscout.one/evergreen-item/mc1247880-2/

### Copilot Tuning が Frontier 向けに展開開始

## GitHub Copilot

### データポリシー変更が本日発効（4/24→4/25時点で適用済）
Free/Pro/Pro+ ユーザーのインタラクションデータ（入力・出力・コードスニペット）がAIモデル学習に使用される。opt-out は Settings > Copilot > Features から可能。Business/Enterprise は対象外。データは Microsoft と共有されるが、サードパーティ AI プロバイダーには共有されない。
- https://github.blog/news-insights/company-news/updates-to-github-copilot-interaction-data-usage-policy/

### 個人プラン変更を発表（4/20）
Pro/Pro+/Student の新規サインアップを一時停止。エージェントワークフローによるコンピュート需要増大が理由。Pro プランから Opus モデルを削除（Pro+ では Opus 4.7 継続）。利用上限を引き締め、Pro+ は Pro の5倍以上の上限を提供。
- https://github.blog/changelog/2026-04-20-changes-to-github-copilot-plans-for-individuals/

### Copilot CLI v1.0.35（4/23）
タブ補完でスラッシュコマンドの引数・サブコマンドをサポート。
- https://github.com/github/copilot-cli/releases

## Anthropic / Claude

### Claude Code Pro プラン除外テスト → 撤回（4/21-22）
Anthropic が新規 Pro（$20/月）サインアップの2%を対象に Claude Code のアクセスを除外するA/Bテストを実施。SNSで大きな批判を受け、撤回。Head of Growth の Amol Avasare が「既存ユーザーは影響なし」と声明。背景には、ヘビーユーザーのトークン消費がプラン価格の10倍以上に達する問題がある。
- https://www.theregister.com/2026/04/22/anthropic_removes_claude_code_pro/

### Claude Code 品質問題の修正完了（4/20-24）
ユーザーから「nerfing」の指摘があった品質低下について、3つの原因を特定・修正。(1) デフォルト推論努力度の低下 (2) thinking history を失うキャッシュバグ (3) コーディング品質を悪化させた verbosity プロンプト変更。モデル自体は変更なし。影響を受けたサブスクライバーの利用制限をリセット。
- https://techbriefly.com/2026/04/24/anthropic-resolves-claude-code-bugs-after-nerfing-allegations/

### Managed Agents ローンチ（4/8、Public Beta）
ホスト型AIエージェント実行レイヤー。セッション管理・サンドボックス・認証情報管理・エラー回復を提供。$0.08/session hour + トークン料金。Notion・楽天・Sentry が本番運用中。
- https://www.infoq.com/news/2026/04/anthropic-managed-agents/

### Claude Mythos Preview（4/7、Gated Research Preview）
サイバーセキュリティタスクに突出した汎用モデル。ゼロデイ脆弱性の発見・実証能力を持つ。Anthropic は「Project Glasswing」を発表し、50社以上に$100M+のクレジットを提供して防御的サイバーセキュリティへの活用を推進。一般公開は見送り。
- https://red.anthropic.com/2026/mythos-preview/

# AI News Digest - 2026-05-16

## 本日の重要イベント

### Copilot Chat ライセンス変更 本日実施（5/16）
- 大規模組織（2,000+ユーザー）: Word/Excel/PowerPoint/OneNoteからCopilot Chatが**完全に削除**（ボタン・サイドペインなし）
- 小規模組織（2,000未満）: 「standard access」として継続するが、サービス容量によりパフォーマンスが変動。有料ライセンスへのアップグレードプロンプトが表示される
- Copilot Chat（Web版/M365 Copilotアプリ/Outlook/Teams）は引き続き利用可能
- ラベル変更: 無料ユーザーは「Copilot Chat (Basic)」、有料ユーザーは「M365 Copilot (Premium)」と表示
- https://chrismenardtraining.com/post/microsoft-365-copilot-chat-changes-on-april-15-2026-what-unlicensed-users-need-to-know/

## Power Platform

### Power Platform May 2026 Feature Update ブログ公開（5/14）
月次Feature Updateが5/14に公開。多数の新機能がGA/Previewに到達。

**Power Apps:**
- **User Defined Types (UDT) GA** — Studio v3.26044。新規アプリでデフォルト有効。複合データ構造の強い型付けをサポート
- **Custom Tools & Rich UI for App-Based Conversations（Public Preview）** — MCP対応カスタムツールでモデル駆動型アプリにビジネスロジックを統合。Fluent UIウィジェット付き
- **Generative Pages Input Support GA** — コマンドバーやグリッドアクションからコンテキスト入力を受け付け可能に
- **Grid Container GA** — CSSグリッドスタイルのレイアウト。ドラッグ&ドロップ、undo/redo対応
- **Data Grid Control（Public Preview）** — Fluent UIベース。検索・ソート・複数行選択対応
- **InfoPath Migration Support GA** — Canvas Authoring MCP ServerでAIエージェント（GitHub Copilot/Claude Code等）がInfoPathフォームをPower Appsに変換

**Power Automate:**
- **Self-Healing（Public Preview）** — デスクトップフローのUI要素変更を検知し自動修復。GPT-4.1 mini + Claude Sonnet 4.5の2モデル併用
- **Desktop Flow Testing GA** — テストケースの作成・管理が正式リリース
- **Record with Copilot（Public Preview）** — AI支援による録画でデスクトップフロー生成

**Power Pages:**
- **GitHub Copilot Plugin（Public Preview）** — Claude Code/Copilot CLIからPower Pages開発を支援

**管理:**
- **Entra Agent Users（Public Preview）** — 自律エージェント用の専用ユーザーアカウント作成
- https://www.microsoft.com/en-us/power-platform/blog/2026/05/14/whats-new-in-power-platform-may-2026-feature-update/

## 開発ツール

### Claude Code v2.1.142（5/14）
- **Fast modeがOpus 4.7をデフォルト使用**（従来Opus 4.6）。`CLAUDE_CODE_OPUS_4_6_FAST_MODE_OVERRIDE=1`でOpus 4.6に固定可能
- `claude agents`にセッション設定フラグ追加: `--add-dir`, `--settings`, `--mcp-config`, `--plugin-dir`, `--permission-mode`, `--model`, `--effort`等
- プラグインのルートレベル`SKILL.md`がスキルとして自動認識。LSPサーバー情報を`/plugin`詳細ペインに表示
- MCP_TOOL_TIMEOUT修正（リモートHTTP/SSE MCP対応）、macOSスリープ後のデーモン再接続修正など多数のバグフィックス
- https://code.claude.com/docs/en/changelog

### Grok Code Fast 1 廃止完了（5/15）
- xAIのモデル提供終了に伴い、GitHub Copilot全体験（Chat/インライン編集/Agent Mode/コード補完）から5/15付で削除済み
- 移行先: GPT-5 miniまたはClaude Haiku 4.5
- Enterprise管理者はCopilot設定のモデルポリシーで代替モデルへのアクセスを有効化する必要あり
- https://github.blog/changelog/2026-05-08-upcoming-deprecation-of-grok-code-fast-1/

### Cursor: Cloud Agent Dev Environment Tools（5/13）
- マルチリポ環境対応: 1つの環境に複数リポジトリを設定し、クロスリポでの変更影響の推論・テスト・検証が可能
- Dockerfileベースのセットアップ（ビルドシークレット対応、キャッシュ高速化）
- ガバナンス・監査ログ・環境レベルのセキュリティコントロール追加
- https://cursor.com/changelog

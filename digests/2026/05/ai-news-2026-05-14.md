# AI News Digest - 2026-05-14

## 最優先: Microsoft 365 Copilot / Copilot Studio

### Copilot Studio 4月アップデート総括 (5/11公開)
- **Apps in Agents GA**: エージェントがCopilot Chat内でリッチなアプリ体験を直接表示。データ確認・レコード更新・承認をチャット内で完結
- **Analytics Viewer Role GA**: 管理者以外にも分析ダッシュボードへのアクセスを付与可能に
- **Workflows Agent一元管理**: Prompts/Agents/M365 Copilotを統合したワークフロー構築インターフェース。AI Workflow Designerで自然言語からフロー生成
- **Agent Store拡充**: Adobe Express、Box、Figma、Monday.com、Wixが新規追加
- **Work IQ API Public Preview**: エージェント間通信（A2A）対応。M365ファイル・メール・会議・チャットからリアルタイム業務インサイトを取得
- **GPT-5.5 Thinking**: Early Release環境で利用可能（GCC除く）
- https://www.microsoft.com/en-us/microsoft-copilot/blog/copilot-studio/new-and-improved-agent-governance-intelligent-workflows-and-connected-app-experiences/

### Copilot Chat ライセンス変更 5/16実施（あと2日）
- 5/16以降、M365 Copilotライセンス未保持ユーザーはWord/Excel/PowerPoint/OneNote内のCopilot Chatが利用不可に
- スタンドアロンCopilot Chatアプリ（Web）経由のアクセスは継続。Outlook内のCopilot（受信トレイ・カレンダーグラウンディング）も継続
- テナント管理者は影響範囲の事前確認が必要
- https://chrismenardtraining.com/post/microsoft-365-copilot-chat-changes-on-april-15-2026-what-unlicensed-users-need-to-know/

### Copilot Calendar Agent + Planner統合
- **Calendar Agent**: 自然言語でルールを設定し、Copilotがバックグラウンドでスケジュール管理を自動実行。ルーティン的な日程調整を委任可能に
- **Planner統合**: Copilot Chat内からタスク作成・期限確認・チーム割り当て表示が可能。ツール切替なしでプロジェクト管理
- https://supersimple365.com/whats-new-in-microsoft-365-and-copilot-may-2026/

### Outlook Copilot強化 (5/10発表)
- **Canvas Draft**: Outlook内でCopilotがメール下書きをキャンバス形式で生成
- **Copilot Insights Classic**: Classic Outlookでもメール本文を選択→Copilotで関連情報を問い合わせ可能に（新Outlookでは既存機能）
- **iOS添付ファイル分析**: Outlook iOS版でファイルプレビューから直接Copilotを呼び出し、添付ファイルの要約・分析が可能に
- https://www.windowslatest.com/2026/05/10/new-outlook-and-outlook-classic-features-in-may-2026-include-copilot-insights-teammates-calendars-automapped-calendars/

### Conditional Access強制適用開始 (5/13)
- 「All resources」ターゲットのConditional Accessポリシーが、リソース除外付きでも強制適用開始
- 数週間かけて段階的に展開。テナント管理者はポリシー設定の確認が必要
- https://mc.merill.net/message/MC1223829

### Claude Code v2.1.139-140 (5/12-13)
- **v2.1.139 (5/12) — Agent View**: `claude agents`コマンドで複数セッションをCLI上で管理。バックグラウンド実行、ステータス確認、必要時のみセッションに復帰する非同期ワークフロー
- **v2.1.140 (5/13)**: Agent toolの`subagent_type`マッチングが大小文字・区切り文字に寛容に（例: "Code Reviewer" → code-reviewer）。`/goal`がhooks無効時にハングする問題を修正。バックグラウンドサービスのアイドル終了時接続断を修正。プラグインのdefaultフォルダ無視警告を追加

## 高優先: Power Platform

### Power Fx: User Defined Types GA (5/13)
- Power Apps Studio v3.26044でUser Defined Types（UDT）が正式GA。新規アプリではデフォルト有効
- 既存アプリは Settings > Updates > New > User-Defined types で有効化
- テーブル・レコードをUDFに渡せるようになり、ParseJSONでのJSON→型付きオブジェクト変換時のバリデーションが自動化。実行時エラーを大幅削減
- https://www.microsoft.com/en-us/power-platform/blog/power-apps/power-fx-user-defined-types-generally-available/

### Power Apps MCP Server: Closed-Loop Learning (5/12)
- エージェントがAgent Feed上で受けた修正フィードバックをデータ入力ツールの改善に自動反映する閉ループ学習機能
- 低リスク操作は自動実行、高リスク操作は人間承認フローを適用
- https://www.microsoft.com/en-us/power-platform/blog/power-apps/power-apps-mcp-server-introduces-closed-loop-learning-for-enterprise-agents/

## 高優先: GitHub Copilot

### Code Review: コメント体験改善 (5/12)
- **Severity labels**: レビューコメントにHigh/Medium/Low重要度ラベルを付与。優先順位判断が容易に
- **Grouped comments**: 類似コメントをグループ化し、大規模PRでのノイズを削減
- 新Pull Requestsエクスペリエンスにオプトインしたユーザーが対象
- https://github.blog/changelog/2026-05-12-copilot-code-review-comment-experience-improvements/

### Copilot CLI v1.0.46 (5/12)
- 非推奨CLIバージョン使用時の警告表示を追加（プレミアムモデルアクセス喪失リスク通知）
- read-only `gh` コマンド（list, view, status, diff等）がユーザー確認なしで自動承認
- diff表示がターミナル幅で折り返し（従来は切り捨て）。PowerShell .NET global tool shimでの起動不具合修正
- https://github.com/github/copilot-cli/releases/tag/v1.0.46

## Microsoft 365 Copilot — July 01 Release Notes バッチ（7/1公開・本日回収）

対象期間 6/17〜7/1。以下は関心の高い順。全文: https://learn.microsoft.com/en-us/microsoft-365/copilot/release-notes

### ガバナンス（組織導入で最重要）
- **AI生成コンテンツの透かし（video/audio）**: Cloud Policy の「Include a watermark when content from Microsoft 365 is generated or altered by AI」ポリシーで、AIが生成・改変した動画/音声に視覚/音声の透かしを付与。**画像は対象外**（ユーザーが `myaccount.microsoft.com` の Privacy > Data options で個別設定）。透明性・誤帰属防止が目的（Roadmap 547831）
- **生成ファイルの秘密度ラベル自動継承**: Copilot がファイル生成時、参照元データの**最高ラベルを自動適用**。ラベル付与できない場合はユーザーに通知して共有前の手当てを促す。Purview/DLP 運用と直結する変更（未ラベルファイルの流出リスク低減）

### エージェント
- **エージェントへのスケジュール実行（Scheduled prompts）**: Analyst / Idea Coach 等の declarative agent に**定期プロンプトを予約**可能に。**実装が Power Automate フロー依存から脱却**（旧 preview で作成したものは期限まで PA 側で継続動作）（Roadmap 531759）
- **長時間実行エージェントタスクを Windows タスクバー表示**: アプリを開かずに agentic ワークフローの進捗/状態を確認
- **Copilot Studio エージェントを Teams チームへ共有**: 共有ダイアログでチームを選択し、メインチャネルへ通知＋インストールリンクを配布

### Outlook / Agent Mode
- **Outlook の Copilot Chat が受信トレイ・予定表・エンタープライズデータ全体を推論対象に拡大**（従来は単一スレッド）。**M365 Copilot ライセンス不要の Copilot Chat でも対象**（Roadmap 531910）
- **PowerPoint Agent Mode 強化**: Teams 会議/チャットを参照してデッキ生成、Work IQ で関連ファイル/会議/メールを自動収集、既存プレゼンの内容・スタイル/テーマを再利用

### その他（要点のみ）
- Copilot Notebooks: ノートブックから PowerPoint 生成 / マインドマップ / OneNote（Web・iPhone のマルチモーダルキャプチャ）
- Copilot Chat 内で Word/Excel/PowerPoint/PDF を直接オープン、SharePoint ライブラリ/フォルダの列メタデータ理解、Word「Audio Overview」のリアルタイム音声対話、PowerPoint に画像モデル MAI‑Image‑2‑Efficient、モバイル版の Siri 連携

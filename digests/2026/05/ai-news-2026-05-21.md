## Microsoft 365 Copilot / Copilot Studio

### Copilot Chat ライセンス変更が5/16に実施
非ライセンスユーザーのWord/Excel/PowerPoint/OneNote内Copilot Chatが削除された。Outlookの受信トレイ/カレンダーグラウンディング付きCopilot Chatと、M365 Copilotアプリ内のCopilot Chat（Basic）は引き続き利用可能。2,000席以上のテナントが主な対象。「Copilot Basic」「M365 Copilot Premium」のラベルも導入。
- https://learn.microsoft.com/en-us/microsoft-365/copilot/release-notes
- https://chrismenardtraining.com/post/microsoft-365-copilot-chat-changes-on-april-15-2026-what-unlicensed-users-need-to-know/

### GitHub Copilot: GPT-5.3-Codex がBusiness/Enterpriseの基本モデルに（5/17）
GPT-4.1を置き換え、エージェントコーディングタスクで最大25%高速化。GitHubとして初のLTS（Long-Term Support）モデルで、2027年2月4日まで12ヶ月間のサポート保証。Business/Enterpriseプランのみ対象。
- https://github.blog/changelog/2026-05-17-gpt-5-3-codex-is-now-the-base-model-for-copilot-business-and-enterprise/

### GitHub Copilot App テクニカルプレビュー（5/14）
スタンドアロンのデスクトップアプリ（Windows/macOS/Linux）。Issue/PR/プロンプトからエージェントセッションを開始し、git worktreeで隔離実行。Pro/Pro+は早期アクセス申請可、Business/Enterpriseは順次展開。
- https://github.blog/changelog/2026-05-14-github-copilot-app-is-now-available-in-technical-preview/

### GitHub Copilot Web: モデル選択更新（5/20）
Geminiモデル全削除、GPT-5.2 Codex、GPT-5.4 nanoも削除。選択可能モデルが整理された。
- https://github.blog/changelog/2026-05-20-updates-to-available-models-in-copilot-on-web/

### Copilot Studio: リアルタイム音声エージェント管理ガイド公開（5/19）
大規模音声エージェント運用の5段階ガバナンスフレームワーク（ビルダー統制→ビルド統制→リリース統制→ランタイム統制→ライフサイクル統制）。本番展開前のエスカレーションパステスト・レイテンシ検証・分析有効化が必須とされる。
- https://www.microsoft.com/en-us/microsoft-copilot/blog/copilot-studio/the-in-depth-guide-to-managing-real-time-voice-agents-at-scale/

## Anthropic / Claude

### KPMG + Anthropic 戦略提携発表（5/19）
KPMGの276,000人のグローバル従業員にClaude導入。「KPMG Digital Gateway Powered by Claude」でMicrosoft Azure上のクライアント配信プラットフォームにClaude統合。税務・PE（プライベートエクイティ）領域から開始。Cowork/Managed Agentsによりエージェントワークフロー構築が数週間→数分に短縮。
- https://www.anthropic.com/news/anthropic-kpmg

### Code with Claude London 開催（5/19-20）
5/19: Research/Claude Platform/Claude Codeの3トラック（ライブ配信あり）。5/20: Extended（独立開発者・初期スタートアップ向けワークショップ、録画後日公開）。次回は東京（6/5-6）。
- https://claude.com/code-with-claude/london

### Anthropic 課金変更: プログラマティック利用に別クレジットプール（6/15実施）
Agent SDK、`claude -p`、GitHub Actions等のプログラマティック利用がサブスクリプションから分離。Pro $20/月、Max 5x $100/月、Max 20x $200/月の固定クレジット（API料金準拠、繰越不可）。対話的利用（Web/デスクトップ/Claude Code端末）は従来通り。
- https://venturebeat.com/technology/anthropic-reinstates-openclaw-and-third-party-agent-usage-on-claude-subscriptions-with-a-catch/

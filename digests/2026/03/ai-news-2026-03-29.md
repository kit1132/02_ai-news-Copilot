# AI News Digest - 2026-03-29

## Copilot Studio / Power Platform

### Copilot Studio: Human-in-the-Loop（プレビュー）
エージェントワークフローに人間の判断を組み込む「情報要求アクション」が追加。Outlook経由で指定レビュアーに確認を送り、回答を受けてフロー再開が可能に。承認フローや例外処理の自動化に直結する機能。
- https://learn.microsoft.com/en-us/microsoft-copilot-studio/whats-new

### Copilot Studio: Agent Copy to Studio
M365 Copilotで作成したエージェントをCopilot Studioにコピーし、マルチステップワークフロー・カスタム統合・幅広いデプロイオプションを利用可能に。M365 Copilot→Studio間の移行パスが明確化。
- https://learn.microsoft.com/en-us/microsoft-copilot-studio/whats-new

### Vibe Power Apps（パブリックプレビュー）
自然言語プロンプトからフルコードのPower Appsを構築する新体験。VS Codeや手動コーディング不要で、アプリの作成・編集・公開まで完結。
- https://www.microsoft.com/en-us/power-platform/blog/power-apps/whats-new-in-power-platform-march-2026-feature-update/

### Power Pages Plugin for GitHub Copilot CLI & Claude Code（パブリックプレビュー）
自然言語でサイトを記述すると、プロジェクトのスキャフォールディング・Web API統合・権限設定・デプロイまでプラグインが自動処理。
- https://www.microsoft.com/en-us/power-platform/blog/power-apps/whats-new-in-power-platform-march-2026-feature-update/

### Power Platform & Copilot Studio アップデートイベント告知
4月15日 9:00 AM PDTにロードマップ詳細共有イベント開催予定。2026 Release Wave 1の具体的な内容が明らかになる見込み。
- https://www.microsoft.com/en-us/power-platform/blog/power-apps/whats-new-in-power-platform-march-2026-feature-update/

## Microsoft 365 Copilot

### GPT-5.4 Thinking / GPT-5.3 Instant 利用可能に
M365 CopilotおよびCopilot Studioで新モデルが利用可能。GPT-5.4 Thinkingは複雑なマルチステップタスクの深い推論、GPT-5.3 Instantはより高速・高精度な応答を提供。
- https://learn.microsoft.com/en-us/copilot/microsoft-365/release-notes

### Copilot Notebooks: 協働・作成機能の大幅強化（3/27発表）
SharePointサイト/フォルダ全体をソースとして参照可能に（自動同期）。OneNoteノートブック丸ごと参照にも対応。M365 Groupsへの直接共有で大規模チーム協働を簡素化。Word/PowerPointエージェントによるドキュメント・プレゼン作成もNotebook内で完結。Frontier Publicで3/28展開完了、GA は5月予定。
- https://techcommunity.microsoft.com/blog/microsoft365copilotblog/copilot-notebooks-enhancements-to-support-creation-collaboration-and-learning/4505360

## セキュリティ・ガバナンス

### Purview DLP for Copilot: 2つのGA（3月末ロールアウト開始）
1. **プロンプトSIT検出GA**: Copilot Chatプロンプト内の機密情報タイプ（SIT）を検出・ブロック。E1/E3/E5テナント対象。3月末GA開始→4月末完了予定
2. **全ストレージロケーション対応GA**: 従来SharePoint/OneDriveのみだったDLP適用範囲がローカルデバイス上のWord/Excel/PowerPointファイルにも拡大。既存ポリシーの再構成不要
- https://mc.merill.net/message/MC1181998
- https://mc.merill.net/message/MC1234661

### Entra Internet Access: Shadow AI Detection GA（3/31）
未承認AIアプリをネットワーク層で検出し、Defender for Cloud Appsのリスクスコアリング＋条件付きアクセスで即時制御。97%の組織が過去1年にID/ネットワークアクセスインシデントを経験、70%がAI関連との調査結果。同日GA: プロンプトインジェクション防御、AI向けセキュリティダッシュボード。
- https://techcommunity.microsoft.com/blog/microsoft-entra-blog/microsoft-entra-innovations-announced-at-rsac-2026/4502146
- https://www.microsoft.com/en-us/security/blog/2026/03/20/secure-agentic-ai-end-to-end/

# AI News Digest - 2026-06-08

## Claude Code v2.1.166（6/6）— フォールバックモデル・セキュリティ強化

v2.1.163（6/4）〜v2.1.166（6/6）で複数の重要機能が追加。

- **フォールバックモデル**: `fallbackModel` 設定で最大3モデルを指定可能。プライマリが過負荷・利用不可時に自動切替。`--fallback-model` がインタラクティブセッションにも適用
- **deny ルール glob 対応**: ツール名にglobパターン（`"*"`で全ツール拒否等）を使用可能
- **クロスセッションメッセージ硬化**: `SendMessage` で他セッションから中継されたメッセージがユーザー権限を持たなくなり、権限要求・autoモードでブロック
- **thinking 無効化**: `MAX_THINKING_TOKENS=0` や `--thinking disabled` でデフォルトthinking有効モデルのthinkingをAPI経由で無効化
- **v2.1.163**: `requiredMinimumVersion` / `requiredMaximumVersion` マネージド設定追加。組織が許可するバージョン範囲を強制可能
- JetBrains IDE（2026.1+）でのフリッカー修正、PowerShellハング修正、orphanプロセス修正等

https://code.claude.com/docs/en/changelog

## Cursor 3.7 Design Mode（6/5）— UIを指差し・声で編集

Cursor 3.7で「Design Mode」がリリース。テキスト記述の代わりにUIを直接操作してエージェントに指示する新しいインタラクション方式。

- **クリック・描画・音声**: ブラウザ内のUI要素をクリックまたは描画して変更箇所を指定。常時オンマイクで音声指示も可能
- **マルチ要素選択**: 2つ以上の要素を選択し、レイアウト関係を考慮した一括変更を指示
- **音声キューイング**: エージェント実行中でも次の変更を音声で予約可能
- **コンテキストレポート**: エージェントのトークン使用量をインタラクティブに可視化（system prompt / tool definitions / rules / skills の内訳）

https://cursor.com/changelog

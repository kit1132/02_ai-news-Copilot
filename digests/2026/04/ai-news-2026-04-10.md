## Copilot Chat 4/15仕様変更まで5日 — Basic/Premiumラベリング詳細判明

4月15日から、Copilot Chatライセンスなしユーザーは「Copilot Chat (Basic)」、M365 Copilotライセンスユーザーは「M365 Copilot (Premium)」とラベル表示される。2,000席以上の組織ではWord/Excel/PowerPoint/OneNoteでCopilotが完全無効化。2,000席未満は「standard access」に格下げされ、混雑時のパフォーマンス低下とライセンス購入促進UIが表示される。Outlookの受信トレイ・カレンダーグラウンディング付きCopilotは引き続き利用可能。

## GitHub Copilot in VS Code March Releases — Autopilot公開プレビュー（4/8公開）

4月8日に公開されたVS Code向けGitHub Copilot March Releases（v1.111〜v1.115）の主要機能:

- **Autopilot（Public Preview）**: 完全自律型エージェントセッション。エラー時の自動リトライ含め、手動承認なしでタスク完了まで動作
- **Agent Permissions**: セッションごとにDefault/Bypass Approvals/Autopilotの権限レベルを選択可能
- **思考努力の調整**: Claude Sonnet 4.6やGPT-5.4のリーズニング深度をモデルピッカーから直接制御
- **Chat Customizations Editor**: インストラクション・カスタムエージェント・スキル・プラグインを統合管理するUI。MCP/プラグインマーケットプレイス直接ブラウズ対応
- **#codebase セマンティック検索統合**: ローカル/リモートインデックスを廃止し単一自動管理インデックスに統合

- https://github.blog/changelog/2026-04-08-github-copilot-in-visual-studio-code-march-releases/

## Cursor Bugbot 自己改善 + MCP対応（4/8）

Cursor Bugbotが大幅アップデート。解決率78%を達成:

- **Learned Rules**: PR上のフィードバック（リアクション・コメント）から自動的にルールを学習。有効なルールは自動昇格、不要なルールは自動無効化
- **MCP Server対応**: コードレビュー時に外部MCP Serverから追加コンテキストを取得可能（Teams/Enterpriseプラン）
- **Fix All**: 複数のBugbot修正を一括適用するアクション追加

- https://cursor.com/changelog/04-08-26
- https://cursor.com/blog/bugbot-learning

## Claude Code v2.1.93〜v2.1.98 — Focus View、高速リリース継続

4月第1〜2週で6バージョンをリリース（v2.1.89から通算10バージョン）。主な新機能:

- **Focus View（v2.1.97）**: Ctrl+O でプロンプト・ツール要約・編集差分・最終回答のみを表示する集中ビュー（NO_FLICKERモード）
- **Deferred Tools**: ToolSearchでスキーマをロードしてから実行する仕組みを導入。未ロードツールの誤実行を防止
- **CLAUDE_CODE_NO_FLICKER=1**: alt-screen描画のちらつき防止環境変数
- **defer権限**: PreToolUseフックに「defer」判定を追加。ヘッドレスセッションでツール呼び出し時に一時停止し、`-p --resume`で再開可能
- **refreshInterval**: ステータスラインコマンドをN秒ごとに再実行する設定
- **/agents ライブインジケーター**: `● N running` でサブエージェント稼働数を表示

- https://github.com/anthropics/claude-code/releases
- https://code.claude.com/docs/en/changelog

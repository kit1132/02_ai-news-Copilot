# AI News Digest - 2026-05-15

## 今日の期限・変更（要対応）

### GitHub Copilot: Grok Code Fast 1 本日廃止（5/15）
xAI側のモデル提供終了に伴い、GitHub CopilotからGrok Code Fast 1が本日全面廃止。Copilot Chat、インラインエディット、エージェントモード、コード補完すべてで利用不可に。代替モデルとしてGPT-5 miniまたはClaude Haiku 4.5への移行が必要。Enterprise管理者はモデルポリシー設定で代替モデルを有効化すること。
- https://github.blog/changelog/2026-05-08-upcoming-deprecation-of-grok-code-fast-1/

### MC1189663: Actionable Messages 外部アクセストークン廃止（5/15）
Outlook/TeamsのActionable Messagesで使用される外部アクセストークンが本日廃止。Microsoft Entra認証への切替が必要。未対応の場合、Actionable Messagesが動作しなくなる。
- https://mc.merill.net/message/MC1189663

### Copilot Chat ライセンス変更 明日実施（5/16）
M365アプリ内（Word/Excel/PowerPoint/Outlook/Teams）でのCopilot Chat利用に新たなアクセス制限が適用。M365 Copilotアドオンライセンスのないユーザーに影響。大規模組織（Enterprise E3/E5等）は特に注意。
- https://learn.microsoft.com/ja-jp/copilot/microsoft-365/microsoft-365-copilot-licensing

### Office 365 コネクタ Teams無効化（5/18-22）
Office 365コネクタがTeamsで無効化される。Webhook URL経由の連携を使用している場合は代替手段への移行が必要。
- https://mc.merill.net/message/MC1181996

## 新リリース・アップデート

### Anthropic「Claude for Small Business」発表（5/13）
中小企業向けの新パッケージを発表。Claude Cowork内のトグルで有効化でき、QuickBooks、PayPal、HubSpot、Canva、DocuSign、Google Workspace、Microsoft 365と接続可能。給与計算、月次決算、キャッシュフロー予測などの自動化ワークフローを提供。追加料金なし（Claudeライセンス内）。PayPalと共同で「AI Fluency for Small Business」コース開設。5/14シカゴからSMBツアー（10都市）を開始。
- https://techcrunch.com/2026/05/13/anthropic-courts-a-new-kind-of-customer-small-business-owners/
- https://siliconangle.com/2026/05/13/anthropic-launches-claude-small-business-new-automation-workflows/

### Claude Code v2.1.141（5/13 深夜リリース）
v2.1.140に続く更新。主な変更点：
- `terminalSequence`フィールド追加（フックJSON出力でデスクトップ通知・ウィンドウタイトル・ベル制御可能に）
- `ANTHROPIC_WORKSPACE_ID`環境変数追加（ワークロードIDフェデレーション用）
- `claude agents --cwd <path>`でセッション一覧をディレクトリスコープに
- リワインドメニューに「Summarize up to here」オプション追加
- 長時間Thinking時のスピナーフィードバック改善（10秒後にアンバー警告）
- バックグラウンドエージェントがパーミッションモードを保持するように修正
- プラグインメニューのナビゲーション改善（矢印キー・クリック対応）
- https://github.com/anthropics/claude-code/releases

### Agent 365 AMA実施（5/12）
Tech CommunityでAgent 365のAMAセッションが開催。ライブデモ付きで、Agent Registry、ガバナンス、セキュリティ統合（Defender/Purview/Entra）について専門家が回答。Agent 365 GA（$15/user/月）の詳細運用情報が共有された。
- https://techcommunity.microsoft.com/blog/agent-365-blog/you-asked%E2%80%94we-answered-inside-the-agent-365-ama/4519485

# AI News Digest - 2026-04-13 (日曜日)

## Copilot Studio: ChatGPT-5 / Claude モデル群がグローバルGA

Copilot Studioで **ChatGPT-5** が全リージョン（GCC除く）で本番エージェント向けGA。加えて **Claude Sonnet 4.5、Claude Sonnet 4.6、Claude Opus** も同様にグローバルGA。従来は米国限定の有料実験プレビューだったが、全パブリックリージョンに拡大された。マルチモデル選択がCopilot Studioの標準機能として定着。

- https://learn.microsoft.com/en-us/microsoft-copilot-studio/whats-new
- https://learn.microsoft.com/en-us/power-platform/release-plan/2026wave1/microsoft-copilot-studio/

## Copilot Studio マルチエージェントオーケストレーション GA展開

マルチエージェント関連の複数機能がGA展開中。**Microsoft Fabric統合**（エンタープライズデータへのエージェント直接アクセス）、**M365 Agents SDK オーケストレーション**、**Agent-to-Agent（A2A）通信プロトコル** が含まれる。エージェントをサイロではなく協調システムとして運用する基盤が整備された。

- https://www.microsoft.com/en-us/microsoft-copilot/blog/copilot-studio/new-and-improved-multi-agent-orchestration-connected-experiences-and-faster-prompt-iteration/

## 【あと2日】Copilot Chat 4/15仕様変更 — Office内アクセス制限

4月15日から、2,000席以上の組織では **Copilot Chat（Basic）がWord/Excel/PowerPoint/OneNoteから削除**される。有料M365 Copilotライセンスなしではこれらアプリ内のCopilotボタン・サイドパネルが非表示に。ラベルも「Copilot Chat (Basic)」「M365 Copilot (Premium)」に分離。Outlookのメール・カレンダー推論は引き続き利用可能。2,000席未満の組織は混雑時に品質低下する「標準アクセス」に制限。

- https://kurtsh.com/2026/04/05/info-copilot-chat-behavior-changes-for-microsoft-365-users-coming-april-15-2026/
- https://office-watch.com/2026/microsoft-removes-copilot-chat-word-excel-powerpoint-april-2026/

## Power Platform April 2026 Feature Update（4/9公開）

4/9公開のPower Platform月次アップデートの主要GA項目：

- **Generative Pages（外部AIコーディングツール対応）GA** — GitHub Copilot CLI等でモデル駆動型アプリのカスタムページを構築可能に。全パブリッククラウドで利用可
- **カスタムテーマ（Modern Look）GA** — Fluent 2デザインでブランドカラー・フォント適用
- **Power Apps → M365 Copilot統合** — モデル駆動型アプリのデータをMCPサーバー経由でCopilotに直接表示
- **Power Platform Monitor アラート GA** — アプリ・エージェント・フローの正常性監視をセットアップ不要で利用可能
- **FetchXML Editor for Offline Profiles** — Public Preview

- https://www.microsoft.com/en-us/power-platform/blog/2026/04/09/whats-new-in-power-platform-april-2026-feature-update/

## M365 Copilot: Copilot Tuning / Authoritative Sources 4月展開

**Copilot Tuning** が4月中にFrontierライセンス向けに展開開始（5,000席以上のM365 Copilotライセンス保有企業が対象）。組織固有のデータでCopilotの応答品質をチューニング可能に。また、**Authoritative Sources管理** がM365管理センターのCopilot Search内で利用可能となり、SharePoint Onlineサイトを組織の権威的コンテンツとして指定可能。

- https://techcommunity.microsoft.com/blog/microsoft365copilotblog/what%E2%80%99s-new-in-microsoft-365-copilot--march-2026/4506322

## M365 Copilot: スケジュールプロンプト編集・Scatter画像エフェクト

**スケジュールプロンプトの編集機能** が追加。従来は削除・再作成が必要だったが、タイミングを直接変更可能に。また、**Scatter画像エフェクト** がCopilot Chatと作成モジュールに追加され、画像にモダンな分散スタイルを適用可能。

- https://learn.microsoft.com/en-us/copilot/microsoft-365/release-notes

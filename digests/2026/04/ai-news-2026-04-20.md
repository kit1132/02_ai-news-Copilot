## 最重要

### Security Copilot が M365 E5 に無料バンドル — 本日展開開始（4/20）
- MC1261596 に基づき、4/20〜6/30 の段階的ロールアウトで全 M365 E5 テナントに Security Copilot を追加
- 1,000 ユーザーあたり月 400 Security Compute Units（SCU）を無償付与（最大 10,000 SCU/月）
- ゼロクリックアクティベーション: テナント有効化7日前に通知 → 有効化日に2回目の通知 → 自動で利用可能に
- Defender / Entra / Intune / Purview に AI 脅威分析・インシデント要約・エージェント防御を提供
- https://mc.merill.net/message/MC1261596
- https://learn.microsoft.com/en-us/copilot/security/security-copilot-inclusion

### Claude Haiku 3 廃止完了（4/19）
- claude-3-haiku-20240307 が昨日サービス終了。API リクエストはエラーを返す（猶予期間なし）
- 移行先: Claude Haiku 4.5（claude-haiku-4-5-20251001）。max_tokens 4K→64K、料金 $0.25/$1.25→$1/$5 に変更

- https://learn.microsoft.com/en-us/partner-center/announcements/2026-april

### GitHub Copilot CLI: Auto Model Selection GA（4/17）
- `auto` モデル選択が全 Copilot プラン（Free/Pro/Pro+/Business/Enterprise）で GA
- Copilot が状況に応じて最適なモデルを自動選択
- Claude Opus 4.7 も Copilot CLI で利用可能に（Opus 4.5/4.6 を置き換え）
- https://github.blog/changelog/2026-04-17-github-copilot-cli-now-supports-copilot-auto-model-selection/

### GitHub Copilot データポリシー変更 — 4/24 発効まで4日
- 4/24 以降、Free / Pro / Pro+ ユーザーのインタラクションデータが AI モデル学習に使用される可能性あり
- Business / Enterprise は対象外
- Opt-out 設定: Settings → Copilot → 学習データ使用をオフに。期限が迫っており SNS で駆け込み対応の情報拡散中
- https://windowsforum.com/threads/github-copilot-privacy-shift-apr-24-2026-training-data-by-default-for-individuals.409998/

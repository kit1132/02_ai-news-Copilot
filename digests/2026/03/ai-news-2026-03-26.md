# AI News Digest - 2026-03-26

## Copilot Studio / モデル更新

### GPT-5.4 Thinking が M365 Copilot & Copilot Studio で GA
- GPT-5.3 Instant に続き、GPT-5.4 Thinking モデルが一般提供開始。複雑なマルチステップタスクへの深い推論が可能に
- Copilot Studio でも GPT-5 モデルのテスト・デプロイがプレビュー利用可能
- さらに Anthropic モデルが Copilot Studio および Researcher で選択可能に。マルチモデル戦略の具体化
- https://www.microsoft.com/en-us/microsoft-365/blog/2026/03/09/powering-frontier-transformation-with-copilot-and-agents/

### Copilot Studio: GPT-4o 廃止 → GPT-4.1 がデフォルトに
- Generative Orchestration を使用するエージェントで GPT-4o が廃止され、GPT-4.1 が新デフォルトモデルに
- パフォーマンス・信頼性・一貫性が向上
- https://learn.microsoft.com/en-us/microsoft-copilot-studio/whats-new

### Work IQ ツール（プレビュー）
- Copilot Studio エージェントを Work IQ サービスに接続し、リアルタイムの業務インサイト・コンテキストにアクセス可能に
- https://learn.microsoft.com/en-us/microsoft-copilot-studio/whats-new

## セキュリティ・ガバナンス

### Security Dashboard for AI が GA（RSAC 2026 発表）
- 2月からパブリックプレビューだったSecurity Dashboard for AIが一般提供開始
- Microsoft Defender、Entra、Purview のリスクシグナルを統合し、M365 Copilot、Copilot Studio エージェント、サードパーティAI（Gemini、ChatGPT等）のリスクを一元可視化
- 追加ライセンス不要。既存の Microsoft Security 製品で利用可能
- https://techcommunity.microsoft.com/blog/microsoft-security-blog/introducing-security-dashboard-for-ai-now-in-public-preview/4494637

### Entra Internet Access: プロンプトインジェクション防御が 3/31 GA
- ネットワークレベルのポリシーで悪意あるAIプロンプトをアプリ・エージェント横断でブロック
- Copilot Studio エージェントのセキュリティポスチャ管理も追加。ミスコンフィグ・脆弱性を攻撃前に検出
- https://www.microsoft.com/en-us/security/blog/2026/03/20/secure-agentic-ai-end-to-end/

### Copilot Studio 管理者向け新機能
- 組織全体へのエージェント共有を制限する管理コントロール
- エージェント所有権の移譲機能（完全な編集・削除権限を付与可能）
- 管理者向けエージェントライフサイクル自動化ルール: リスクの高いエージェントのブロック、非アクティブエージェントの削除、所有者不在エージェントの管理
- https://learn.microsoft.com/en-us/microsoft-copilot-studio/whats-new

## Power Platform

### Power Platform March 2026 機能アップデート（3/19公開）
- **テーマ: エージェンティックアプリケーション** — 従来のワークフロー自動化から、意思決定・学習・複雑な業務プロセスを自律実行するアプリへの転換
- **Power Apps**: モダンコントロール9種の包括リフレッシュ、プロンプトからフルコードアプリ生成（プレビュー）、オフラインファーストCanvas Apps の Dataverse リアルタイムアクセス
- **Power Automate**: AI駆動フロー提案の高度化、エージェントワークフロー連携、自己修復デスクトップフロー
- **Power Platform インベントリ GA**: テナント管理者がクラウドフロー・Copilot Studio エージェントフロー・Workflows を環境横断で一元管理
- https://www.microsoft.com/en-us/power-platform/blog/power-apps/whats-new-in-power-platform-march-2026-feature-update/

### Copilot Studio 開発者向け新機能
- **Agent API for Power Pages**: カスタムチャットUI を Copilot Studio エージェントと統合
- **Client SDK**: Android/iOS/Windows アプリにエージェントを埋め込み、マルチモーダル対話を実現
- **コードインタープリター（プレビュー）**: Excel/CSV/PDFファイルをアップロードし、Python コードで分析
- https://learn.microsoft.com/en-us/microsoft-copilot-studio/whats-new

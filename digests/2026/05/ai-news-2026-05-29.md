# AI News Digest - 2026-05-29

## Copilot Studio / M365 Copilot

### Copilot Studio に Mistral Medium 3.5 追加（5/28）
Copilot Studioのモデルプロバイダーに Mistral Medium 3.5 が新規追加（パブリックプレビュー）。長時間タスク・マルチツール呼び出し・構造化出力に最適化。EU組織向けにリージョン内データ処理が可能。M365管理センター＋Power Platform管理センターの二段階オプトインで利用開始。
- https://www.microsoft.com/en-us/microsoft-copilot/blog/copilot-studio/mistral-joins-copilot-studios-growing-lineup-of-model-providers/

### Copilot Studio 5月ブログ: オーケストレーション大幅改善（5/26）
5月の月次ブログが公開。主な内容:
- **オーケストレーション層刷新**: 評価性能20%向上、トークン消費50%削減
- **Computer Use GA**: 資格情報管理とインターフェース変更への適応機能を追加した企業向けバージョン
- **ワークフロー再設計**: エージェントを直接埋め込めるビジュアルデザイナー（早期リリース環境で利用可能）
- **Work IQ REST API / CLI**: リモートMCPサーバー接続対応。A2A通信もGA
- **リアルタイム音声エージェントGA**: 北米でDynamics 365 Contact Center経由で利用可能
- https://www.microsoft.com/en-us/microsoft-copilot/blog/copilot-studio/new-and-improved-computer-using-agents-a-new-workflows-experience-and-real-time-voice-experiences/

### Copilot Studio Question Themes Analysis GA（5/29予定）
ユーザー質問をAIがテーマ別にグループ化し、テーマごとの質問数・回答率・応答品質・ユーザーフィードバックを可視化する機能がGAに。エージェントのパフォーマンス改善に直結。
- https://learn.microsoft.com/en-us/microsoft-copilot-studio/analytics-themes

### Copilot Studio に GPT-5.5 Reasoning (Deep) プレビュー追加
深い分析的推論が必要なエージェント向けの実験モデルとして GPT-5.5 Reasoning (Deep) が選択可能に（Preview）。
- https://learn.microsoft.com/en-us/microsoft-copilot-studio/whats-new

### M365 Copilot Notebooks 5月更新（5/26）
Notebooksのリフレッシュ版が公開。会議のトランスクリプト・メールスレッド・Web調査など多様なソースをコンテキストとして取り込み可能に。OneNote iPhoneアプリからライブ音声・ホワイトボード撮影・ノートを1セッションで記録し、Copilotが構造化ページに変換。
- https://techcommunity.microsoft.com/blog/microsoft365copilotblog/what%E2%80%99s-new-in-notebooks--may-2026/4519838

## GitHub Copilot

### 組織ごとのモデルルール（5/26, パブリックプレビュー）
Enterprise向けに組織単位でCopilotモデルの可用性を制御する「Model Rules」が登場。Enabled（全組織で自動有効）/Optional（組織ごとに判断）を設定可能。Business/Enterpriseプランで利用可。
- https://github.blog/changelog/2026-05-26-target-copilot-models-to-organizations-with-model-rules/

### Individual プラン変更: 新規サインアップ一時停止
既存ユーザー保護のため新規サインアップを一時停止。利用制限強化、Opusモデルの Pro プランからの削除（Pro+ では引き続き利用可）。VS CodeとCopilot CLIで利用制限を表示。
- https://github.blog/news-insights/company-news/changes-to-github-copilot-individual-plans/

## AI ツール・プロダクト

### Claude Opus 4.8 リリース（5/28）
Anthropicの最新フラグシップモデル。API・Amazon Bedrock・Vertex AIで即日利用可能。主な改善:
- **SWE-bench**: 69.2%（Opus 4.7: 64.3%、GPT-5.5: 58.6%）
- **USAMO 2026**: 96.7%（Opus 4.7: 69.3%、数学で大幅ジャンプ）
- **コード品質**: Opus 4.7比で欠陥見逃しが4倍減少
- **Dynamic Workflows**: 数百ファイル規模の移行・コードベース監査を並列サブエージェントで実行
- 1Mコンテキスト、128k出力トークン、価格据え置き（$5/$25 per M tokens）
- https://www.anthropic.com/news/claude-opus-4-8

### Cognition（Devin）Series D: $1B調達、$26B評価額（5/27）
Lux Capital・General Catalyst・8VCがリード。ARR $492M、企業利用が年初から10倍成長。Mercedes-Benz・NASA・Goldman Sachsなどが顧客。8ヶ月前の$10.2Bから評価額2.5倍以上に。
- https://techcrunch.com/2026/05/27/ai-coding-startup-cognition-raises-1b-at-25b-pre-money-valuation/

### Claude Code v2.1.153-154（5/27-28）
v2.1.153で36件の変更。バックグラウンドセッション・claude agents の安定化。MCP/認証のセキュリティ修正。ワークツリー内のgitignoreパス出力が消失するバグ修正。カスタムAPIゲートウェイのトークン送信リグレッション修正。
- https://github.com/anthropics/claude-code/releases

### Cursor Composer 2.5（5/18）
Kimi K2.5ベースの新モデル。長期タスクでの性能と指示追従性が大幅改善。SWE-Bench MultilingualでOpus 4.7と同等のスコアを出しつつ、コストは約1/10。Standard: $0.50/$2.50 per M tokens。
- https://cursor.com/blog/composer-2-5

### Devin: v3 API 正式化 + マージコンフリクト自動検出
v3 APIがベータ卒業。RBAC・セッションアトリビューション・リポジトリ名フィルタ対応。PRのマージコンフリクトを自動検出しユーザーに通知する機能も追加。GitHub Enterprise Server対応も強化。

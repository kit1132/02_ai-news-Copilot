# AI News Digest - 2026-04-05

## Microsoft 365 Copilot

### Security Copilot が M365 E5 に標準バンドル（4/20〜段階展開）
- M365 E5 / E7 テナントに Security Copilot が自動有効化（ゼロクリックアクティベーション）。4/20〜6/30 にかけて段階展開
- 1,000 ライセンスあたり月 400 Security Compute Units（SCU）を無償提供。最大 10,000 SCU/月
- Microsoft Entra / Intune / Purview / Defender およびスタンドアロン Security Copilot ポータルのチャット・プロンプトブック・エージェントシナリオが対象
- https://mc.merill.net/message/MC1261596

### Copilot Chat が Outlook 共有/委任メールボックスで利用可能に（4月初旬〜）
- MC1246031: Outlook の共有メールボックス・委任メールボックスで M365 Copilot（Copilot Chat 含む）が利用可能に
- 4月初旬から展開開始、5月初旬に完了予定
- https://mc.merill.net/message/MC1246031

### Researcher に新出力形式追加（4月中旬〜下旬）
- PowerPoint / PDF / インフォグラフィック / 音声概要の4形式が追加予定
- 既存の Critique / Council モード（Frontier プログラム）と組み合わせ、リサーチ成果の表現手段が拡充
- https://learn.microsoft.com/en-us/copilot/microsoft-365/release-notes

### Copilot Notebooks にパブリック Web リンク参照機能（4月下旬）
- 公開 Web ページの URL を参照として追加可能に。Copilot の回答を特定のWebコンテンツでグラウンディング
- https://learn.microsoft.com/en-us/copilot/microsoft-365/release-notes

### Copilot Chat 仕様変更（4/15）まであと10日
- 4/15 より Word / Excel / PowerPoint / OneNote での Copilot Chat アクセスが有料ライセンス（M365 Copilot Premium）専用に
- Copilot Chat ユーザーは「Basic」ラベルに変更。管理者は準備期間中
- https://blogs.imperial.ac.uk/office365/2026/03/20/copilot-chat-changes-april-2026/

## AI ツール・開発環境

### Anthropic が OpenClaw 等サードパーティツールからの Claude サブスク利用を遮断（4/4）
- Claude Pro / Max プランのサブスクリプションを OpenClaw 等のサードパーティ AI エージェントフレームワーク経由で利用することを4/4正午（PT）に停止
- Boris Cherny（Claude Code責任者）：「サブスクリプションはこれらサードパーティツールの利用パターンを想定して設計されていない」
- 対象ユーザーに月額プラン相当の一回限りのクレジットを付与（4/17 まで有効）。追加使用量バンドル事前購入で最大30%割引
- OpenClaw 作者は2月に OpenAI に入社済み。コミュニティでは「裏切り」との批判も
- https://thenextweb.com/news/anthropic-openclaw-claude-subscription-ban-cost

### Claude API: 1M コンテキストベータ終了予定（4/30）
- Claude Sonnet 4.5 / Sonnet 4 の 1M トークンコンテキストウィンドウベータが4/30に終了
- 4/30 以降、標準 200k を超えるリクエストはエラーに。Claude Sonnet 4.6 / Opus 4.6 への移行が必要
- https://releasebot.io/updates/anthropic

### Devin エンタープライズ機能強化（4/3）
- **Enterprise-Scoped Secrets**: エンタープライズレベルでのシークレット管理
- **Enterprise ACU Visibility Control**: 使用量情報の可視性制御
- **Enterprise MCP Registry Enforcement**: MCP サーバー許可リストの強制適用
- https://docs.devin.ai/release-notes/overview

# AI News Digest: 2026-05-07 (水)

## 本日のトップニュース

### Copilot Notebooks → Word/PowerPoint 生成、5月グローバルロールアウト中
- Copilot Notebooks のコンテンツ・参照情報から **PowerPoint プレゼンテーション**と **Word ドキュメント**を直接生成可能に。5月に世界展開中
- PowerPoint: フォーカス、詳細レベル、スライド枚数、デザインテーマを選択。ノートブックの構造・参照を反映したスライドを自動生成
- Word: ドキュメントタイプ、メイントピック、対象者、テーマを指定。レポート・提案書・要約を即座にドラフト化
- https://windowsforum.com/threads/copilot-notebooks-to-word-and-powerpoint-march-may-2026-rollout.406824/

### Claude モデル、M365 Copilot で default-on に切替（Word/Excel/PowerPoint）
- MC1296489: Anthropic Claude モデルが **対象テナントでデフォルト有効**に。従来の opt-in から方針転換
- Word 文書編集時に Claude モデルを選択可能。5月下旬にかけてワールドワイドロールアウト
- Excel では GPT-5.5 と Claude Opus 4.7 の両モデルをサポート。チャット/編集切替、プランモード、変更追跡が強化
- https://mc.merill.net/message/MC1296489

### Claude Opus 4.7、Copilot Cowork (Frontier) で利用可能に
- Tech Community「(Co)work in Progress」ブログで発表。Copilot Cowork 内で Anthropic の最新モデル Claude Opus 4.7 が選択可能に
- Frontier ユーザーがマルチモデル環境で推論品質・コストのバランスを最適化できる
- https://techcommunity.microsoft.com/blog/microsoft365copilotblog/cowork-in-progress/4511672

## Microsoft 365 / Copilot

### MC1279072: Copilot Credits prepaid billing（PAYG不要化）
- M365 管理センターの課金ポリシーで **prepaid Copilot Credits のみでの利用が可能に**。Azure サブスクリプション不要
- 容量パック: $200/テナント/月（25,000クレジット）。AI 管理者が部門・ユーザーグループ単位でクレジット配分可能
- 4月下旬開始、5月上旬にロールアウト完了予定
- https://mc.merill.net/message/MC1279072

### AI in SharePoint、5月ワールドワイド展開中

- 技術的専門知識なしで SharePoint コンテンツを作成・管理。3月 Public Preview → 5月グローバルロールアウト
- https://www.microsoft.com/en-us/microsoft-365/roadmap

## AIツール（Anthropic / Claude）

### Claude Code v2.1.129（5/6リリース）
- `--plugin-url <url>` フラグ追加: URL から .zip プラグインアーカイブを直接取得してセッションに適用
- MCP サーバーの一時的エラー時に**最大3回自動リトライ**（従来は切断のまま放置）
- ターミナルタブのセッションタイトルが**設定言語で生成**されるように
- `/context` のASCII可視化グリッドが会話に流入し **~1.6kトークン浪費していたバグ修正**
- Opus 4.7 セッションが**ネイティブ1Mコンテキストウィンドウ**で正しく計算（膨張した/context表示と早期autocompaction修正）
- https://github.com/anthropics/claude-code/releases

### Claude for Word アドイン GA（Office スイート完成）
- Claude for Word が4月にGA。**Excel・PowerPoint・Word の3アドインが会話コンテキストを共有**
- Excel で分析 → PowerPoint でスライド化 → Word でレポート作成を、Claude が全コンテキストを保持したまま横断実行
- Word の編集はネイティブの変更履歴として表示
- https://support.claude.com/en/articles/13892150-work-across-excel-powerpoint-and-word

## 開発ツール

### Cursor SDK（TypeScript）Public Beta
- Cursor のデスクトップアプリ・CLI・Web アプリを動かすエージェントランタイムが **TypeScript SDK として利用可能に**
- `npm install @cursor/sdk` で開始。ローカルまたは Cursor Cloud（専用VM・サンドボックス・リポジトリクローン・開発環境構成済み）で実行
- コードベースインデックス・セマンティック検索・MCP サーバー統合・Skills・Hooks をサポート。トークンベース課金
- https://cursor.com/blog/typescript-sdk

### Cursor Security Review Beta（Teams / Enterprise）
- **Security Reviewer**: 全PRをセキュリティ脆弱性・認証リグレッション・プライバシーリスク・プロンプトインジェクション攻撃でチェック
- **Vulnerability Scanner**: 常時稼働のセキュリティエージェント
- Teams / Enterprise プランで利用可能
- https://cursor.com/changelog

### Devin: 5月アップデート
- **Streaming Shell Output**: バックグラウンドシェルプロセスの出力がワークログ内にインラインストリーミング
- **Cookie JSON Builder**: Cookie シークレットの新タブ付きUI（raw JSON ペースト、base64自動エンコード、有効期限警告）
- **Org-Level Metrics API**: 組織スコープのメトリクス・消費データ API エンドポイント追加
- **Devin Reviews Analytics**: 使用分析ページにレビューメトリクスセクション追加
- **Secrets on Initial Prompt**: 新規セッション作成時にシークレットを添付可能に
- https://docs.devin.ai/release-notes/2026

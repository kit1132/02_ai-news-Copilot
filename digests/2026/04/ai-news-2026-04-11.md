# AI News Digest - 2026-04-11

## Copilot Notebooks Overview Page 展開開始（MC1275564、4/10公開）

Microsoft 365 Copilot Notebooksに新しい **Overview Page** 体験が展開される。Notebook内のすべての参照資料を横断的に要約し、キーとなるトピック・テーマ・インサイトをワンクリックで更新できる。3カラムレイアウト（参照資料／Copilot Pages／Copilot chat）への刷新も行われ、チャット継続中にPagesで思考を並行キャプチャできる。参照可能な資料はWord/PowerPoint/Excel/OneNote/PDF/Copilot Pagesへ拡大。

- https://mwpro.co.uk/blog/2026/04/10/microsoft-365-copilot-updated-copilot-notebooks-overview-page-experience-mc1275564/
- https://techcommunity.microsoft.com/blog/microsoft365copilotblog/meet-the-updated-copilot-notebooks-experience-your-home-for-understanding-work-p/4501383

## Claude Cowork GA + Claude Managed Agents Public Beta（4/8-9）

Anthropicが4月8-9日に2つの大型発表を実施。前日 4/10 のダイジェストでは触れていない内容。

- **Claude Cowork GA（4/9）**: macOS/Windows版デスクトップアプリが研究プレビューを卒業し全有料プランでGA。エンタープライズ向け6機能追加 — RBAC、グループ支出上限、拡張利用分析、OpenTelemetry対応、Zoom MCPコネクタ、ツールごとのコネクタ制御
- **Claude Managed Agents Public Beta（4/8）**: 本番グレードのクラウドホスト型エージェント構築APIスイート。サンドボックス・認証・ツール実行・長時間セッション（切断を超えて継続）・セッショントレース組み込み。Notion/Asana/Sentryが先行利用。料金: **$0.08/runtime hour + モデル利用料**（24時間稼働で月$58のランタイム費用）。outcomes・multiagent・memoryはリサーチプレビュー

- https://claude.com/blog/claude-managed-agents
- https://9to5mac.com/2026/04/09/anthropic-scales-up-with-enterprise-features-for-claude-cowork-and-managed-agents/
- https://the-decoder.com/claude-cowork-expands-to-all-paid-plans-on-macos-and-windows-with-new-org-controls/

## M365 Copilot Researcher — Critique Mode と新出力フォーマット

Researcherエージェントに **Critique Mode** が追加。OpenAI GPTが下書きを生成 → Anthropic Claudeが正確性・網羅性・引用品質をレビューして最終化する二段階構成。マルチモデル戦略の第一弾。また **Researcher Council**（複数モデル並列実行で合意・対立を可視化）も追加。新出力フォーマットとして **PowerPoint / PDF / インフォグラフィック / 音声概要** が4月中旬〜下旬に展開予定。

- https://dataconomy.com/2026/04/02/microsoft-upgrades-365-copilot-researcher-with-new-critique-mode/
- https://www.geekwire.com/2026/microsoft-365-copilot-and-the-end-of-the-single-model-era-in-enterprise-ai/

## Security Copilot — M365 E5 に組み込み開始（MC1261596、4/20〜）

4月20日から6月30日にかけて段階展開。M365 E5顧客は追加コストなしで **1,000ライセンスあたり毎月400 SCU（最大10,000 SCU）** が自動付与される。Defender/Entra/Intune/Purviewに統合されたAI脅威分析・インシデント要約・エージェントベース防御が利用可能。テナントにはアクティベーション7日前と当日に通知。ゼロクリックで有効化される。

- https://mc.merill.net/message/MC1261596
- https://www.microsoft.com/en-us/security/security-copilot-in-microsoft-365-e5

## Power Automate — Process License Capacity共有が4/21 GA

最大 **25個のクラウドフロー間** でProcess Licenseキャパシティを共有できる機能が4月21日にGA。フローグループを作成してキャパシティをプールできる。ライセンス運用の柔軟性向上。Windows 365コネクタはPower Platform / Azure Logic Apps向けにPublic Preview（4/2）— Cloud PCの通知・プロビジョニングポリシー一括管理・管理者イベントへの自動応答を自動化可能。

- https://learn.microsoft.com/en-us/power-platform/release-plan/2026wave1/power-automate/
- https://4sysops.com/archives/windows-365-connector-for-power-platform-and-logic-apps-automate-cloud-pc-management/

## Claude Code v2.1.99 〜 v2.1.100 — /powerup 追加

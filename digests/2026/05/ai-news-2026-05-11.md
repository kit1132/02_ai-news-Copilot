# AI News Digest - 2026-05-11

## 最優先: Microsoft 365 Copilot / Copilot Studio

### Copilot Cowork: iOS/Android対応 + Skills + Plugins (5/5)
- Copilot CoworkがiOS/Androidに対応。モバイルからタスクを委任し、デスクトップで引き継ぐシームレスなワークフローが可能に
- **Cowork Skills**: 再利用可能な指示セットを保存し、エージェントが一貫した方法でタスクを実行。公開Skillsライブラリも提供開始
- **Plugins**: Fabric IQ (Power BI)、Dynamics 365、サードパーティ（LSEG、Miro、monday.com、S&P Global Energy）のネイティブ接続を追加
- **Federated Copilot connectors**: HubSpot、LSEG、Moody's、NotionがM365 CopilotとResearcherでGA。Excel対応は夏予定
- https://www.microsoft.com/en-us/microsoft-365/blog/2026/05/05/copilot-cowork-from-conversation-to-action-across-skills-integrations-and-devices/

### Agent 365 GA: Shadow AI検出 + Purview統合 (5/1 GA)
- $15/user/月。プリビルトエージェント、Copilot Studio製エージェント、パートナーエージェント（Genspark、Zensai、Egnyte、Zendesk等）を統一管理
- **Shadow AI検出**: Microsoft Defender + Intuneで、ローカルエージェント（OpenClaw、Claude Code、GitHub Copilot CLI等）の利用を検出。6月までに18種類対応予定
- **Purview Communication Compliance**: エージェント⇔人間のやり取りにポリシー適用、リスク行動の検出・調査が可能に
- **Defender連携**: エージェントの不審な権限乱用（例: MCP Server経由のメール送信）をブロック・アラート
- https://www.microsoft.com/en-us/security/blog/2026/05/01/microsoft-agent-365-now-generally-available-expands-capabilities-and-integrations/

### Copilot Studio MCP Server GA: Streamable HTTP + トレース強化
- MCP Serverが正式GA。新機能: tool listing（接続先MCPのツール一覧表示）、Streamable HTTP転送、Activity Mapでのトレーシング（どのMCPサーバーのどのツールが実行されたか可視化）
- SSEサポートは廃止済み、Streamable HTTPが標準トランスポート
- https://www.microsoft.com/en-us/microsoft-copilot/blog/copilot-studio/model-context-protocol-mcp-is-now-generally-available-in-microsoft-copilot-studio/

### Copilot Chat ライセンス変更 5/16実施
- 5/16以降、M365 Copilotライセンス未保持ユーザーはWord/Excel/PowerPoint/OneNote内のCopilot Chatが利用不可に
- ライセンスなしユーザーはM365 Copilotアプリ（Web）経由のみアクセス可能
- 元々4/15予定 → 5/16に延期されたものが実施へ

### Anthropic Claude モデル M365 Copilot内でデフォルト有効化
- Word、Excel、PowerPointでClaude（Opus 4.7含む）がデフォルト有効に。以前はopt-in
- M365 Admin Centerの「Copilot in M365 apps with Anthropic models」設定で管理

### Teams AI Recap: トランスクリプト不要の会議要約
- 録画・トランスクリプトを保存せずにAI会議要約を生成可能に
- データ保持・eDiscovery・プライバシーポリシーで録画禁止の組織向け

### Flex Routing EUテナント拡大
- EU/EFTAテナントでCapacity不足時、Copilot LLM推論がUS/Canada/Australiaにルーティングされる
- GDPR・データ主権要件のある組織は影響確認が必要

## 高優先: Anthropic / Claude

### Code with Claude 2026 (5/6) — Managed Agents 3大新機能
- **Dreaming (Research Preview)**: エージェントセッション・メモリを定期レビューし、パターン抽出・メモリ整理を自動実行。チーム横断の暗黙知を蒸留
- **Outcomes (Public Beta)**: 成功基準をルーブリックで定義→別コンテキストのgraderが出力を評価→不合格なら再実行。エージェントの推論バイアスに影響されない独立評価
- **Multiagent Orchestration (Public Beta)**: リードエージェントがタスク分割→専門サブエージェントに委任。各サブエージェントは独自のモデル・プロンプト・ツールを持ち、共有ファイルシステム上で並列実行
- https://claude.com/blog/new-in-claude-managed-agents

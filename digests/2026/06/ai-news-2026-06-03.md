## Anthropic / Claude

### Anthropic、IPOに向けS-1をSECに秘密裏提出
Anthropicが6/1にSECへ非公開でS-1を提出。直前の$65BのSeries Hで評価額は約$965Bに到達し、$852BのOpenAIを上回った。年間収益ランレートは$47B（前年$10Bから急成長）。上場時期は市場環境次第だが、OpenAIに先行する形となった。
- https://techcrunch.com/2026/06/01/anthropic-files-to-go-public/
- https://www.cnbc.com/2026/06/01/anthropic-ipo-s1-prospectus.html

### Claude Mythos - Project Glasswing、150組織・15カ国以上に拡大
Anthropicが6/2、サイバーセキュリティ特化モデル「Claude Mythos」のProject Glasswingを大幅拡大。電力・水道・医療・通信・ハードウェア業界の150新組織に提供。4月のプログラム開始以来、10,000件超の高〜重大脆弱性を発見済み。対象組織のサイバー攻撃影響範囲は各1億人以上。一般公開時期は未定（堅牢なセーフガード整備後）。
- https://techcrunch.com/2026/06/02/anthropic-scales-claude-mythos-to-critical-infrastructure-in-15-countries/
- https://www.anthropic.com/news/expanding-project-glasswing

### Claude Code v2.1.160（6/2リリース）
セキュリティ強化が中心。シェル起動ファイル（`.zshenv`、`.bash_login`等）や`~/.config/git/`への書き込み前にプロンプト確認を追加。`acceptEdits`モードではビルドツール設定ファイル（`.npmrc`、`.bazelrc`、`.pre-commit-config.yaml`等）の書き込みにも確認を追加。grep結果がRead前提チェックを満たすようになり、Edit操作が効率化。
- https://github.com/anthropics/claude-code/releases

## GitHub Copilot

### UBB移行後の反響激化 — クレジット枯渇・コスト増の報告相次ぐ
6/1のUsage-Based Billing（UBB）移行後、開発者からの反発が拡大。Pro+（$39/月、7,000クレジット）ユーザーが2時間で8%消費、2日以内に枯渇見込みと報告。別の開発者は単一の変更要求で$6以上消費。The Registerは「Angry devs vow to flee」と報道。コード補完・Next Edit提案はクレジット非消費だが、Copilot Chat・Agent Modeがクレジットを急速に消費する構造が問題視されている。新規Pro/Pro+/Student申込は引き続き停止中。
- https://www.ghacks.net/2026/06/02/github-copilot-usage-based-billing-takes-effect-drawing-developer-backlash-over-rapid-credit-depletion/
- https://www.theregister.com/ai-and-ml/2026/06/02/github-copilot-users-threaten-exit-as-metered-billing-kicks-in/5249826
- https://github.blog/changelog/2026-06-01-updates-to-github-copilot-billing-and-plans/

## Microsoft 365 / Copilot

### M365パッケージング・価格改定 — 7/1発効、6月から展開開始
7/1からM365商用スイートの価格改定。M365 E3 +8%（$36→$39）、E5 +5%、Business Basic +16%（$6→$7）、Business Standard +12%（$12.50→$14）、Office 365 E3 +13%（$23→$26）。パッケージ強化としてDefender for Office 365 Plan 1、Intune Remote Help/Advanced Analytics/Plan 2/Privilege Management等を追加（8/1までに展開完了予定）。既存顧客は更新時まで現行価格。テナントへのMessage Center通知は展開30日前。
- https://www.microsoft.com/en-us/licensing/news/2026-m365-packaging-pricing-updates
- https://licenseq.com/microsoft-licensing-update-june-2026/

### MC1317195: Copilot Notebooks — インフォグラフィック機能（6月下旬〜7月中旬）
Copilot Notebooksにインフォグラフィック生成機能が追加予定。6月下旬からロールアウト開始、7月中旬完了見込み。
- https://mc.merill.net/message/MC1317195

### Copilot Studio Build 2026.5.3 — 環境全体分析・カスタムメトリクス・MCPツーリング
Preview USのみ配信済みのBuild 2026.5.3の詳細が公開。主な新機能: (1) 環境全体の分析概要（複数エージェント横断でセッション数・自律実行数・DAU等を表示）、(2) カスタムメトリクス（独自の分析カテゴリを定義・追跡）、(3) MCPツーリング（MCP対応コネクタのツール一覧・プレビューが可能に）、(4) 音声ASRドメイン言語モデルがデフォルト有効化。破壊的変更: カスタム発音辞書が500エントリ上限に。セキュリティ面では機密メッセージの外部エージェント転送防止、DLP/ガバナンスのSharePointナレッジソース適用が強化。
- https://learn.microsoft.com/en-us/power-platform/released-versions/copilotstudio/2026.5.3

## AIコーディングツール

### Cursor: Canvas共有・/loop機能・Jira統合
Cursor 3.3（5/7）で追加されたJira統合の詳細が明らかに。Jiraからワークアイテムを@Cursorに割り当て、またはコメントで@Cursor言及するとクラウドエージェントが作業開始。PRはJiraに自動リンク。その他、Canvas共有機能（チーム内でライブスナップショットを共有）、`/loop`コマンド（定期実行エージェント）が追加。Teams向け料金改定ではStandard席の含有使用量を増加し、パワーユーザー向けPremium席を新設。Bugbot UBB移行は6/8更新後から適用。
- https://cursor.com/changelog
- https://www.atlassian.com/blog/company-news/cursor-in-jira

### Devin: ブループリントエディタ刷新・クラシック環境6/30廃止
ブループリントエディタが再設計（共有レイアウト、セクション別実行ボタン、ボトムターミナルドロワー）。リポジトリのブループリントエディタへのディープリンクも可能に。6/30にクラシック環境セットアップが廃止され、全組織が宣言的構成（ブループリント）に移行。v3 APIがベータ卒業し正式なプライマリAPIに。レガシーAPI（v1/v2）は将来廃止予定。
- https://docs.devin.ai/release-notes/2026
